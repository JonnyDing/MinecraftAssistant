介绍
煤炭（Coal）是采集煤矿石获得的物品。
获得方式
挖掘煤矿石或深层煤矿石会掉落一个煤炭，击杀凋灵骷髅时也会有1/3的概率掉落一个煤炭。
远古城市、林地府邸、地牢、废弃矿井、沉船、海底废墟、村庄、要塞、雪屋的奖励箱也会有煤炭。
用刷子刷扫古迹废墟和海底废墟里的可疑的方块有概率掉落煤炭。
用途
可用于作为熔炉的燃料使用，一块煤炭在熔炉中可以燃烧80秒（8个物品），给动力矿车提供4分钟的动力。
渔夫、盔甲商、武器商和工具商村民在新手等级交易以 1 个绿宝石收购 16-24 个煤炭。屠夫村民在学徒等级交易以 1 个绿宝石收购 16-24 个煤炭。（摘自中文Minecraft Wiki）
版本信息

在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为263。
在 Java 版 Indev20100219、Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
