介绍
红树根（Mangrove Roots）是一种和红树一起生成的装饰性方块。
生成
通过种植红树获得，当红树生成的时候，红树根会作为红树底部的根须生成，是一个可以含水的完整方块。当含水的红树根悬空的时候，水不会从旁边流下去。
用途
红树根可以作为熔炉的燃料，每一个红树根可以燃烧15秒（1.5个物品）。
把音符盒放置在红树根的上面可以使其发出贝斯的声音。红树根有30%的概率可以把堆肥桶的堆肥层数增加1。
版本信息
红树根在 Java 版 1.19 快照22w11a 版本和 基岩版 1.19.0 预览版1.19.0.20 版本中加入。
在 Java 版 1.19 快照22w17a 版本中修改了方块模型，使内部面和外部面的大小一致。
