铜门（Copper Door）是一种门，与木门一样可以直接开关铜门。
铜门与其他铜类方块一样，都会氧化，都可以涂蜡。
铜门与木门有着相同的机制和特性。
铜门在 Java 版 1.20.3 快照23w43a 版本和基岩版 1.20.50 预览版1.20.50.22 版本中加入，在 Java 版 1.20.3 快照23w44a、1.20.3-pre3 版本和基岩版 1.20.50 预览版1.20.50.22、1.20.60 预览版1.20.60.22 版本中修改了纹理。
