简介
注：下文的“熔岩”指熔岩源和流动的熔岩，不要混淆。
熔岩（Lava）是一种流速较慢且有伤害的流体方块。它可作为一种光源或陷阱。
熔岩可以发出等级为 15 的亮度。流动的熔岩具有 0 的爆炸抗性，而静止的熔岩具有 500 的爆炸抗性。在原版的生存模式中，无法获取物品形式的熔岩，但可以用桶采集后获取熔岩桶。
多数实体在每接触熔岩时 0.5 秒时会受到 4 点生命值伤害。同时还会被火烧着。在实体/玩家接触熔岩时，会立刻向火焰计时器添加 300 火焰刻。玩家每接触熔岩 1 刻，那么就会向计时器添加 2 个火焰刻。例如：
在熔岩里待 10 秒将会造成 700 个火焰刻（35 秒），可导致玩家着火（初始值 300 + 400 火焰刻，其中 200 刻是在熔岩里获得的），或者更确切地说，当他们离开熔岩源时 500 火焰刻（25 秒）内会着火。
下界里部分免疫火焰的生物和受到抗火药水效果的玩家或生物都不会受到伤害。如果着火的实体接触水或有雨落到它们身上，那么火就会熄灭，但是熔岩仍然会继续伤害它们。
熔岩下方如果有实体方块，那么会有熔岩粒子从方块下面滴落。不会造成伤害也不能填充炼药锅。
TNT、书架、树叶、地毯、羊毛、栅栏、煤炭块、藤蔓、草、木头、木板、木质台阶和木质楼梯都是易燃方块。如果易燃方块靠近熔岩上方或侧面一定距离，那么它们就会着火（TNT 会被点燃）。熔岩下方的易燃方块不会被直接点燃。火可以传播到其他方块上，利用这种方式也可以点燃下界传送门。
在主世界或末地或 Java 版 1.16 前的下界的熔岩流动速度是水的 1/8，流动距离是水的 1/2（4 格），并且不可再生。
存活时间
假定饥饿值是满的：
穿着全套无附魔的皮革护甲可存活 3.5 秒。
穿着全套无附魔的金护甲可存活 5 秒。
穿着全套无附魔的链甲可存活 5.5 秒。
穿着全套无附魔的铁护甲可存活 6.5 秒。
穿着全套无附魔的钻石护甲或下界合金护甲（下界合金护甲不会消耗耐久）可存活 14 秒。
生成
熔岩是自然生成的方块，生成的位置具体如下：
熔岩会替换在第 1 层到第 10 层的洞穴和山沟的空气方块。这意味着如果你创建一个没有任何洞穴和山沟的自定义世界，熔岩就不会出现。熔岩不会替换在 1-10 层的废弃矿井、地牢或要塞里的空气方块。
熔岩可以从熔岩源形成一条熔岩河的形式出现，从洞穴流进湖里。熔岩源方块可以在洞穴的一侧、峡谷、废弃矿井或地面上方的石头悬崖上出现。
熔岩可以以不同大小、尺寸的湖的形式生成。它们可以在任何海拔高度，任何生物群系找到，可导致森林火灾，有时候会在石头、泥土或沙子的上方生成浮空的区块。藤蔓和树会偶尔生长在这些浮空的区块上方（很小的岛屿有时候也会生成在熔岩湖的中心位置）。在地下生成时还会在其上方创造一个小型空间，导致地面下面为空的，或者其上面浮空的物品会掉下来。在自定义世界里，当设置生成熔岩海的值为 yes 时才会生成熔岩。
村庄里铁匠的屋子里拥有 2 格熔岩。
非寒冷生物群系的废弃传送门会带有若干个熔岩源。
在下界中，出现熔岩的频率比水在主世界出现的频率还要高。第 31 层海平面处会出现熔岩海，熔岩海约占下界的总高度的 1/4（因为 63 是主世界高度的 1/4）。它们最多可以向下延伸 22 层。熔岩也会在地狱岩里以单个方块的形式随机分布。（在主机版，熔岩是可再生资源，因为你可以在世界选项中重置下界。）
如果熔岩的上方或水平方向是蓝冰，下方是灵魂土，那么熔岩就会变成玄武岩。
互动
熔岩与水还会发生一些互动：
如果熔岩从顶部流到水或水流时，会生成石头。
如果熔岩水平流到水里，会生成圆石。
如果水水平流向熔岩，会发出嘶嘶声，出现一阵烟，但之后没有任何改变。
如果水垂直流向熔岩，会生成圆石或者是发出嘶嘶声。熔岩源会在这个过程中被摧毁，所以不像圆石那样可以再次创造出来，也不是可再生资源。
如果水流向熔岩源方块，会生成黑曜石。
如果垂直的水流流下来触及到熔岩源方块或熔岩源方块的任何一侧会生成黑曜石，即使水不会碰到熔岩那一格。
版本信息
熔岩在 Java 版 Classic 0.0.12a 中加入，在 Classic0.0.19a 和 1.5 快照 13w02a 中修改纹理。
在 Java 版 1.16 之后，下界的熔岩流动速度与流动最大距离和主世界的水一样。但是，这并不能使熔岩再生（无限岩浆别想了）。
在 Java 版 1.17 之后，熔岩终于可再生了，如果熔岩源下方的滴水石块带有滴水石锥，且下方有炼药锅，那么熔岩粒子逐渐滴落并缓慢填充炼药锅，成为新的熔岩，但是这没有伤害。
在 1.13扁平化之前，流动的熔岩数字 ID 为 10，静态的熔岩数字 ID 为 11。
