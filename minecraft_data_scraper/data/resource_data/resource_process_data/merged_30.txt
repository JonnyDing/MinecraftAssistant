马铃薯精炼炉（Potato Refinery）是一种给物品润滑和制作马铃薯油的功能方块。
马铃薯精炼炉的硬度为 3.5，没有合适挖掘工具，被破坏后不会掉落，内容物也不会掉落。
马铃薯精炼炉有 4 个槽位：1 个燃料格、2 个材料格和 1 个成品格，从左到右，从上到下依次为材料格、材料格、燃料格和成品格：马铃薯精炼炉的 GUI
马铃薯精炼炉有以下两个用途：
当炼制马铃薯油或毒马铃薯油时，左侧的材料格必须为马铃薯或毒马铃薯，右侧的材料格放置玻璃瓶，燃料格可放置任何燃料驱动，等待 125gt（6.25 秒）后，会在成品格中得到一瓶马铃薯油或毒马铃薯油，并消耗 2 个材料格中的物品，获得 0.1 点经验。
当给物品润滑时，左侧的材料格必须为马铃薯油（不可以是毒马铃薯油），右侧的材料格放置任何物品，等待 20gt（1 秒）后，会在成品格中得到经过润滑的物品，并消耗马铃薯油。
马铃薯花（Potato Flower）是一种自然生成于马铃薯维度中的花。
马铃薯花可以合成中毒效果的谜之炖菜。
放进堆肥桶时 85% 的概率使堆肥层数加 1。
马铃薯芽（Potato Bud）是一种类似于滴水石锥的方块，自然生成于马铃薯维度的腐败区生物群系中。
马铃薯芽只能放置或生长在马铃薯泥土、薯皮草方块、腐败薯皮草方块和马铃砂上。
与滴水石锥一样，下落时对生物造成伤害。
马铃薯芽接在收到随机刻后有概率生长一格。
放进堆肥桶时 30% 的概率使堆肥层数加 1。
马铃薯茎柄（Potato Stem）是构成马铃薯树的枝干的一种原木。
马铃薯茎柄没有对应的去皮变种，不可被引燃，不可作为燃料，无法被烧制成木炭。
放进堆肥桶时 30% 的概率使堆肥层数加 1。
马铃薯钩索（Lashing Potato）是一种可合成的工具。
当玩家右击掷出马铃薯钩索并在一定范围内钩到方块，玩家会被拽向目标方块，此时玩家会因为这个力导致玩家的移动限制在浮漂周围的几格方块内，再次右击可以收回光束。如果钩住的方块被破坏，浮漂不会下落，而是保留在原来的位置。马铃薯钩索不能钩实体。
马铃薯钩索还可以免受掉落伤害。
马铃薯锤（Potato Hammer）是一个自带击退X 的类似剑的武器。
马铃薯锤拥有 11 点（）攻击伤害和 6 攻击速度，可以进行无限次数的攻击。

骨块（Bone Block）是一种可以用来储存骨粉或装饰的方块。
它可以被任何镐开采，如果不用镐破坏，将不会掉落物品。金镐对它的破坏速度最快，为0.25秒/个。
用途
它可以由 9 个骨粉合成，主要起到压缩和储存骨粉的作用。
骨块可以放置在音符盒的底部，从而使音符盒发出“木琴”的声音。
自然生成
骨块会作为能在沙漠和沼泽生物群系找到的化石结构的一部分自然生成。

简介
黏土块（Clay）是一种生成在水中或繁茂洞穴中的方块，可以用任意工具开采，但用锹能更快。
如果不使用带有精准采集附魔的工具开采黏土块，则会掉落 4 个黏土物品。
自 Java 版 1.16.5 版本后，译名由粘土块变为了黏土块，1.20.5 版本后由黏土块变为了黏土。由于基岩版翻译质量较差，中文译名始终使用黏土块；英文译名于 1.21.30 预览版1.21.30.22 版本后，由Clay Block改为了Clay。
获取
黏土块通常生成于以下几个结构或生物群系中：
河流、湖泊和沼泽底部，呈圆盘状；
繁茂洞穴；
平原、热带草原和沙漠村庄的石匠小屋；
温水海洋的海床。
在 Java 版 1.20 快照23w12a 版本和基岩版 1.19.80 预览版1.19.80.22 版本后，可以在古迹废墟中用刷子刷可疑的沙砾，有 4.44% 的概率掉落黏土块。
在 Java 版中，还有一种获取方法，在 1.14 快照19w13a 版本后，石匠村民会向带有村庄英雄效果的玩家投掷黏土块，概率为 100%。
用途
在黏土块上，可以直接种植两种垂滴叶、红树胎生苗和杜鹃花丛，杜鹃花丛在长成树后会将黏土块转为缠根泥土。
自 Java 版 1.19 快照22w14a 版本和基岩版 1.19.0 预览版1.19.0.24 版本开始，可以使用泥巴、滴水石锥和滴水石块生产黏土块，将泥巴放置在滴水石块上，并在滴水石块下面悬挂滴水石锥，每随机刻有 45/256（约 17.6%）的概率将泥巴转化为黏土块，平均转化时间为 388.4 秒。
自 Java 版 1.12 快照17w17a 版本和基岩版 1.13.0 预览版1.13.0.9 版本后，黏土块放置在音符盒下方时，会使其发出长笛的音色。
在末影水晶（15w44a 版本）加入之前，Java 版 1.9 快照 15w31a 版本可以通过摆一个 8*8 的苦力怕的脸复活末影龙。

介绍
黏土球（Clay）是一种可以通过开采黏土块所得到的物品。它主要被用于合成黏土块或烧炼制成红砖。
获得方式
黏土球能通过使用没有精准采集附魔的工具开采黏土块获得，1个黏土块可获得4个黏土球。
黏土球有14.3％的概率可以在沙漠村庄屋子的箱子里以每组1个找到，有20.8％的概率可以在村庄的石匠箱子里以每组1-3个找到。
用途
可用于合成黏土块，也可以在熔炉中烧制成红砖。

介绍
黏液球（Slimeball）是掉落于史莱姆的物品。

流浪商人有8.06%（Java版）或21.74%（基岩版）的概率以4个绿宝石的价格出售1个黏液球。
用途
可用于合成黏液块、黏性活塞等物品，也可以用来繁殖青蛙并加速蝌蚪的成长。

在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为341。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
概述
黑曜石（Obsidian）是一种硬度和爆炸抗性极高的深紫色方块。
硬度：Java 版为 50，基岩版为 35（Java 版需要使用钻石镐挖掘 9.4s，基岩版则仅需 6.6s）；
爆炸抗性：1200（正常情况下生存模式可获得的爆炸抗性最高的方块）（除基岩、命令方块等特殊方块）因此可以抵抗原版除蓝色凋灵之首外的所有爆炸；
可用于建造下界传送门，也可用于合成信标、附魔台和末影箱。
不会被活塞和黏性活塞推动；
上方可放置末影水晶；
自然生成
黑曜石会自然生成在海底废墟、废弃传送门、林地府邸的黑曜石房间和熔岩房、黑曜石柱和末地船中。
当熔岩源和水接触时，黑曜石会取代熔岩源生成。
黑曜石也可通过一些特定事件生成：
当下界传送门在下界或主世界生成时会生成，其边框大小为 4*5 的黑曜石；
当玩家进入末地时，在玩家生成的地方（100，49，0）会生成 5*5 的黑曜石，这被称之为末地平台（Java 版）/黑曜石平台（基岩版）。这些黑曜石可以被挖掘，并且会在实体重新进入末路之地时再次生成（某些刷黑曜石机就是这样制作的）。
末地主岛上还会生成黑曜石柱，这些黑曜石柱会在末影龙复活时刷新，顶部会生成末影水晶。
获取
村庄武器匠小屋（旧版为铁匠铺）的箱子里有 25.64% 的概率生成 3～7 个黑曜石；
废弃传送门的箱子里有 46.44% 的概率生成 1～2 个黑曜石；
下界要塞的箱子里有 7.97% 的概率生成 2～4 个黑曜石；
堡垒遗迹塔尖的箱子里有 24.37% 的概率生成 4～6 个黑曜石；
使用不带有精准采集魔咒的工具破坏末影箱会掉落 8 个黑曜石；
猪灵获得金锭后有 8.71% 的概率给玩家 1 个黑曜石。
