闪长岩（Diorite）是一种生成在地下的方块。
生成
闪长岩会以团簇的形式生成在主世界Y＝0～60或Y＝64～128的范围，前者会生成2次，后者每区块会生成6次且有5/6的概率失败，团簇大小为0～64个方块。
闪长岩也有概率生成在在雪原村庄的铁匠铺和盔甲匠小屋里。

阳光探测器是一种类似于太阳能电池板的装置，在阳光下可以发出一定强度的红石信号。在 1.8 及以上，通过右键阳光探测器可以让它变为反向阳光探测器。
阳光探测器在 Java 版 1.5 快照 13w01a 中加入。
阳光探测器可以很容易地徒手破坏，但是用斧更快。反向阳光探测器不能被收集，破坏它们将会掉落正常的阳光探测器。
产生信号
阳光探测器可用来产生与日光强度和夜晚时间成正比的红石信号。
阳光探测器高 0.375 格（3/8 格）。阳光探测器不能被活塞推动（仅 Java 版和原主机版）。水和熔岩都会绕着阳光探测器流过而不会影响它们。
早上到中午红石信号会越来越强，中午到晚上红石信号会越来越弱，直至没有（夜晚）。
对其右键可以翻转其工作方式，让其变成反向阳光探测器。（1.8 以前的版本可以用阳光探测器+非门电路代替）这会反转由阳光探测器产生的信号强度。
阳光探测器在暴露在阳光下时激活，光源（火把、荧石等）不能激活阳光探测器，当阳光探测器上方有非透明方块遮挡时，其发出的信号强度会减弱，甚至不会输出信号，因为它与所接收到的天空光线亮度成正比；反向阳光探测器在暴露在足够低的阳光时激活。
阳光探测器的输出在时间从 13680 到 22340 时是没有信号的。
在下界和末地中，因为没有阳光，所以阳光探测器在夜光模式下一直都是输出充能等级 15 的信号，阳光模式下输出一直都是 0。
因 MC 中是有月相的，所以月光探测器在一天之中不会像正常模式那样有规律的变化。
此外，阳光探测器还会：
充能邻近的包括它下面的红石线以及背对阳光探测器的红石比较器的充能等级至依赖于时间、天气以及阳光传感器所接收到的天空光线亮度所输出的数值。
充能邻近的背对阳光探测器的红石中继器至充能等级 15。
激活邻近的包括上面的和下面的红石元件。例如活塞、红石灯等。

图书管理员村民的1、4、5级交易中包含附魔书的销售，这意味着每个村民会贩卖最多3本附魔书。售价在5-64个绿宝石之间，售价只根据附魔等级而定（比方说，无限 I 和保护 I 都花费5-19个绿宝石），不过像经验修补这样的宝藏魔咒是双倍售价。村民销售的附魔书含有所有种类、所有等级的附魔。
玩家可以在附魔台给书附魔来制作附魔书。 一本书获得多重附魔的几率较低，因为将要生成多个魔咒时会随机移除其中一个，并且书附到的魔咒的等级比大多数物品低。宝藏魔咒无法从附魔台获得。
用途
在生存模式中，附魔书是某些工具附上特定魔咒的唯一途径，例如为盾牌附上耐久魔咒。附魔书拥有一种附魔的光芒。
要使用附魔书，玩家需要在铁砧界面的第一栏放入待附魔物品（或者要合并得到的附魔书），第二栏放入要牺牲的附魔书。完成此过程还需要消耗经验。附魔书可以进行合并，得到一本更高等级或有更多魔咒的附魔书，与附魔工具和武器的合并操作类似。注意：当附魔书堆叠时（安装模组或用指令调出来），位于铁砧第二栏的多本附魔书会同时被消耗，为避免浪费，应确保该格子只放入了一本附魔书。
使用附魔书开始附魔时，可被待附魔物品接受的魔咒从附魔书转移到物品上。若待附魔物品和附魔书之间存在相同种类的魔咒，则取等级高的魔咒；若二者共有的魔咒等级相同且非最高等级，则合并为一个高一等级的魔咒。如果附魔书拥有待附魔物品不能接受的魔咒，则不能接受的部分会丧失，可接受的部分正常转移。附魔书是一次性的。
附魔书记载的魔咒并不会对附魔书本身有任何加成效果。例如，带有锋利 IV 魔咒的附魔书并不会比普通书（或者其他非武器物品）造成更多伤害。
附魔书可为可在附魔台上附魔的物品进行附魔，但不同于附魔台，附魔书可以增强已有魔咒，且可以应用在其他一些物品上。
仅在创造模式中，玩家可以对任何物品或方块附上任何魔咒，并使其产生效果。例如，木棍附上精准采集魔咒便可以成功采集草方块。即使切换回生存模式，物品的这些能力也不会丢失。相互冲突的魔咒仍然无法共存。例如，保护魔咒和爆炸保护魔咒仍然不能附在同一件物品上。若方块被附魔，则其在被放置后失去魔咒。

陶瓦（Terracotta）（曾称作硬化粘土（Hardened clay ））是一种由粘土块制成、硬度和爆炸抗性与石头相似的方块。陶瓦可以染成和羊毛一样的16种颜色，虽然它们的本来的颜色是偏橙色的。

红色、橙色、黄色、棕色、白色和浅灰色陶瓦生成在平顶山生物群系中，数量非常庞大。蓝色和橙色陶瓦也能在沙漠神殿找到。
陶瓦具有与大多数石头相同的爆炸抗性，明显高于烧制前的粘土块。1层陶瓦或染色陶瓦能抵御1米外的一次爆炸，保护后面的方块，但其本身会被破坏。

陷阱箱是一种打开时会向四周发出红石信号的一种箱子，锁扣的周围纹理偏红。
陷阱箱在Java版1.5快照13w01a中加入。
当箱子被破坏时，如果箱子里存放着物品，那么这些物品都将会掉落。如果将一个大箱子（由2个相邻的箱子组成）的其中一个箱子破坏，那么将会掉落最后27个储存格的物品，而剩下的27个储存格的物品则储存在剩下的一个箱子里。
陷阱箱会自然地生成在林地府邸里的伪末地传送门房间，打开将会引燃两侧的TNT，炸开房间内的石头怪物蛋并释放出蠹虫。
用途
陷阱箱可以被用来存储物品，也可以用来发射红石信号。陷阱箱不能放在大陷阱箱旁边，也不能放在即将在小陷阱箱旁生成大陷阱箱的地方（例如：陷阱箱不能放在两个小陷阱箱之间）。但陷阱箱能放在陷阱箱上方或下方。普通的箱子可以放在（大）陷阱箱旁边。
陷阱箱可以用作物品容器和红石元件。
把一个陷阱箱放在另外一个陷阱箱的旁边会变成一个“大陷阱箱”（也可称作“2个陷阱箱”）。如果要把两个陷阱箱放在一起而不让它们变成一个大陷阱箱，玩家需要在放置第二个陷阱箱时潜行‌‌[仅Java版]或放置时使第二个陷阱箱的朝向与第一个不同‌‌[仅基岩和Java版] 普通箱子不会与陷阱箱合并。陷阱箱不能放在大陷阱箱旁边，也不能放在即将在小陷阱箱旁生成大陷阱箱的地方（例如：陷阱箱不能放在两个小陷阱箱之间）‌‌[仅原主机版]，但陷阱箱能放在陷阱箱上方或下方。
陷阱箱能够被活塞推动‌‌[仅基岩版]，并且水和熔岩就算在陷阱箱周围流过，也不会造成什么影响。熔岩会在陷阱箱周围的空气方块生成火，陷阱箱看起来好像是可燃的，但陷阱箱不会被燃烧掉。（并且也不会以其他方式燃烧）
陷阱箱可用作熔炉中的燃料，每个陷阱箱可烧炼1.5个物品。
界面
一个陷阱箱拥有27格储存空间，大陷阱箱拥有54格储存空间。在大陷阱箱里，最上面的三行是这个大箱子的西边或北边部分的箱子，下面三行是南方或东方的箱子。
红石
陷阱箱可用于检测玩家何时访问了它的物品栏。
陷阱箱在不被打开时是不被激活的，但是当被玩家打开时会被激活（见上）。使用大陷阱箱的任一部分会激活大陷阱箱的两边。生物不能够使用或激活陷阱箱，并且陷阱箱不会因为物品通过投掷器或漏斗被传送进来或出去而被激活。 当其被激活时，陷阱箱：
1.为任何相邻的红石粉充能，包括在陷阱箱下方的。充能等级等于正在使用陷阱箱的玩家数（最大为15）。
2.为任何背朝陷阱箱的红石中继器充能使给出的信号强度达到15。
3.给在下面的任何完整的不透明固体方块（泥土、石头、金块等等）充能。充能等级等于正在使用陷阱箱的玩家数。（最大为15）
4.激活任何临近的红石元件，包括在其上下方的。比如活塞、红石灯等等。
激活的陷阱箱不会给相邻但背朝向它的红石比较器充能。红石比较器只能够测量陷阱箱的方块状态，根据陷阱箱被填满的程度产生一个强度在0到15内的信号。任何其他的被激活的陷阱箱充能过的方块（包括其下方的方块）会正常地为红石中继器充能。

该物品仅存在于基岩版！
隐形的基岩生成于旧世界边界以及基岩层下方40格位置，即Y=-41（基岩版1.18以前）或Y=-105（基岩版1.18及以后）的位置，玩家坠入虚空中时，可以防止玩家继续掉下去。只能用指令破坏，如果想要获得隐形基岩只能通过一些技术性手段，这样做可能会导致系统崩溃。这么大代价就为了这个？
隐形基岩不能被看见，但在程序设计的过程中是真实存在的，落在上方会有类似于石头的粒子特效，在物品栏中的材质也是石头的其中一面。
隐形基岩类似水和岩浆，玩家对方块的操作会将其忽略。例如尝试破坏它会穿过它而破坏后面的方块；尝试对它放置方块会以它后面的方块为支撑来放置，这导致即使是创造模式的玩家也无法直接破坏它。
消除隐形基岩只能使用指令/setblock或/fill替换。
概述
雕刻过的南瓜是南瓜雕刻过的变种。
雕刻过的南瓜可以自然生成在：
林地府邸的“铁路室”中；
掠夺者前哨站的稻草人标靶中。
可使用剪刀右击南瓜获得，同时掉落 4（仅 Java 版）/1（仅基岩版）个南瓜种子，该过程不可逆。使用剪刀剪去雪傀儡的南瓜头时也会掉落 1 个雕刻南瓜（这同时也会使南瓜的顶部纹理旋转方向，官方不予修复此特性）。
可以戴在玩家头上，不提供任何护甲值，玩家视野受阻挡（视野与南瓜的雕刻的外形镂空部分类似），看向末影人时不会激怒末影人。放入堆肥桶中有 65％ 的概率将堆肥层数增加 1。
雕刻过的南瓜可以外加4个铁块组成铁傀儡或外加两个雪块组成雪傀儡，或是用于装饰。
雕刻南瓜可以通过铁砧获得绑定诅咒和消失诅咒。
南瓜不能被活塞拉动，被推动时会被破坏并掉落，可利用此特性建造自动南瓜农场的采集装置。
历史

Java 版 1.13 快照17w47a 版本中，将“南瓜”重命名为“雕刻过的南瓜”，加入了没有脸图案的南瓜。
Java 版 1.17 快照21w11a 版本中，更改了雕刻过的南瓜顶面纹理的一个像素点以避免摆放时出现卐字。
Java 版 1.19.4 版本后修改译名“雕刻过的南瓜”为“雕刻南瓜”。
雕纹书架是一种能存放各类书的功能方块。可以存放书，书与笔，成书，附魔书，知识之书等。
雕纹书架可以使用任何工具或空手破坏，但是使用斧是最快的。被破坏后如果内部有书，书会掉落变为物品形式。
只有使用带有精准采集魔咒的工具才能采集该方块，否则不会掉落本身。
雕纹书架共有 6 个槽位，每个槽位都能放一本书，手持书并右击槽位可以将这本书放入。右键槽位中的书可以将其取出。
可以被红石比较器检测并发出红石信号，信号强度为最后与雕纹书架交互的槽位的序号。
可以与漏斗，漏斗矿车和投掷器交互。
雕纹书架可以作为熔炉的燃料，每个雕纹书架能烧炼 1.5 个物品。
雕纹书架在 Java 版 1.19.3 快照 22w42a 加入。

雪在Java版Alpha v1.0.4中加入。
在寒冷的生物群系，雪会覆盖在草方块上，且草方块会变为顶端带有雪的材质。在降雪天气也会自然生成在降雪的方块表面，每个厚度是0.125格（1/8格，2格像素），可以叠加到8个（1格，16格像素）。
雪可以使用带有精准采集魔咒的锹采集，掉落数量由雪的层数决定，且当采集满8层的雪时会直接掉落1个雪块，如果用无该魔咒的锹破坏雪，则每层雪都会掉落1个雪球。由TNT或苦力怕产生的爆炸也会使雪掉落为雪球，否则挖掘后不会掉落任何物品。
由8层雪叠起来的物质与普通雪块外形相同，但性质不完全一样。
将数个雪块叠起来后将下方雪块挖掉，上方雪块不会掉落，而雪恰恰相反。
周围有光源或加热块（仅基岩和教育版）时，在亮度等级大于 12 的区域内会融化。
霜冰（Frosted Ice）是一种特殊的冰，可以通过玩家穿着带有冰霜行者魔咒的靴子靠近水源来产生。
霜冰在 Java 版 1.9 快照 15w42a 中加入。
霜冰有在大多数方面表现得像冰，有些滑，因此可以让实体和物品滑动。
霜冰的自然融化速度很快。它刚刚生成的时候 age 为 0，每 1-2 秒重新计算。计算时，有 1⁄3 的几率 age 加 1 或者这个方块相邻的 6 个方向只有 4 个以下的霜冰方块时 age 加 1，前提是光照等级（包括阳光）高于（11-age）的数值。当 age 达到 4 时，这个方块会变成水，并且会使所有相邻的方块的 age 都会增加（除非这些方块也融化了，这效果就不会重复叠加）。这意味着在低亮度的地方，例如在洞穴内或夜晚，它永远不会自然融化。
当霜冰融化或被摧毁时，任何相邻的霜冰，在它相邻的 6 个面剩余的霜冰少于 2 个时就会立即融化。
当霜冰形成时，在其附近生长的作物和甘蔗不会因缺水而掉落。
在 Java 版中，若水源上方有障碍物阻挡（比如船和睡莲），霜冰将不会生成。睡莲可以放置于霜冰上。
