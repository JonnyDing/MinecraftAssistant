介绍
混凝土（Concrete）是有16种颜色的固体方块。
获得
红色混凝土和白色混凝土会自然生成在试炼密室中。
混凝土粉末与水或熔岩接触时会变成混凝土。
用途
混凝土的硬度比石头要高一些，但爆炸抗性却不如石头。
它相比羊毛的优势是不可燃，因此没有火灾的隐患。
版本信息
混凝土在 Java 版 1.12 快照17w06a 版本和 携带版 1.1.0 预览版1.1.0.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为251。
在 Java 版 1.20 快照23w12a 版本和 基岩版 1.19.80 预览版1.19.80.22 版本中，青色混凝土和红色混凝土会自然生成在古迹废墟中，且在基岩版中属于实验性玩法。
在 基岩版 1.20.0 预览版1.20.0.21 版本中，1.19.80.22 版本的改动不再属于实验性玩法，但在 Java 版 1.20 快照23w16a 版本和 基岩版 1.20.0 预览版1.20.0.22 版本中，青色混凝土和红色混凝土不再生成于古迹废墟中。

混凝土粉末（Concrete Powder）是有16种颜色、可以被水转变为混凝土的固体方块。
获得方式
可以用4个沙子、4个沙砾和1个任意染料无序合成8个。
如果混凝土粉末与水或熔岩接触的话，它会凝固成混凝土方块。特别的是，方块需要被放置、邻近或落入流动的水/熔岩或者水/熔岩源方块。如果混凝土粉末在降落时从侧面经过水，它不会凝固，如果落在侧面邻近水或熔岩的位置，它只会在方块更新之后凝固。 雨或喷溅型水瓶对混凝土粉末也没有影响。物品形式的混凝土粉末同样不会凝固。

漏斗是一种可以用来捕捉物品实体或用来转入或转出在容器中的物品的方块。
漏斗在Java版1.5快照13w01a中加入。
漏斗可以用5个铁锭和1个箱子合成。
使用
漏斗可用作容器、合成材料以及红石元件。漏斗在它的底部有一个“输出”管，它可以朝向下或朝向旁边，并提供存在容器时漏斗输出方向的视觉反馈。要放置漏斗，对着漏斗要输出管所朝向的面按下使用物品键（漏斗不会自动调整方向来对准容器）。要放置一个朝向容器（或其他按下使用物品键有互动的方块）漏斗，放置漏斗时还要按下潜行键。对着方块底部放置漏斗会使漏斗重新调整方向使其朝向下。漏斗在放置后不能改变朝向，也不能“附着”它所朝向的可被移除的容器，并且漏斗会继续朝向相同的方向。
内容
漏斗可用作容器，它有5个物品槽。要打开漏斗GUI，按下鼠标右键。在GUI打开时，要在漏斗物品栏和漏斗物品栏或快捷栏之间移动物品，则要拖动或按下⇧Shift然后点击物品。要关闭漏斗GUI，按下Esc按键。
用途
漏斗可用来吸入在它上方的物品实体，或运输物品到其他容器里。
漏斗是一种红石元件，它可通过下列方式激活：
1.一个邻近的激活的电源。例如红石火把（除了红石火把不会激活它附着的漏斗）、红石块、阳光传感器等。
2.一个邻近的充能方块（例如，一个底下有激活的红石火把的不透明方块）。
3.一个背对漏斗的激活的红石比较器或红石中继器。
4.邻近的指向漏斗（或在漏斗上面的）的有方向或无方向的充能红石粉。漏斗不会被邻近的背对漏斗的红石粉激活。
漏斗的行为与多数红石元件相反，因为它在未激活时进行操作，在激活时停止操作。因此，未激活的漏斗会被描述为开启，激活的漏斗会被描述为关闭。
开启的漏斗可做下面的三件事：
1.吸取它上方的物品实体（在世界中自由漂浮的物品）到它的物品栏里。
2.拉取它上方容器里的一个物品到它的物品栏里。
3.输出它自己的物品栏里的一个物品到它朝向的容器里。
在漏斗上方的要拉取物品的容器（熔炉、箱子、其他漏斗等）上面的物品实体不会被漏斗吸取（甚至不会检查它们，从而减少了漏斗所需要的更新数）。如果物品实体在漏斗上面那个方块的位置里，物品实体会被吸取。因此物品即使一部分在漏斗上面（例如，在台阶上），或完全“没入”漏斗上面的方块（因为它们正在向上穿过固体方块或它们被生成在那里）也能被吸取。物品实体如果在距离漏斗高于一格的位置（如在漏斗上面的石头的上面的物品）不会被吸取。漏斗每刻检查它上方是否有物品实体，甚至还可以在熔岩摧毁物品之前或玩家捡起它们之前把它们吸进漏斗。
漏斗有“运输冷却”时间。漏斗拉取或输出物品后，在漏斗再次拉取或输出物品之前会等待4红石刻（8游戏刻或不计延迟的0.4秒）。拉取物品和输出物品的操作都在同一游戏刻中进行，但是漏斗先进行拉取然后才进行输出（详见右边图解）。物品实体可在任何时间中被吸取而不会影响物品运输冷却时间，并且可以吸取一组物品（而不是每次吸取一个物品）。
漏斗总是尝试拉取或吸收物品到最左边的可用物品槽里（因为最左边的物品槽为空或最左边的物品槽还能和要被拉取的物品堆叠）。漏斗在拉取最右边的物品槽之前它会拉取最左边物品槽的物品（漏斗不会一开始输出第二个物品槽的物品，除非第一个物品槽是空的，或前两个物品槽是空的，如此类推），除非漏斗正在从它最右边的物品槽中输出可用物品到容器里（因为满的容器有未到堆叠上限的物品，这些物品与漏斗最右边的物品相匹配）。漏斗如果没有可用的物品槽空间（没有空的物品槽或未到堆叠上限的物品与要拉取或吸收的物品不匹配），漏斗不会吸取或拉取物品，甚至在它开启时也不会。相似地，漏斗会把物品输出到容器右下角的物品槽之前，把它输出到左上角的可用物品槽。如果没有可输出的物品，漏斗将会停止输出物品。

火（Fire）是一种有害的非固体方块。在java版1.8（14w25a）后，生存模式不管通过什么途径，火都不能以物品形式获得。它可以使用打火石和火焰弹放置，也可以通过熔岩点燃方块进行传播。
火在Java版Indev 0.31 20100109中加入。
火的具体产生方式如下：
在下界自然生成；
熔岩周围会自然生成火；
闪电可以创造火；
火也可以由床、重生锚、末影水晶、恶魂和烈焰人火球的爆炸生成。
当被放置时，火会在短暂而随机的时间内燃烧。如果其邻近没有可燃方块，火会消失。如果水碰到火，火会熄灭。
·传播机制
火会在任何一个可燃物的表面传播，同时会烧毁可燃方块（TNT则会被点燃）。
下表中，传播度是一个方块能够传播给别的方块火焰的能力。可燃度是该方块可以被周围的火传播而着火的难易度。这只是相对数值且并不代表实际时间，实际的时间取决于难度，雨，火燃烧的时长，还有因为方块刻而产生的随机性等。如果可燃方块处于潮湿的生物群系中，火势传播也会显著减少。潮湿的生物群系如沼泽、蘑菇岛和丛林以及其变种，但不包括丛林边缘。

·永恒之火
在下界岩、岩浆块、灵魂沙或灵魂土上点燃的火（灵魂沙和灵魂土上的为灵魂火）可以永久燃烧，除非被人为扑灭。雨不会熄灭此类火焰。末路之地中基岩上的火也会永久燃烧。
·对生物的影响
生物和玩家在接触到火时着火。火会轻微阻碍玩家视线，并且会以每秒1点生命值的速度缓慢减少他们的生命值。在玩家穿戴没有火焰保护的装备并且玩家没有防火BUFF时，装备可以给玩家提供的火焰伤害保护为护甲值的10%。在和平难度里，火伤害玩家的速度与玩家恢复生命值的速度一样，因此在这个难度里着火不会导致玩家死亡。离开火方块后，火会持续燃烧8秒，额外造成4颗心的伤害（除非火被雨或水扑灭）。在Java版中，着火的僵尸攻击玩家会导致玩家着火。掉进火里的物品或方块会着火，并且会很快消失（除非是下界合金制成的装备或远古残骸）。如果一个可以掉落肉的生物在其着火时死亡，会掉落其煮熟的版本，北极熊除外。大部分下界生物（除猪灵和猪灵蛮兵，疣猪兽）既不会被火造成伤害，也不会着火。
生物在接触到岩浆时也会着火，而且时间远长于接触火方块的时间，至少15秒而且每在熔岩内多待1秒都会多燃烧1秒。

简介
火把（Torch）是一种提供光源的非固体方块。
自然生成
火把会自然生成于以下结构中：
废弃矿井的走廊；
要塞的一些房间；
雪屋的地下室；
村庄的灯柱（雪原村庄除外）、针叶林村庄的农田、一些房屋、会合点、牲畜栏等；
掠夺者前哨站的瞭望塔；
林地府邸的一些房间；
远古城市的小型前哨站；
返回传送门的基岩柱上；
奖励箱的周围（在一个高度时会生成 4 个火把，否则不会生成火把）。
获取
火把可以作为战利品在一些结构中生成：
火把也可以通过煤炭或木炭与矿词包含stickWood的物品有序合成。
用途
火把可以发出亮度为 14 的光，因此可以照亮以及防止怪物生成。火把还可以融化曼哈顿距离 2 格内的雪和冰。
火把可以放置在大多数方块的上面和侧面，前提是该方块的上面和侧面必须是完整的，无法放置在任何方块的下面。
当火把附着的物品遭到破坏、推动等行为，或有水流过、活塞或黏性活塞试图推拉火把时，火把会被破坏并掉落本体，若熔岩流过则不会掉落本体，但在游戏的早期版本中（Java 版 Indev 开发期间），破坏附着方块后火把会转移到其他的方块上。
当沙子、沙砾、龙蛋等有重力的方块在坠落时遇到火把会变成物品形式，但可疑的沙子和可疑的沙砾会被直接破坏且不会掉落本体。
火把还可以作为合成材料参与一些物品的制作，在教育版或基岩版中还可以合成水下火把和彩色火把。
版本信息

简介
火把花种子（Torchflower Seeds）是能够种出火把花植株的物品，植株成熟后为火把花。
获取
火把花种子可以被嗅探兽刨挖出。
破坏未完全成熟的火把花植株会掉落一个火把花种子。
用途
火把花种子可用于吸引和繁殖嗅探兽和鸡（吸引嗅探兽功能在 Java 版 1.20 快照23w14a 版本和基岩版 1.20.0 预览版1.20.0.22 版本中加入），喂食给幼年个体时会减少 10% 的剩余生长时间，火把花种子还可以吸引和驯服鹦鹉。
给嗅探兽喂食火把花种子会恢复其 2 点（）生命值。
放进堆肥桶时 30% 的概率使堆肥层数加 1。
种植
在亮度等级≥7 时，对着耕地使用火把花种子可种植出火把花植株，火把花植株有 3 个生长阶段，只有在亮度等级≥9 时，瓶子草植株才能生长到下一个阶段。
使用骨粉有 75% 的概率增加 1 个生长阶段，有蜂巢且采集了花粉的蜜蜂飞过时，植株也有概率增加 1 个生长阶段。
成熟的火把花植株实际上是火把花。

灰化土（Podzol）是一种泥土变种。
生成
灰化土只会在针叶林或竹林生物群系生成。
获得
灰化土能通过使用任何工具或手挖掘并掉落成物品，但使用铲子挖掘最快。它只能通过精准采集附魔来得到，否则，只会掉落泥土。
灰化土和菌丝一样允许蘑菇在它上面生长并且不受亮度限制，从而允许巨型蘑菇的生长。树苗、所有花、甘蔗都可以在上面生长。

灵魂沙（Soul Sand）是自然生成在下界、能够减慢实体移动速度的方块。
生成
1.16前
灵魂沙会自然生成在下界Y＜65 的地方或者下界要塞的地狱疣房间里。
1.16后
灵魂沙会在灵魂沙峡谷作为地表层与灵魂土大量混合，在灵魂沙峡谷以外的下界生物群系，灵魂沙则以 1 - 12 个方块大小的团簇生成于Y ≤ 31 的地方，每区块尝试生成12次团簇。
灵魂沙也会在下界要塞的下界疣房、堡垒遗迹的藏宝室和疣猪兽棚、远古城市的中心和营房中生成。
用途
灵魂沙会降低所有站在其上方的实体（生物和玩家）的移动速度，穿上带有灵魂疾行附魔的靴子可避免此效果（并且还会让你加速）。并导致他们在灵魂沙上面行走时下沉 2 个像素。当灵魂沙被放置在水里或冰、浮冰或粘液块上时，这种效果会更加显著。它也可以导致尺寸小于一个台阶的生物窒息，也就是蠹虫、小海龟和末影螨。
灵魂沙可以用于种植下界疣以及制造凋灵。
也可用于合成灵魂火把和灵魂营火。
