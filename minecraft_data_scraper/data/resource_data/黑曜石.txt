概述
黑曜石（Obsidian）是一种硬度和爆炸抗性极高的深紫色方块。
硬度：Java 版为 50，基岩版为 35（Java 版需要使用钻石镐挖掘 9.4s，基岩版则仅需 6.6s）；
爆炸抗性：1200（正常情况下生存模式可获得的爆炸抗性最高的方块）（除基岩、命令方块等特殊方块）因此可以抵抗原版除蓝色凋灵之首外的所有爆炸；
可用于建造下界传送门，也可用于合成信标、附魔台和末影箱。
不会被活塞和黏性活塞推动；
上方可放置末影水晶；
自然生成
黑曜石会自然生成在海底废墟、废弃传送门、林地府邸的黑曜石房间和熔岩房、黑曜石柱和末地船中。
当熔岩源和水接触时，黑曜石会取代熔岩源生成。
黑曜石也可通过一些特定事件生成：
当下界传送门在下界或主世界生成时会生成，其边框大小为 4*5 的黑曜石；
当玩家进入末地时，在玩家生成的地方（100，49，0）会生成 5*5 的黑曜石，这被称之为末地平台（Java 版）/黑曜石平台（基岩版）。这些黑曜石可以被挖掘，并且会在实体重新进入末路之地时再次生成（某些刷黑曜石机就是这样制作的）。
末地主岛上还会生成黑曜石柱，这些黑曜石柱会在末影龙复活时刷新，顶部会生成末影水晶。
获取
村庄武器匠小屋（旧版为铁匠铺）的箱子里有 25.64% 的概率生成 3～7 个黑曜石；
废弃传送门的箱子里有 46.44% 的概率生成 1～2 个黑曜石；
下界要塞的箱子里有 7.97% 的概率生成 2～4 个黑曜石；
堡垒遗迹塔尖的箱子里有 24.37% 的概率生成 4～6 个黑曜石；
使用不带有精准采集魔咒的工具破坏末影箱会掉落 8 个黑曜石；
猪灵获得金锭后有 8.71% 的概率给玩家 1 个黑曜石。
版本信息
黑曜石在 Java 版 Classic0.28 版本和 携带版 Alpha0.1.0 版本中加入。
在 Java 版 1.13 快照 17w47a 版本扁平化前，数字 ID 为49。
在 Java 版 1.14 快照18w43a、18w44a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
