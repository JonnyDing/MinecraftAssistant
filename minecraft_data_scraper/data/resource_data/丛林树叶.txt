丛林树叶（Jungle Leaves）是丛林生物群系中丛林树上的叶子，破坏有几率获得丛林树苗，用剪刀破坏可获得本体，在 Java 版 1.14 快照 18w43a 和基岩版 1.10.0 快照 1.10.0.3 版本后破坏有小几率掉落木棍。在 Java 版 1.3.1 快照 12w18a 版本中破坏丛林树叶有概率掉落可可豆，但随后在 12w19a 中不再掉落。
丛林树叶和所有树叶一样，属于透明方块，但在当图像品质设置为“流畅”时使用不透明纹理。
在基岩版中，丛林树叶的结霜纹理中有两个黄色果子，虽然看起来长了果子，但是仅是纹理变化，掉落物并不会改变。也不能吃结霜纹理
丛林树叶在 Java 版 1.2.1 快照 12w03a 版本和携带版 Alpha 0.8.0 build 1 版本中加入。
在 Java 版 1.8 快照 14w25a 版本中使用了橡木树叶的纹理，但在下一个快照版本 14w25b 中改了回来，并修改了纹理，此外，丛林树叶还在 Java 版 1.14 快照 18w43a 版本和携带版 Alpha 0.9.0 build 1 版本、基岩版 1.10.0 快照 1.10.0.3 版本中修改纹理。
