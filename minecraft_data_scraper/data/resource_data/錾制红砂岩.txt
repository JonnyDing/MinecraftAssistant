介绍
雕纹红砂岩（Chiseled Red Sandstone）是红砂岩的雕纹变种。
合成
雕纹红砂岩可以用两个红砂岩台阶合成。
在 Java 版1.14 版本中，也可以用红砂岩在切石机中合成得到。
用途
可用于装饰。
雕纹红砂岩侧面的图案与凋灵相似。
版本信息
雕纹红砂岩在 Java 版 1.8 快照 14w32a 版本和 携带版 Alpha 0.14.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为179。
在 Java 版 1.14 快照18w43a、18w47a、18w50a版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版1.19.3 版本中，其译名由錾制红砂岩改为雕纹红砂岩。
