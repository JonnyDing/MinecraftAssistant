简介
骨头（Bone）是一种可以用来合成骨粉的道具。与木棍与烈焰棒一样，骨头的手持方式与工具类似。
获取
骨头可以作为箱子战利品生成在一些结构中。
Java 版
基岩版
除此之外，骨头还可以作为骷髅、凋灵骷髅、沼骸、流浪者和骷髅马的掉落物掉落 0-2 个，都受抢夺魔咒影响，每级增加 1 个。
在基岩版 1.4.0 预览版1.2.14.2 版本及以后，击杀小体型或中体型的鲑鱼有 25% 概率掉落 1 个骨头，热带鱼、河豚、鳕鱼和大体型的鲑鱼有 25% 概率掉落 1-2 个骨头。
骨头还可以作为钓鱼时的垃圾被钓上，无海之眷顾和饵钓时的概率为 1.0%，在丛林生物群系中，Java 版的概率为 0.9%，基岩版为 0.8%。
用途
在 Java 版 Beta1.4 版本后，骨头可用于驯服非敌对的狼，每使用一次有 1/3 的概率驯服，狼也会看向手持骨头的玩家，但不会被其吸引。
骨头还可以在无须合成 3 个骨粉。
版本信息
骨头在 Java 版 Beta1.2 版本和携带版 Alpha0.3.3 版本加入，在 Java 版 1.14 快照18w43a、18w44a 版本和基岩版1.10.0 版本中修改了纹理。
