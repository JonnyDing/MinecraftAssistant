介绍
深板岩（Deepslate）是一种在主世界深处代替石头生成的方块。
获取
深板岩不可再生。
自然生成
世界生成
在主世界中，石头在 y=1~7 高度处逐渐向深板岩过度；
在 y=-64~0，深板岩作为世界生成的基石完全代替了石头；
深板岩在 y=-64~-60 逐渐被基岩替换。
深板岩在 y=-59 丰度最高，约为 90.26%。
深板岩和石头过渡的景观
结构
在远古城市中心的一种地下室房间中会生成一个熔炉，其输出槽会生成 24 个深板岩。
合成
将深板岩圆石在熔炉中熔炼 10 秒可以获得 1 块深板岩和 0.1 点经验（x 0.1）。
创造模式物品栏
深板岩在建筑方块[1.19.2及以前]或自然方块[1.19.3及以后]标签页中。
用途
在工作台中合成各种深板岩材质的装饰性方块（如磨制深板岩、深板岩砖、深板岩瓦墙等）；
切石机有相同作用，且更省材料；
深板岩本身就是一种美观的建筑材料。
特性
基本特性
破坏
深板岩的硬度是 3，为石头（1.5）的两倍，需要使用木镐或更高等级的镐挖掘才能掉落。
若使用带有精准采集附魔的镐挖掘深板岩，将掉落深板岩本身，否则将掉落深板岩圆石。
深板岩的爆炸抗性为 6，同大多数石头相同。
蠹虫钻入
蠹虫会钻入周围的深板岩，并将其转化为被虫蚀的深板岩。
方块标签和额外特性
具有#minecraft:mineable/pickaxe标签，使用镐类工具挖掘深板岩更快；
具有#minecraft:azalea_root_replaceable和#minecraft:moss_replaceable标签，杜鹃树生长时会将其下方的深板岩替换为缠根泥土；
具有#minecraft:stone_ore_replaceables、#minectaft:deepslate_ore_replaceables和#minecraft:dripstone_replaceable_blocks标签，深板岩在世界生成时可以被各种深层矿石、泥土等方块替换；
具有#minecraft:base_stone_overworld标签，标记了深板岩是主世界的基石之一；
具有#minecraft:lush_ground_replaceable标签。
方块状态
和原木类似，深板岩方块具有一定朝向。
在世界中生成的深板岩默认为上下走向。
玩家放置的深板岩的朝向由玩家的朝向决定。
一般性质
深板岩是不透明方块，可以作为红石导体，可以窒息生物。
版本信息
深板岩在 Java 版 1.17 快照21w07a 版本和 基岩版 1.17.0 预览版1.16.230.52 版本中加入，此时在基岩版中属于实验性玩法，且名称为阴沉石（Grimstone）。
在 Java 版 1.17 快照21w08a 版本中，名称由阴沉石（Grimstone）改为深板岩（Deepslate）。
在 基岩版 1.17.0 预览版1.17.0.52 版本中，深板岩不再属于实验性玩法。
在 Java 版 1.17 快照21w18a 版本前，深板岩会生成于 y=0~16 处，如果加载了洞穴与山崖预览数据包，深板岩仍会生成在 y=0 以下；否则深板岩会在 y=0~16 以团簇的形式生成。
在 Java 版 1.18 快照21w44a 版本中，深板岩和石头的过渡从 y=-7~-1 改为 y=1~7。
在 Java 版 1.17 快照21w08a、21w10a 版本和 基岩版 1.17.0 预览版1.17.0.54 版本中修改了纹理。
