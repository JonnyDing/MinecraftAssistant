钻石锄在Java版Indev 20100206中加入。
用钻石做的锄，右击草方块或泥土使其变成耕地，而右击砂土则变成泥土。
1.16后，锄可以附魔效率、时运、精准采集，可以快速挖掘地狱/诡异疣块、干草块、标靶、干海带块、菌光体、干/湿海绵和树叶。
在1.16前用坏一把钻石锄可获得成就“终极奉献”，快照20w20a后改为将钻石锄升级为一把下界合金锄。哪个都很浪费
耐久度：金 < 木 < 石 < 铁 <钻石＜下界合金。
挖掘效率：木 < 石 < 铁 <钻石< 下界合金 < 金。
作为武器时，任何锄都会造成1点（）伤害，但会大幅提高玩家的攻击速度。
攻击速度：木 = 金 < 石 < 铁 <钻石＝下界合金。
