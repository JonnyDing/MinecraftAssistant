介绍
海晶碎片（Prismarine Shard）是一种守卫者和远古守卫者的可再生掉落品。
获得方式
海晶碎片在击败守卫者或远古守卫者时掉落，每次会掉落0-2个海晶碎片。在抢夺魔咒下会增加到最多5个。
用途
海晶碎片能被合成为海晶石及其变种，例如海晶石砖、暗海晶石和海晶灯。
版本信息
海晶碎片在 Java 版 1.8 快照14w25a 版本和 携带版 Alpha 0.16.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本的扁平化之前，数字 ID 为 409。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理，相比之前的纹理，中心部分颜色更加浅了。
