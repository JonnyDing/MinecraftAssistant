介绍
糖是一种由甘蔗或蜂蜜瓶合成的食物材料。
获得方式
女巫有概率在死亡时掉落 0～6 个糖，每 1 级抢夺魔咒都会多增加 3 个最大掉落量，最大为抢夺 III 的 15 个。
在教育版和开启了“教育版”选项的基岩版中，糖也可以用化合物创建器从其对应元素创建，配方为 6 个碳、12 个氧、6 个氢（解析：现实中葡萄糖的化学式为 C6H12O6）。
用途
可用于合成蛋糕、南瓜派和发酵蛛眼，也可以用来酿造迅捷药水和平凡的药水。
糖能喂马、驴和骡来恢复1点（）生命值并加快 30% 生长速度和增加 3% 驯服概率。
版本信息
糖在 Java 版 Beta1.2 版本和 携带版 Alpha0.3.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为353。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理，相比之前的纹理，更亮的同时高光部分没杂色。
