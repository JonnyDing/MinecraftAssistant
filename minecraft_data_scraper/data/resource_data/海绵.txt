介绍
海绵（Sponge）是一种能够吸水的方块。
获得方式
通过烧制湿海绵获得。
用途
可用于吸水，吸水后会变为湿海绵，湿海绵无法吸水，需要重新烧制。
可以吸收与其曼哈顿距离（|X1-X2|+|Y1-Y2|+|Z1-Z2|）不大于7格的水。
在 Java 版 1.8 快照14w25a 版本前，海绵无任何用途。
版本信息
海绵在 Java 版 Classic0.0.19a 版本和 携带版 Alpha0.8.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为19。
在 Java 版 1.8 快照14w25a 版本、1.14 快照18w43a 版本和 携带版 Alpha 0.16.0 预览版build 1 版本、基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.16 快照20w12a 版本后，海绵可以用锄类工具快速采集，速度由锄的质量和附魔效率等级决定。
在 Java 版 1.20.2 快照23w33a 版本和 基岩版 1.21.50 预览版1.21.50.20 版本中，海绵拥有了新的独特音效。
