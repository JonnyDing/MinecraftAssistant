介绍
附魔之瓶（Bottle o' Enchanting）是一种投掷后会破坏变为经验球的物品。
版本信息
附魔之瓶在 Java版 1.2.1 快照12w04a 版本和 携带版 Alpha 0.12.1 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 是384。
在 Java版 1.14 快照18w43a 版本和基岩版 1.10.0 快照1.10.0.3 版本中修改了纹理。
获得方式
附魔之瓶只能通过村民交易或远古城市、沉船、埋藏的宝藏与掠夺者前哨站中的战利品宝箱获得，右击投掷或使用发射器扔在地上会摔碎并出现一些经验球。每个经验瓶提供3~11个经验球。
大师级别的牧师村民会以3个绿宝石的价格出售1个附魔之瓶。
