介绍
切制砂岩（Cut Sandstone）是砂岩的切制变种。
生成
沙漠村庄由砂岩、切制砂岩、平滑砂岩组成。
热带海底废墟和沙漠神殿由砂岩、切制砂岩、錾制砂岩组成。
获得方式
切制砂岩可使用砂岩合成获得。
版本信息
切制砂岩在 Java 版1.2.4 版本和 携带版 Alpha0.6.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本和 基岩版 1.9.0 预览版1.9.0.0 版本中，平滑砂岩（Smooth Sandstone）被重命名为切制砂岩（Cut Sandstone）。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为24。
在 Java 版 1.14 快照18w43a、18w44a、18w47a、18w50a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
