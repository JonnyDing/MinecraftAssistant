介绍
骨块（Bone Block）是一种可以用来储存骨粉或装饰的方块。
它可以被任何镐开采，如果不用镐破坏，将不会掉落物品。金镐对它的破坏速度最快，为0.25秒/个。
用途
它可以由 9 个骨粉合成，主要起到压缩和储存骨粉的作用。
骨块可以放置在音符盒的底部，从而使音符盒发出“木琴”的声音。
自然生成
骨块会作为能在沙漠和沼泽生物群系找到的化石结构的一部分自然生成。
在 Java 版 1.16 快照20w06a 版本和 基岩版 1.16.0 预览版1.16.0.57 版本后，骨块会作为能在灵魂沙峡谷生物群系找到的下界化石结构的一部分自然生成。
版本信息
骨块在 Java 版 1.10 快照16w20a 版本和 携带版 1.1.3 预览版1.1.3.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为216。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.20.3 快照23w45a 版本和 基岩版 1.20.60 预览版1.20.60.20 版本中，骨块可以生成在试炼密室中骷髅的试炼刷怪笼旁，此时在Java版和基岩版均属于实验性内容和实验性玩法。
在 Java 版 1.21 快照24w18a 版本和 基岩版 1.21.0 预览版1.21.0.22 版本中，23w45a 版本和1.20.60.20 版本的改动不再属于实验性内容和实验性玩法。
