简介
眼眸花（Eyeblossom）一种自然生成于苍白之园的花。
眼眸花分闭合的眼眸花和张开的眼眸花两种。
生成
眼眸花会自然生成在苍白之园中，使用骨粉在该生物群系可以催出眼眸花。
在白天时，眼眸花是闭合的，在夜晚则是张开的。
在有昼夜交替的维度中，若玩家在对应的时间手动放置一个与之相反状态的眼眸花，在几秒后会发出粒子效果，并转换为对应状态的眼眸花。在没有昼夜交替的维度中则维持原状态。
用途
可用于装饰，闭合的眼眸花和张开的眼眸花分别可以合成具有失明效果和反胃效果的谜之炖菜，以及灰色染料和橙色染料。
玩家手持眼眸花会吸引蜜蜂，在与之交互时，蜜蜂会中毒。
放进堆肥桶时有 65% 的概率使堆肥层数加 1。
版本信息
眼眸花在 Java 版 1.21.4 快照24w44a 版本和基岩版 1.21.50 预览版1.21.50.25 版本加入。
