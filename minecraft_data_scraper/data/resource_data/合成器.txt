合成器（Crafter）是一种在接收到红石信号时自动合成并丢出产物的方块。
在 Java 版中，合成器需要用任意镐挖掘，但在基岩版中，挖掘最低所需要的工具为石镐。
合成器可以通过合成获得，无法自然生成。
合成器属于红石装置，可以合成物品和作为容器，放置时，输出口是正对着玩家的。对着合成器右击可以打开其 GUI 界面。因为是红石装置，所以会被红石信号激活。GUI 界面
空手单击合成格可以禁用该合成格，再次点击或在该格手动放入物品则启用。
漏斗、投掷器和其他合成器可以与合成器的任何一面互动，将物品转入合成器的未禁用合成格，转入的物品从左到右，从上到下依次填满未禁用合成格。
合成器不能像工作台一样直接被取走，而是会像投掷器一样，向前方掷出产物，如果前方有容器，则会存储在容器中。
如果一个配方会在合成格中留下一些物品，那么这些物品会跟随产物一起被投掷。
在 Java 版中，合成器无法被活塞和黏性活塞推动。
合成器在 Java 版 1.20.3 快照 23w42a 版本加入，在基岩版 1.20.50.21 版本加入。
