介绍
红色下界砖块（Red Nether Bricks）是一种下界砖块的红色变种。
获得方式
红色下界砖块可以通过下界疣和下界砖合成得到。
它需要被任何镐开采。若不用镐开采，则不会掉落任何东西。其中，金镐开采的速度最快，为0.25秒/个。
用途
红色下界砖块与下界砖块都是不可燃方块，且对恶魂火球免疫，这使得它成为合适的下界庇护所材料。
当走在红色下界砖块上面时有特殊的脚步声。
版本信息
红色下界砖块在 Java 版 1.10 快照16w20a 版本和 携带版 1.1.3 预览版1.1.3.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID为215。
在 Java 版1.16 版本中，译名由红色地狱砖块更改为红色下界砖块。
在 Java 版 1.14 快照18w43a、18w50a 版本、1.21.4 快照24w44a 版本和 携带版 Alpha0.14.2 版本、基岩版 1.10.0 预览版1.10.0.3 版本、1.21.50 预览版1.21.50.28 版本中修改了纹理。
在 基岩版 1.21.50 预览版1.21.50.29 版本中，修复了红色下界砖块在 1.21.50 预览版1.21.50.28 版本中的纹理未使用的漏洞。
