介绍
泥土（Dirt）是主世界最常见的不透明方块之一。
生成
泥土会生成在大多数主世界生物群系的地表。
泥土还会以团簇方式生成在任何生物群系的地下且Y= 0~160 处，每区块尝试生成7次，团簇大小为1-33个方块。
挖掘用时
可以用任何工具开采，但用锹最快。
当使用没有精准采集附魔的工具开采草方块、灰化土以及菌丝时，它们会掉落泥土而不是它们自身。
用途
泥土可以用锄右击来变成耕地，但如果生物在上面践踏或者周围4格内没有水时，耕地也会变回泥土。
把树苗、甘蔗、蘑菇和竹子直接放在泥土或砂土上，在合适的条件下就会生长。在泥土使用锄就能把它变为耕地，使小麦种子、南瓜种子、西瓜种子、马铃薯、胡萝卜和甜菜种子能在上面生长。
当泥土方块与草方块相邻并暴露在亮度4或以上的环境时，它会在随机的时段后变成草方块。菌丝的传播方式也相似，但需要亮度9或以上的环境。
对泥土使用水瓶会将其变为泥巴。
版本信息
泥土在 Java 版 pre-Classicrd-20090515 版本和 携带版 AlphaPre-release 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为3。
在 Java 版 1.0.0 快照Beta 1.9-pre5 版本中修改了纹理。
在 Java 版 1.20.3 快照23w45a 版本和 基岩版 1.20.60 预览版1.20.60.20 版本中，泥土可以生成在试炼密室的橡树下面一格，此时在Java版和基岩版均属于实验性内容和实验性玩法。
在 Java 版 1.21 快照24w18a 版本和 基岩版 1.21.0 预览版1.21.0.22 版本中，23w45a 版本和1.20.60.20 版本的改动不再属于实验性内容和实验性玩法。
