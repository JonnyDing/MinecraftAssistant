介绍
下界砖块（Nether Bricks）是一种构成下界要塞的方块。
获得方式
下界要塞由下界砖块组成。
也可由下界砖合成。
用途
下界砖块是不可燃尽方块，且对恶魂火球免疫，这使得它成为合适的下界庇护所材料。当走在地狱砖块上面时有特殊的脚步声。
如果一个下界砖块是在一个下界要塞中的一个完全封闭的空间里（不仅仅在单独的部分里），会在下界要塞生成的生物（比如烈焰人和凋灵骷髅）可以在其他生成条件都符合时生成在上面。
版本信息
下界砖块在 Java 版Beta 1.9-pre1 版本和 携带版 Alpha0.6.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为112。
在 Java 版1.16 版本后，译名由地狱砖块改为下界砖块。
在 Java 版 1.14 快照18w43a、18w47a版本、1.21.4 快照24w44a 版本和 携带版 Alpha0.14.2 版本、基岩版 1.10.0 预览版1.10.0.3 版本、1.16.0 预览版1.16.0.63 版本、1.21.50 预览版1.21.50.28 版本中修改了纹理。
在 基岩版 1.21.50 预览版1.21.50.29 版本中，修复了下界砖块在 1.21.50 预览版1.21.50.28 版本中的纹理未使用的漏洞。
