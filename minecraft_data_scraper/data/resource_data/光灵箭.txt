光灵箭（Spectral Arrow）是一种 1.9 版本新增的弓和发射器的弹药，只有在 java 版中才可以获得，在 1.14 以后版本也是弩的弹药。
光灵箭有持续 10 秒的发光状态效果。发光状态会对目标画出轮廓，轮廓能够穿透方块被看见，轮廓的颜色是基于目标的队伍（默认为白色）。
光灵箭是用荧石粉和箭制作的，为什么合成的是两根光灵箭，难道是被切开的箭？
光灵箭不受无限附魔影响，即附魔无限的弓仍会消耗光灵箭。
光灵箭在 Java 版 1.9 快照 15w31a 中加入，在1.14 快照 18w43a 中修改纹理。
1.16.2 快照 20w28a 后，可以和猪灵以物易物获得。
