简介
可疑的沙子（Suspicious Sand）是可供考古的方块，会在沙漠神殿内部的地窖、沙漠水井底部或热带海底废墟中自然生成。
获取
可疑的沙子受重力影响，但在落地后会直接消失，不会重新成为方块，也不会掉落物品。直接挖掘该方块不会掉落，这会使得物品丢失（精准采集魔咒无效）。
在 Java 版中，可以让可疑的沙子掉落到蜘蛛网或涡流（上升）气泡柱上，成为下落的方块（此时战利品已经消失）超过 30 秒后（在建筑高度之外为 5 秒）就会变成掉落物，这也是唯一一种获取可疑的沙子的方式。在基岩版中，只有使用创造模式或命令才能获得该物品。
用途
对可疑的沙子持续使用刷子刷一段时间过后，方块将变成普通的沙子，纹理上的裂缝不断扩大，同时掉落一个物品。若中途停止，方块会逐渐恢复原状。被刷子刷扫的一面会露出物品，并且刷扫完毕后会掉落。
可疑的沙子只能容纳一个物品，存储在方块数据中，只能通过/data命令来修改。自然生成的可疑的沙子才会包含物品，种类取决于所属的结构与地物。通过其他方法获取的可疑的沙子没有方块数据，因此没有物品。方块被破坏后方块数据消失，因此物品也会消失。
类似沙子，可疑的沙子上方也可以用来种植仙人掌、竹子、甘蔗和枯萎的灌木，浮空未下落时也会产生 falling_dust 粒子。
版本信息
可疑的沙子在 Java 版 1.19.4 快照23w07a 版本和基岩版 1.19.70 预览版1.19.70.23 版本中加入。
