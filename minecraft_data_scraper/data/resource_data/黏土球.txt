介绍
黏土球（Clay）是一种可以通过开采黏土块所得到的物品。它主要被用于合成黏土块或烧炼制成红砖。
获得方式
黏土球能通过使用没有精准采集附魔的工具开采黏土块获得，1个黏土块可获得4个黏土球。
黏土球有14.3％的概率可以在沙漠村庄屋子的箱子里以每组1个找到，有20.8％的概率可以在村庄的石匠箱子里以每组1-3个找到。
用途
可用于合成黏土块，也可以在熔炉中烧制成红砖。
新手石匠村民会以1个绿宝石的价格购买10个黏土球。
版本信息
黏土球在 Java 版 Alphav1.0.11版本和 携带版 Alpha0.3.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为337。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.16 快照20w06a 版本中，名字从黏土（Clay）变为黏土球（Clay Ball），在 基岩版 1.21.30 预览版1.21.30.22 版本中，英文名从Clay变为Clay Ball以匹配 Java 版。
