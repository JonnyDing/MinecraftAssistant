介绍
玻璃（Glass）是一种视觉透明而可以使光通过的装饰性方块。
获得方式
玻璃可在熔炉中烧制沙子获得。
必须使用带有精准采集附魔的工具破坏玻璃，否则不会掉落任何物品。
老手级别的图书管理员村民会以 1 个绿宝石的价格销售 3-5 个玻璃。
玻璃也会生成在远古城市中心的地下室里。
用途
可用于合成末影水晶、信标、遮光玻璃、染色玻璃、阳光探测器、玻璃瓶等物品。
需要放在固体方块上面的方块大多不能放在玻璃或染色玻璃上。仅有的例外是火把和红石火把，它们可以放在玻璃的上面，但是不能附着到玻璃的侧面。‌‌
透过玻璃看其他邻近的玻璃的时候，邻近的玻璃会隐形。
作为一种固体方块，僵尸增援可以在玻璃顶部生成，导致了意料之外的僵尸“潜入”结构的安全区域。
版本信息
玻璃在 Java 版 Classic0.0.19a 版本和 携带版 Alpha0.1.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为20和95。
在 Java 版 Classic0.0.19a、0.0.19a_01、0.0.19a_01 版本、Java 版 1.14 快照18w43a 版本中修改了纹理。
