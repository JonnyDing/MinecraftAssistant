介绍
平滑石头（Smooth Stone）是石头的变种。
生成
石头会生成在平原村庄、热带草原村庄和雪原村庄的屠夫房屋中。
获得方式
平滑石头也可通过烧制石头获得。
版本信息
平滑石头在 Java 版 Beta1.3 版本和 基岩版 1.9.0 预览版1.9.0.0 版本中加入，此时在Java版只能通过编辑物品栏获得，且名称为无缝双石台阶。
在 Java 版 1.5 快照13w04a 版本中，无缝双石台阶可以通过命令/give <target> 43 1 8获得了。
在 Java 版1.8 版本中，由于双石台阶的物品形式不能通过命令来获得，无缝双石台阶的物品形式也不能通过命令获得了。
在 Java 版 1.13 快照17w47a 版本中，无缝双石台阶改名为平滑石头，同时其物品形式加入了创造模式物品栏内。
在 Java 版 1.14 快照18w43a 版本中，平滑石头可以通过烧炼石头获得。
在 Java 版 1.20 快照23w12a 版本和 基岩版 1.19.80 预览版1.19.80.22 版本中，平滑石头会自然生成在古迹废墟中，此时在基岩版属于实验性玩法。
在 基岩版 1.20.0 预览版1.20.0.21 版本中，1.19.80.22 版本的改动不再属于实验性玩法，而在 Java 版 1.20 快照23w16a 版本和 基岩版 1.20.0 预览版1.20.0.22 版本中，平滑石头不再自然生成于古迹废墟中。
