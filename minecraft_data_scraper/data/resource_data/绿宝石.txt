简介
绿宝石（Emerald）是一种稀有的物品，可以用来和村民交易，绿宝石块可以做信标基座。

掉落
卫道士和唤魔者会掉落 0-1 个绿宝石。
挖掘
绿宝石矿石被铁镐或钻石镐挖掘会掉落 1 个绿宝石。使用时运附魔的镐挖掘会每等级增加 1 可能的绿宝石数量，时运III 最多会掉落 4 个绿宝石。如果使用精准采集附魔的镐采集绿宝石矿石，会掉落绿宝石矿石而不是绿宝石。

版本信息
绿宝石在 Java 版 1.3.1 快照12w21a 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理，在 Java 版 1.14 快照19w08a 版本和 基岩版 1.11.0 预览版1.11.0.5 版本中纹理再次被修改。
在 2012 年 5 月 21 日到 1.3.1 版本，名称为红宝石，之后被蓝宝石替代，原因是 Dinnerbone 是红绿色盲无法清晰的辨认红色和绿色。
红宝石的纹理在 1.3.1 版本到 1.16 版本保留在游戏数据里，在 1.17 版本移出游戏数据，用途是交易货币。
