简介
鳞甲（Scute）是幼年海龟在成年时掉落的物品，用于制作海龟壳。
在 1.20.5 快照23w51a 版本和基岩版 1.20.60 预览版1.20.60.24 版本中重命名为海龟鳞甲（Turtle Scute），注册名更改为minecraft:turtle_scute，因此获取指令更改为：
用途
鳞甲最主要的用途是制作海龟壳，但还有以下几个用途：
专家级别的牧师村民有 2/3（Java 版）或 50%（基岩版）的几率以 1 个绿宝石的价格购买 4 个鳞甲。‌‌
专家级别的皮匠村民以 1 个绿宝石的价格购买 4 个鳞甲。
鳞甲可以用来在铁砧上修复海龟壳。
虽然游戏中海龟鳞甲可以合成海龟壳，但与现实不同的是，现实中水生龟类脱落的鳞甲并非厚实的板块，而是薄如蛇皮的鳞片薄膜。换言之现实中的海龟鳞甲薄膜是很难制作海龟壳的。
版本信息
鳞甲在 Java 版 1.13 快照18w07a 版本和基岩版 1.5.0 预览版1.5.0.4 版本中加入，在 Java 版初次加入时的名称为“Turtle Shell Piece”，在18w07b 版本改为了“Scute”。
