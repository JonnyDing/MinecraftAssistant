介绍
菌丝体（Mycelium）是在蘑菇岛生物群系自然生成的泥土变种，其表面会不断释放出微小的孢子粒子。
生成
菌丝体会自然生成在蘑菇岛生物群系的地表，并有如同孢子散发一般的粒子效果（即使被雪覆盖）。
菌丝体需要用附魔有精准采集的工具开采才能掉落本身，否则只会掉落泥土。
菌丝体和草方块相似，也可以蔓延到泥土方块上。蔓延时菌丝需要9级以上的亮度，而被蔓延的泥土则需要4及以上的亮度，并且不能被任何可阻挡光线的方块或不透明方块覆盖。菌丝会在被可阻挡光线的方块覆盖或亮度低于4的时候变为泥土。如果泥土上有植物，受菌丝蔓延后植物就会成为掉落的物品。
末影人可以像拿起其它可拿起的方块一样拿起菌丝。
版本历史
菌丝体在 Java 版 1.0.0 快照Beta 1.9 pre-1 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入，此时名为菌丝。
在 Java 版 1.14 快照18w43a 版本、1.19 快照22w11a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本、1.19.0 预览版1.19.0.20 版本中修改了纹理。
在 Java 版 1.17 快照21w16a 版本和 基岩版 1.18.0 预览版1.18.0.24 版本中，菌丝体上面可以种植植物了。
在 Java 版1.19.4 版本中，其译名由菌丝改为菌丝体。
