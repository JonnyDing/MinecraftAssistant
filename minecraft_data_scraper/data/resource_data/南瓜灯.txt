介绍
南瓜灯（Jack o'Lantern）是一种可以发光的方块。
获得方式
南瓜灯会自然生成在针叶林村庄的南瓜堆中。
南瓜灯被活塞推动会掉落物品。
也可使用雕刻过的南瓜和火把无序合成。
僵尸、僵尸村民、尸壳、骷髅、流浪者、凋灵骷髅和僵尸猪灵在万圣节期间有概率头戴南瓜灯（或南瓜）生成。使用带有抢夺附魔的工具击杀头戴南瓜灯（或南瓜）的生物时，根据工具的抢夺附魔等级有不同概率掉落南瓜灯（或南瓜），最高为3%。
用途
南瓜灯可建造雪傀儡与铁傀儡。
雪傀儡以竖直放置的2个雪块为作躯干，铁傀儡以T字形摆放的4个铁块作躯干。南瓜灯必须在最后放上，否则不会生成傀儡。
作为光照方块，南瓜灯能提供亮度等级为15的光照。
版本信息
南瓜灯在 Java 版 Alphav1.2.0 版本和 携带版 Alpha 0.8.0 预览版build 1 版本中加入。
在 Java 版 1.11 快照16w39a 版本中，南瓜灯可在铁砧上附上诅咒类附魔，而在 Java 版 1.13 快照17w47a 版本中，南瓜灯不再能在铁砧上附上诅咒类附魔。
在 Java 版 1.13 快照17w47a 版本中，南瓜灯的 ID 从lit_pumpkin变为jack_o_lantern，且在扁平化前，数字 ID 为91。
在 Java 版 1.14 快照18w43a、18w44a 版本、1.17 快照21w11a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
