介绍
末地石（End Stone）是一种构成末地维度的方块。
挖掘用时
末地石的硬度为3，爆炸抗性为45。必须使用镐来开采，否则不会掉落任何东西。其中金镐效率最高，为0.4秒/个。它既不能被末影人捡起来，也不能被末影龙摧毁。
Java 版1.9 版本后，末路之地里会生成附加岛屿，可以制造出接近无限（但不可再生）数量的末地石。
在 Java 版中，末影龙重新生成时，会替换在返回传送门的角落下面的16个末地石，末影龙死亡后会再次变回。由于末影龙可再生，因此末地石也变成了可再生资源。
用途
可用于合成末地石装饰方块，也可用于复制盔甲纹饰锻造模板。
版本信息
末地石在 Java 版 1.0.0 快照Beta 1.9-pre4 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本前，数字 ID 为121。
在 Java 版 1.14 快照18w43a、18w50a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
