简介
下界石英矿石（Nether Quartz Ore）是一个挖掘时会掉落下界石英的矿石方块，会生成于下界。
下界石英矿石需要木镐或更高级的镐来开采它。
生成区域
下界石英矿石以团簇的形式生成于下界Y=10~117 范围间的下界岩之间，矿团大小为1-14个方块。矿团在玄武岩三角洲生物群系之外每区块生成16次，在玄武岩三角洲每区块生成32次。

掉落物
下界石英矿石的掉落物是 1 个下界石英和 2～5 点经验，使用时运附魔镐会掉落更多。使用精准采集附魔的镐开采它会掉落它本身。烧炼一个下界石英矿石会获得 1 个下界石英和 0.2 点经验。
版本信息
下界石英矿石在 Java 版 1.5 快照13w01a 版本和携带版 0.12.1 快照build 1 版本中被加入。
在 Java 版 1.13 快照17w47a 版本的扁平化之前，数字 ID 为 153。
在 Java 版 1.14 快照18w43a、18w50a版本和基岩版 1.10.0 预览版1.10.0.3版本中更改了纹理。
