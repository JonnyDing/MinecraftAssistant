菌光体（Shroomlight）是一种会发光的方块，外观和荧石类似。
菌光体方块会自然生成在下界的绯红巨型菌和诡异巨型菌上，或是在菌岩上用骨粉催熟菌类得到。
菌光体可以用任何工具开采，包括空手，锄能加快开采速度。
菌光体会发出亮度等级为 15 的光，放进堆肥桶时 65% 的概率使堆肥层数加 1。
菌光体上无法生成生物，与南瓜灯不同。
菌光体在 Java 版 1.16 快照20w06a 版本和基岩版 1.16.0 预览版1.16.0.51 版本中加入，
在 Java 版 1.16 快照 20w11a 和基岩版 1.16.0 预览版 1.16.0.57 版本后可以用锄快速破坏，
在 Java 版 1.16 快照 pre3 和基岩版 1.16.100 预览版 1.16.100.55 版本后可用于堆肥。
