简介
可疑的沙砾（Suspicious Gravel）是可供考古的方块，会在古迹废墟或寒带海底废墟中自然生成。
获取
可疑的沙砾受重力影响，但在落地后会直接消失，不会重新成为方块，也不会掉落物品。直接挖掘该方块不会掉落，这会使得物品丢失（精准采集魔咒无效）。
在 Java 版中，可以让可疑的沙砾掉落到蜘蛛网或涡流（上升）气泡柱上，成为下落的方块（此时战利品已经消失）超过 30 秒后（在建筑高度之外为 5 秒）就会变成掉落物，这也是唯一一种获取可疑的沙砾的方式。在基岩版中，只有使用创造模式或命令才能获得该物品。
用途
对可疑的沙砾持续使用刷子刷一段时间过后，方块将变成普通的沙砾，纹理上的裂缝不断扩大，同时掉落一个物品。若中途停止，方块会逐渐恢复原状。被刷子刷扫的一面会露出物品，并且刷扫完毕后会掉落。
可疑的沙砾只能容纳一个物品，存储在方块数据中，只能通过/data命令来修改。自然生成的可疑的沙砾才会包含物品，种类取决于所属的结构与地物。通过其他方法获取的可疑的沙砾没有方块数据，因此没有物品。方块被破坏后方块数据消失，因此物品也会消失。
类似沙砾，可疑的沙砾上方也可以用来种植竹子。
版本信息
可疑的沙砾在 Java 版 1.20 快照23w12a 版本和基岩版 1.19.80 预览版1.19.80.21 版本中加入。
