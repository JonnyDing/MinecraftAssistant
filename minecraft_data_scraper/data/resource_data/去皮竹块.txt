介绍
去皮竹块（Block of Stripped Bamboo）是用任意种类的斧右击竹块产生的对应变种。
用途
可作为熔炉的燃料，每块可以烧炼1.5个物品。
也可用于合成竹板。
版本信息
去皮竹板在 Java 版 1.19.3 快照22w45a 版本和 基岩版 1.19.60 预览版1.19.60.20 版本中加入，此时在Java版和基岩版中属于实验性内容和实验性玩法。
在 Java 版 1.19.3 快照22w46a、pre3版本和 基岩版 1.19.60 预览版1.19.60.22 版本中修改了纹理。
在 Java 版 1.20 快照23w12a 版本和 基岩版 1.20.0 预览版1.20.0.21 版本中，去皮竹块不再属于实验性内容和实验性玩法。
