介绍
雕纹深板岩（Chiseled Deepslate）是深板岩圆石的雕纹变种。
获得方式
可使用深板岩圆石台阶合成获得，也会自然生成在远古城市。
用途
可用于装饰。
版本信息
雕纹深板岩在 Java 版 1.17 快照21w07a 版本和 基岩版  1.17.0 预览版1.16.230.52 版本中加入，此时在基岩版中属于实验性玩法，且名称为錾制阴沉石。
在 Java 版 1.17 快照21w08a 版本中，名称由錾制阴沉石改为錾制深板岩。
在 基岩版 1.17.0 预览版1.17.0.52 版本中，錾制深板岩不再属于实验性玩法。
在 Java 版 1.17 快照21w08a 版本中修改了纹理。
在 Java 版1.19.3 版本中，其译名从錾制深板岩改为雕纹深板岩。
