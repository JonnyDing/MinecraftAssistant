概述
下界合金锭是原版游戏中最稀有的锭，由下界合金碎片和金锭合成获得，用于制造下界合金工具、武器和盔甲。
下界合金锭可用于在锻造台中将钻石制品升级为下界合金制品（保留耐久度、魔咒和自定义名称等 NBT 数据），可以给任意盔甲加上纹饰（不会提升盔甲属性），还可以用于合成磁石。
下界合金锭不会被熔岩和火烧毁，但是仙人掌、爆炸和下落的铁砧依然能够销毁该物品。下界合金制品（包括下界合金碎片和远古残骸）除磁石外都具有此特性。
下界合金锭有概率在下界的堡垒遗迹的藏宝室箱子中找到，但是每个箱子里只有 38.59% 的概率会生成1个，又因为堡垒遗迹的组成部分过于复杂导致不是所有堡垒都有宝藏室，所以实际上你可能跑 5 个堡垒才能找到一个完整的下界合金锭并用找到的下界合金碎片再拼出2个。
下界合金制品可以用下界合金锭在铁砧上修复，一个下界合金锭可以使耐久值恢复约 25%（修复的耐久度向下取整）。
像铁锭、金锭、绿宝石或钻石那样，下界合金锭可用于从信标中选择并激活状态效果。玩家必须选择一种可用的状态效果，然后从物品槽中放入一个下界合金锭。
历史
Java 版 1.16 快照20w16a 版本和基岩版 1.16.0 预览版1.16.0.51 版本加入了下界合金锭。
Java 版 1.19.4 快照23w04a 版本和基岩版 1.19.80 预览版1.19.80.21版本后，下界合金锭可以作为锻造原材料，在锻造台中为盔甲加上纹饰。
