简介
藤蔓（Vines）是一种自然生成的植物，会自然生长。
获取
流浪商人有一定概率以 1 个绿宝石为价格出售 1 个藤蔓。
使用剪刀或附有精准采集的工具采集藤蔓会掉落本体。
生成
藤蔓可以自然生长，在每个方块刻，藤蔓方块有 25% 的概率尝试在随机选择的方向上传播。
藤蔓会在自然生成于丛林和繁茂洞穴生物群系中，也会附着在丛林树、红树、沼泽的橡树、倒下的橡树（基岩版）、倒下的丛林树（基岩版）和覆藤树木（基岩版）上。
藤蔓会作为丛林神庙、林地府邸和掠夺者前哨站的一部分自然生成。
用途
藤蔓是一种植物，当其附着在方块上时，玩家按跳跃键可以攀爬，按住 Shift 键可以悬浮在藤蔓上。
藤蔓会免去摔落伤害。
藤蔓只会附着在完整方块的侧面，当两个完整方块形成夹角时，对着侧面放置也会在底面生成藤蔓。
当藤蔓的附着方块被破坏时，藤蔓也会被破坏，除非上方有其他藤蔓与其相连。
藤蔓可以作为合成材料，参与合成一些物品。
放进堆肥桶时有 50% 的概率使堆肥层数加 1。
在物品栏中，颜色为#48b518，与其他植物一样，在不同的生物群系下，会显示出不同的颜色。
版本信息
藤蔓在 Java 版Beta 1.8-pre1 版本和携带版 Alpha0.9.0 build 1 版本中加入，在 Java 版 1.14 快照18w43a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.8 快照14w07a 版本中，没有连接其他藤蔓的藤蔓显示了错误的纹理，在14w10a 版本中此方块状态的藤蔓会导致游戏崩溃，在14w10b 版本中用南朝向的藤蔓纹理修复了这个特性。
