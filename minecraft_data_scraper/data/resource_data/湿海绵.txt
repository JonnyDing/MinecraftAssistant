介绍
湿海绵（Wet Sponge）是吸过水的海绵。
它的硬度为0.6，可以被徒手挖掘，并掉落自身。
获得方式
击杀远古守卫者必定掉落 1 块湿海绵，不受抢夺附魔影响。
海底神殿有概率生成装有湿海绵的房间，每个房间平均有 30 块湿海绵。
在水旁放置的海绵会转换为湿海绵。
用途
可通过在熔炉中烧制或放置在下界中使其转化成海绵。
湿海绵在熔炉里被烧制成海绵时，如果燃料格里有桶便会将其变成水桶。
成就
版本信息
湿海绵在 Java 版 1.8 快照14w25a 版本和 携带版 Alpha 0.16.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为19。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.20.2 快照23w33a 版本和 基岩版 1.21.50 预览版1.21.50.20 版本中，湿海绵拥有了新的独特音效。
