介绍
砂土（Coarse Dirt）是一种不会长草的泥土的变种。
生成
砂土会在巨型针叶林、被风蚀的恶地/平顶山（岩柱）、热带草原生物群系生成，也可以使用2个泥土和2个沙砾合成得到4个。
砂土也会在古迹废墟中随机替换掉原有的沙砾。
挖掘用时
砂土能通过使用任何工具或手挖掘并掉落成物品，但使用锹挖掘最快。效率最高的是钻石锹或金锹，均为0.1秒/个。
用途
把树苗、甘蔗、蘑菇直接放在砂土上，在合适的条件下就会生长。草方块和菌丝无法扩散至砂土上。对砂土使用锄就能把它变为泥土。
对砂土使用水瓶会将其变为泥巴。
版本信息
砂土在 Java 版 1.8 快照14w25a 版本和 携带版 Alpha0.9.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为3。
在 Java 版 1.8 快照14w25a 版本中修改了纹理。
