介绍
末地石砖（End Stone Bricks）是末地石的石砖版本。
获得方式
末地石砖可以通过4个末地石合成得到，也可以在末地城里找到。
末地石砖的硬度为0.8，爆炸抗性为4。它可以被任何镐所开采。若不用镐开采，将不会掉落任何的东西。其中，金镐的效率最高，为0.1秒/个。
用途
可用于装饰。
版本信息
末地石砖在 Java 版 1.9 快照15w31a 版本和 携带版 1.0.0 预览版0.17.0.1 版本中加入。
在 Java 版 1.13 快照17w47a 版本中，末地石砖的ID由end_bricks更改为end_stone_bricks，且在扁平化前，数字 ID 为206。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
