介绍
潜影壳是潜影贝的掉落物。
获得方式
在Java 版中，潜影贝有 50% 概率掉落 1 个潜影壳，使用带有抢夺魔咒的武器则每等级增加 6.25% 掉落概率（抢夺III最多为 4 个）。
在基岩版中，潜影贝会掉落 0～1 个潜影壳，使用带有抢夺魔咒的武器则每等级增加 1 个掉落量上限（抢夺III最多为 4 个）。
用途
可用于合成潜影盒。
版本信息
潜影壳在 Java 版 1.11 快照16w39a 版本和 携带版 1.0.0 预览版1.0.0.1 版本中加入。
在 Java 1.13 快照17w47a 版本扁平化前，数字 ID 为450。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改纹理。
在 Java 版 1.17 快照20w45a 版本和 基岩版 1.18.30 预览版1.18.30.22 版本后，潜影贝被其它潜影贝导弹击中有概率生成新的一只潜影贝，潜影壳成为可再生资源。
