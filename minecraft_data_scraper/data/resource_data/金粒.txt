简介
金粒（Gold Nugget）是一种较为珍贵的合成材料。
获取
金粒可以作为生物掉落物获取：
僵尸猪灵死亡时会掉落 0-1 个金粒，受抢夺魔咒影响，每增加 1 级，掉落上限增加 1，抢夺III 掉落 0-4 个。
金粒可以通过挖掘以下方块获取：
使用未附有精准采集魔咒的镐开采：
下界金矿石会掉落 2-6 个金粒。受时运魔咒影响，可增加掉落数量，时运III 最多掉落 24 个金粒。
镶金黑石有 10% 的概率掉落 2-5 个金粒。受时运魔咒影响，时运I 使得掉落金粒的概率提升至 14.29%（1/7），时运II 25%，时运III 100%。
金粒可以通过金锭在工作台中分解得到。
可通过冶炼金工具、金剑、金套、金马铠的方式获得铁粒。

用途
金粒是一种合成材料，可以参与合成一些物品。
金粒可以合成出效果为星型的烟花火箭。
版本信息
金粒在 Java 版 1.0.0 预发布版Beta 1.9-pre1 版本和携带版 Alpha0.12.1 build 1 版本中加入，在 Java 版 1.14 快照18w43a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
