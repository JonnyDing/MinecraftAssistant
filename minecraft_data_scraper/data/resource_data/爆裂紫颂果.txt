爆裂紫颂果（Popped Chorus Fruit）是一种可以由烧炼紫颂果而获得的物品，它可以用来合成紫珀块和末地烛，但不可食用。

基岩版同样更改了命名空间 ID，对应1.16.100.56 版本。
爆裂紫颂果在 Java 版 1.9 快照15w31a 版本和携带版 1.0.0 预览版0.17.0.1 版本加入，在 1.14 快照18w43a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改纹理。
