简介
铁锭（Iron Ingot）是一种用途广泛的金属合成材料。
获得方式
可通过冶炼铁矿石、深层铁矿石或粗铁获得。
烧炼铁矿石、深层铁矿石或粗铁是获得铁锭最常见的方法。
同时，可以在以下结构中可以找到铁锭。
（表格数据仅限 Java 版，按照获取概率大小降序排序。数据来自中文 Minecraft Wiki）
铁傀儡死亡时掉落 3-5 个铁锭。抢夺附魔不会增加掉落数量。
僵尸有约 0.833% 的概率掉落一个铁锭，每等级抢夺魔咒使得几率提高 1%。
破坏试炼密室走廊的饰纹陶罐也有概率获得铁锭。
用途
学徒级别的盔甲匠、工具匠和武器匠村民会以 1 个绿宝石的价格收购 4 个铁锭，作为其第三项交易。（仅基岩版）
学徒级别的盔甲匠村民有 50% 的概率以 1 个绿宝石的价格收购 4 个铁锭；学徒级别的工具匠和武器匠总是以 1 个绿宝石的价格收购 4 个铁锭，作为其交易的一部分。（仅 Java 版）
铁头盔、胸甲、护腿、靴子、剑、镐、斧、锄和锹，以及锁链头盔、胸甲、护腿和靴子能在铁砧用铁锭来修复。
铁锭可以用于从信标中选择并激活状态效果。玩家必须选择一种可用的状态效果，然后在项目槽中放入一个铁锭。铁锭可以代替金锭、绿宝石、钻石或下界合金锭为信标选择状态效果。
铁傀儡使用 1 个铁锭能恢复它25点（× 12.5）的生命值（仅 Java 版）
版本信息
铁锭在 Java 版 Indev 0.31 快照20100128 版本和 携带版 Alpha0.3.2 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为265。
在 Java 版 Indev 0.31 快照20100129、1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
