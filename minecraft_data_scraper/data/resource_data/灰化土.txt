介绍
灰化土（Podzol）是一种泥土变种。
生成
灰化土只会在针叶林或竹林生物群系生成。
获得
灰化土能通过使用任何工具或手挖掘并掉落成物品，但使用铲子挖掘最快。它只能通过精准采集附魔来得到，否则，只会掉落泥土。
灰化土和菌丝一样允许蘑菇在它上面生长并且不受亮度限制，从而允许巨型蘑菇的生长。树苗、所有花、甘蔗都可以在上面生长。
版本信息
灰化土在 Java 版 1.7.2 快照13w36a 版本和 携带版 Alpha0.9.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本前，数字 ID 为3。
在 Java 版 1.13 快照18w06a 版本和 基岩版 1.16.0 预览版1.15.0.51 版本中，大型云杉树长成后会将它周围的草方块、泥土和砂土转变为灰化土，这使得灰化土可再生。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.14 快照19w11a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中，可通过与流浪商人交易获得灰化土。
在 Java 版 1.20.3 快照23w45a 版本和 基岩版 1.20.60 预览版1.20.60.20 版本中，灰化土可以生成在试炼密室中使用剧毒之箭的骷髅和生成洞穴蜘蛛的试炼刷怪笼旁，此时在Java版和基岩版均属于实验性内容和实验性玩法。
在 Java 版 1.21 快照24w18a 版本和 基岩版 1.21.0 预览版1.21.0.22 版本中，23w45a 版本和1.20.60.20 版本的改动不再属于实验性内容和实验性玩法。
