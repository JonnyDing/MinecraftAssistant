介绍
鸡蛋（Egg）是一种可以由鸡产下的物品。
获得方式
每隔5-10分钟，成年鸡会下一颗蛋。
鸡蛋也可以在村庄的制箭师小屋的箱子里找到。
叼着鸡蛋的狐狸死亡时有8.5%掉落鸡蛋。（仅JAVA版）
试炼密室中决斗室的发射器装有鸡蛋。
用途
可用于制作蛋糕和南瓜派。
也可用于投掷，玩家能对其他玩家和生物丢鸡蛋并造成击退，但不会造成伤害。对中立型生物丢鸡蛋会激怒它们，鸡蛋也能从发射器发射并被重力影响。鸡蛋被丢到方块上时会有31/256的概率生成1只小鸡，有1/256的概率生成4只小鸡。
版本信息
鸡蛋在 Java 版 Alphav1.0.14 版本和 携带版 Alpha0.4.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为344。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 快照1.10.0.3 版本中修改了纹理。
