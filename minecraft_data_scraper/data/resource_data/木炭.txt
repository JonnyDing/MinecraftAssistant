介绍
木炭（Charcoal）是煤炭的替代品，主要通过烧制原木或木头获得，用途和煤炭基本相同。
获得方式
木炭可以通过在熔炉里烧炼木头获取。
使用没有精准采集附魔的工具破坏营火会掉落 1-2 个木炭。
用途
一个木炭可以烧炼 80 秒（即支持燃烧 8 个物品）。木炭作为熔炉的燃料是木板或木头持续时间的 5 倍。木炭也是动力矿车的动力源之一，一个木炭可提供约 4 分钟的动力。
也可用于合成火把和火焰弹。

版本信息
木炭在 Java 版 Beta1.2 版本和 携带版 Alpha0.3.0 版本中加入，此时木炭使用的是煤炭的纹理。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为263。
在Java版 1.6.1 快照13w19a、1.14 快照18w43a和 携带版 Alpha0.8.1、1.10.0 预览版1.10.0.3 版本中修改了纹理。
