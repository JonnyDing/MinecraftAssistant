介绍
海晶砂粒（Prismarine Crystals）是一种可以用来合成海晶灯的可再生的稀有道具。
获取方式
海晶砂粒在击杀守卫者、远古守卫者时分别有40%和1⁄3的概率掉落。抢夺魔咒每提升一级，最大掉落量就提高一个。
当使用没有精准采集附魔的工具采集时，海晶灯会掉落2-3个海晶砂粒。带有时运附魔的工具每提升一级，采集海晶灯时掉落的海晶砂粒数量就会同时增加一个至最多5个。

版本信息
海晶砂粒在 Java 版 1.8 快照14w25a 版本和 携带版 Alpha 0.16.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照 17w47a 版本的扁平化之前，数字 ID 为410。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
