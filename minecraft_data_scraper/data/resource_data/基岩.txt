简介
基岩（Bedrock）是一种在生存模式中无法破坏的方块（但某些模组的道具可以破坏它）。
1.14快照18w50a前，基岩使用的是高对比度的石头的纹理，是 Minecraft 中硬度最大（在代码中它的硬度是 -1，在 2022 年的愚人节版本中硬度是 600）、爆炸抗性第 2 大的方块（爆炸抗性是 3600000，爆炸抗性最大的方块是屏障，爆炸抗性是 3600000.8）。它的主要作用是防止玩家或生物掉入虚空或进入下界世界的上方，也防止生物生成。它也是唯一信标和末地折跃门的光束能穿透的固体方块，这使得信标在下界可以使用。
它有可能生成在地图最底部的五层，但一般在最底部的第四层就全部由没有任何间隙存在的基岩组成，这使得最底部的第五层无法到达。下界中底部和顶部四层都存在基岩。在下界的顶部，基岩防止玩家到达高度 127 以上，它也能在主机版下界的边界找到。无论使用了什么种子，在主世界和下界的基岩总以相同模式生成。（仅 Java 版）（Java 版 1.18快照21w40a 及以后，主世界的基岩分布和种子有关，但是在下界其分布仍然是固定的。）
它不能被原版任何工具破坏，也不能被活塞推动或被正常游戏的爆炸破坏。原版中基岩只能通过创造模式物品栏或使用命令获得。
在末路之地的基岩上点燃火（包括床和重生锚的爆炸），火会像在地狱岩上面一样不会熄灭。
有趣的是，虽然在原版生存模式中没有正常的破坏方法，但是可以利用某些游戏中的 bug 来破坏它，以达到一些特殊的目的。尽管绝大多数的这些手段都只能破坏、而不能获得它。
可以利用末影珍珠和梯子穿过下界的顶层基岩。
尽管如此，该物品仍然可以在原版生存模式下用某些特殊手段获得，原理是利用粘性活塞的推动判定（该方法适用于 Java 版的 1.8 版本）。
在极其罕见的情况下，可能会发现包裹在基岩中的矿石。
基岩可以被自然生成的要塞和废弃传送门破坏。
恼鬼和末影龙可以穿过基岩。
指令可以破基岩，但是在基岩版中才会掉落。
在 1.13 扁平化之前，数字 ID 为 7。
冷知识
在 Java 版 1.14 纹理更新之前，基岩和石头的纹理是一样的，不过对比度更高，所以颜色区别更大；
原版游戏中只有 5 种方块存在于三个维度之中，而基岩是其中一种（其余为黑曜石，铁栏杆，箱子和空气）；
基岩版中有极小概率碰到废弃传送门生成在峡谷中，而恰好峡谷底部接近基岩层，这时下方基岩可能会被下界岩取代。
版本信息
基岩在 Java 版 Classic 0.0.12a 中加入，在1.14快照18w50a中修改纹理。
