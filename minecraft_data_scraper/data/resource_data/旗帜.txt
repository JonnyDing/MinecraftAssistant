旗帜在 Java 版 1.8 快照 14w30a 中加入。
旗帜是一种装饰性方块，可以通过工作台或织布机向旗帜添加图案（消耗染料，部分图案需要特殊物品。例如花朵图案，在工作台添加时需额外添加滨菊，而在织布机添加时需要花朵旗帜图案），同一面旗帜可添加多个图案，新图案直接覆盖在旧图案上方，最多叠加 6 层图案， 图案数量＞6 个的旗帜仅能通过指令获取。
用织布机向旗帜添加图案时，左侧会出现各种图案的小图标，选中所需图案样式，并向右上方的格子中依次放入旗帜、染料、旗帜图案（非必要），右侧图案效果，从右下方输出成品；用工作台向旗帜添加图案时，需将多个染料摆成所需图案进行有序合成，不同的摆放方式对应不同的图案样式。
用有图案的旗帜右键装水的炼药锅可以洗去最后一次添加的图案，每次洗涤都会使炼药锅中的水会减少一层。
旗帜可以与盾牌合成，以此在盾牌上附加图案。（附加的图案会被缩小，因此在一定程度上失真）。
用地图右键旗帜，可以在地图上添加一个标记，标记颜色与旗帜底色相同，若旗帜在铁砧中重命名过，标记将显示相应的名称。若旗帜方块被破坏，未经制图台锁定的地图将同步移除该旗帜的标记。
有图案的旗帜与另一底色相同的空白旗帜无序合成时（此处“底色”指最初合成旗帜时使用的羊毛的颜色），可以把图案直接复制到空白旗帜上，无需消耗染料。
作为燃料在熔炉中使用时，每个旗帜可以燃烧 1.5 个物品。
灾厄旗帜是一类特殊的旗帜，可以被灾厄村民携带。
在 Java 版中，灾厄旗帜是一个有总共 8 个图案的白色旗帜。
低配电脑请谨慎使用，因为旗帜属于实体方块，占用性能。
