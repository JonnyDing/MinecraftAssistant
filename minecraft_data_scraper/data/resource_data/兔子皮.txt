介绍
兔子皮（Rabbit Hide）是一种兔子的掉落物。
获得方式
杀死兔子时会掉落 0-1 个兔子皮，抢夺附魔会每等级增加 1 掉落数量，使用抢夺III 会掉落 0 至 4 个兔子皮。
已驯服的猫会与玩家一同睡觉，并在玩家醒来时有 16.129% 的概率掉落 1 个兔子皮。
钓鱼时兔子皮可作为垃圾被钓上来。
嘴里叼着兔子皮的狐狸被杀死后有 8.5%‌‌ 的概率掉落兔子皮。玩家也可向狐狸丢出食物，让它吐出嘴里的兔子皮。
用途
可用于合成皮革。
老手级别的皮匠村民会以 1 个绿宝石的价格购买 9 个兔子皮。
版本信息
兔子皮在 Java 版 1.8 快照14w27a 版本和 携带版 Alpha 0.13.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为415。
在 Java 版 1.8 快照14w33b、1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
