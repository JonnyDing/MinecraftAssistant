概述
铜锭（Copper Ingot）是通过烧炼铜矿石、深层铜矿石和粗铜获得的金属锭。
溺尸被玩家杀死时有 11% 的概率掉落铜锭，每一级抢夺魔咒增加 2% 的掉落概率（抢夺 III 最多为 17%）。
可以在工作台中合成望远镜、避雷针、铜块、刷子、铜门和铜活板门，可以在锻造台中给任意盔甲加上纹饰（不会提升盔甲属性）。
历史
Java 版 1.17 快照20w45a 版本和基岩版 1.16.210 预览版1.16.210.57 版本中加入铜锭。
Java 版 1.17 快照20w45a 版本中，9 个铜锭可以合成 1 个铜块，1 个铜块可以分解为 9 个铜锭。
Java 版 1.17 快照20w46a 版本和基岩版 1.16.210 预览版1.16.210.57 版本中，4 个铜锭可以合成 1 个铜块，1 个铜块可以分解为 4 个铜锭。
Java 版 1.17 快照21w17a 版本和基岩版 1.17.0 预览版1.17.0.54 版本后，9 个铜锭可以合成 1 个铜块，1 个铜块和涂蜡铜块可以分解为 9 个铜锭。
