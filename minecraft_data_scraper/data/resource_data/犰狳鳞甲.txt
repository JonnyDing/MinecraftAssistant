介绍
犰狳鳞甲（Armadillo Scute）是由犰狳产出的物品，可以用于合成和在铁砧上修复狼铠。
获得方式
可使用刷子右击成年犰狳获得（该过程会消耗刷子 16 点耐久）（也可以用发射器），犰狳每隔 5～10 分钟也会掉落一个犰狳鳞甲。
玩家在狼坐下时，使用犰狳鳞甲可以修补受损的狼铠，每个犰狳鳞甲可以增加 8 耐久度。
版本信息
犰狳鳞甲在Minecraft Live 2023中展示了最初设计的纹理，在 Java 版 1.20.5 快照23w51a 版本和基岩版1.20.60.23 版本中加入。
在 Java 版24w03a 版本和 基岩版1.20.70.20 版本中修改了纹理。
在 Java 版24w09a 版本和 基岩版1.20.80.22 版本后，可用于修复狼铠。
