介绍
磨制安山岩（Polished Andesite）是安山岩的磨制变种。
生成
磨制安山岩会生成在雪屋的地下室红色地毯下、林地府邸和试炼密室的补给箱底座。
获得方式
磨制安山岩也可通过使用安山岩合成或在切石机切石获得。
老手级别的石匠村民有2/7（Java版）或25%（基岩版）的概率以 1 个绿宝石的价格出售 4 个磨制安山岩。
用途
可用于装饰。
版本信息
磨制安山岩在 Java 版 1.8 快照14w02a 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为1。
在 Java 版 1.14 快照18w44a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
