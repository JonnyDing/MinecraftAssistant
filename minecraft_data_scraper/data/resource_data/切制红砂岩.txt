介绍
切制红砂岩（Cut Red Sandstone）是红砂岩的切制变种。
获得方式
可通过使用红砂岩合成获得。
用途
可用于装饰。
版本信息
在 Java 版 1.8 快照14w32a 版本和 携带版 Alpha 0.14.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本和 基岩版 1.9.0 预览版1.9.0.0 版本中，平滑红砂岩（Smooth Red Sandstone）被重命名为切制红砂岩（Cut Red Sandstone）。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为179。
在 Java 版 1.14 快照18w43a、18w47a、18w50a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
