介绍
磨制花岗岩（Polished Granite）是花岗岩的磨制变种。
生成
磨制花岗岩会生成在海底废墟和沙漠村庄的盔甲匠小屋中。
获得方式
磨制花岗岩也可通过使用花岗岩合成或在切石机上切石获得。
老手级别的石匠村民有2/7（Java版）或25%（基岩版）的概率出售 4 个磨制花岗岩。
用途
可用于装饰。
版本信息
磨制花岗岩在 Java 版 1.8 快照14w02a 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为1。
在 Java 版 1.14 快照 18w43a 版本和 基岩版 1.10.0 预览版 1.10.0.3 版本中修改了纹理。
