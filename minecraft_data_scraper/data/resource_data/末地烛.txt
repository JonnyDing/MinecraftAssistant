简介
末地烛（End Rod）是一种装饰性的可提供光源的方块。
自然生成
末地烛会自然生成在末地城中的一些房间。
获取
末地烛可以通过合成获取。
用途
末地烛主要用于装饰和提供光源，会发出亮度等级为 14 的光。
末地烛本身具有碰撞箱。
末地烛可以放置在任何方块的任意面，不会因其附着的方块被破坏而掉落，也就是可以悬空。
末地烛的顶端朝上时，下落的方块落在上面不会被破坏，但当下落的方块落在侧面时会被破坏。
在末地城里，末地烛被用于形成一种螺旋楼梯供玩家达到顶端。
末地烛可以被活塞推动和黏性活塞拉动。
版本信息
末地烛在 Java 版 1.9 快照15w31a 版本和携带版 1.0.0 预览版0.17.0.1 版本中加入，在 Java 版 1.14 快照18w43a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
