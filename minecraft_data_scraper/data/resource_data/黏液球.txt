介绍
黏液球（Slimeball）是掉落于史莱姆的物品。
获得方式
小型史莱姆被杀死时会掉落0-2个黏液球，每1级抢夺魔咒都会增加1个最大掉落量。小型史莱姆被青蛙杀死时固定掉落1个黏液球。
幼年熊猫打喷嚏时有0.1%的概率掉落黏液球。
流浪商人有8.06%（Java版）或21.74%（基岩版）的概率以4个绿宝石的价格出售1个黏液球。
用途
可用于合成黏液块、黏性活塞等物品，也可以用来繁殖青蛙并加速蝌蚪的成长。
版本信息
黏液球在 Java 版 Alphav1.0.11 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为341。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
