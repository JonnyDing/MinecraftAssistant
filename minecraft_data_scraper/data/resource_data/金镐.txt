挖掘等级：木镐 =金镐< 石镐 < 铁镐 < 钻石镐 < 下界合金镐
挖掘速度：木镐 < 石镐 < 铁镐 < 钻石镐 < 下界合金镐 <金镐
看似十分高端，但它实际的挖掘等级却和木镐相同（0级，Stone/石头级），只能挖掘石头、煤矿石/制品以及混凝土、部分石制方块。
同时它还是耐久最低的挖掘工具，33点耐久甚至只有木镐的一半。
不过作为金制工具，它能在原方块硬度的基础上提供原版最高的x12倍挖掘速度加成（与空手相比），比钻石镐（x8）还要高出二分之一。这意味着金镐是原版最快的挖掘工具。
并且它也拥有原版最好的22点附魔偏移值，这意味着金镐能轻而易举地附出吊打30级的神附魔。举个例子，精准采集附魔在铁镐上需要30级才有5%几率附出，而金镐在任意等级都有附出精准采集的可能。
所以如果使用得当，它完全可以作为钻石镐的替代品。
附魔金镐可以在废弃传送门的箱子中找到。
