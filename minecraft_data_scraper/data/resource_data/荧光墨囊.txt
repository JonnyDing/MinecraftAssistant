介绍
荧光墨囊可以击杀发光鱿鱼获得，正常情况下掉落 1～3 个。使用带有抢夺魔咒的武器则每等级增加 1 个掉落量上限（抢夺III最多为 6 个）。
用途
可对着告示牌使用以增亮文本，使文本带有外边框（颜色取决于文本颜色）并同时达成进度“眼前一亮”。此时对着告示牌使用墨囊可以移除发光效果。
也用于合成荧光物品展示框。
版本信息
荧光墨囊在 Java 版 1.17 快照21w03a 版本和基岩版 1.16.210 预览版1.16.210.59 版本首次加入（在基岩版 1.16.210 预览版1.16.210.60 版本移除，但是在基岩版 1.16.220 预览版1.16.220.50 版本重新加入）。
在 Java 版 1.17 预发布版1.17-pre1 版本和基岩版 1.17.10 预览版1.17.10.21 版本后，荧光墨囊可对着告示牌使用以增亮文本。
在 Java 版 1.17.1 预发布版1.17.1-pre1 版本和基岩版 1.17.30 预览版1.17.30.20 版本后，僵尸、僵尸村民、尸壳和溺尸不再捡起荧光墨囊。
