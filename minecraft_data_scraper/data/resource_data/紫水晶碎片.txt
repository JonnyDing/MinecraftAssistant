概述
紫水晶碎片是开采紫水晶簇掉落的矿物。
使用镐（并且不带精准采集魔咒）挖掘紫水晶簇可以获得 4 个紫水晶碎片。使用带有时运附魔的镐则每等级增加 4 个掉落量上限（时运III最多为 16 个）。使用其他方法破坏紫水晶簇只能获得 2 个紫水晶碎片。
远古城市远的箱子中有 23.24% 的概率生成 1～15 个紫水晶碎片，试炼密室的交叉口有 39.72% 的概率生成 8～20 个紫水晶碎片。
可以用于合成望远镜、遮光玻璃、紫水晶块和校频幽匿感测体，可以在锻造台中给任意盔甲加上纹饰（不会提升盔甲属性），也可以给跳舞中的悦灵以在其所在位置生成另一只悦灵。
历史
在 Java 版 1.17 快照20w45a 版本和基岩版 1.17.0 预览版1.17.0.50 版本中加入。
在 Java 版 1.17 快照20w46a 版本中修改纹理。
在 Java 版 1.19.1 快照22w24a 版本和基岩版 1.19.0 预览版1.19.0.26 版本之后，跳舞中的悦灵获得紫水晶碎片时，其所在位置会生成另一只悦灵。
