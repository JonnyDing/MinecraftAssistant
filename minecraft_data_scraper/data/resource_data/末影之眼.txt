介绍
末影之眼（Eye of Ender/Ender Eye）是一种用于定位并激活要塞里面的末地传送门的可再生道具。它可通过合成获得。
Java 版 1.3.1 快照12w22a 版本至 1.9 快照15w41a 版本中，可从牧师村民那里购买末影之眼。
用途
末影之眼可以用于定位并激活末地传送门。
12个末影之眼可以激活末地传送门，每个末地传送门框架在生成时有10%的概率会被放置末影之眼。对着末地传送门框架按下使用键（默认为鼠标右键）就可以放置末影之眼，一旦12个全部放满后，末地传送门就会激活。
合成末影箱和（1.9及之后）末影水晶。
定位要塞的过程（以及传送门所在的房间）
手持末影之眼并按下使用键（默认为鼠标右键），末影之眼会飞向最近的要塞，穿过必要的方块，并在其经过的地方留下一条带有紫色颗粒的痕迹。
（仅限PC版）末影之眼会飞向世界里生成的第一个要塞。如果退出并重新加载，那么它就会飞向生成的第一个螺旋楼梯。
（仅限主机版）末影之眼只会飞向要塞，而不是在要塞里面的末地传送门。
如果距离末地传送门很远，末影之眼就会飞得更高，以便在垂直方向上更明确指示玩家必须通过的路径。
当玩家接近末地传送门所在的纵向坐标轴时，末影之眼会更精准的飞行来与定位这条 Y轴。这也就意味着末影之眼将会向下飞行，指示出玩家在末地传送门的上方，让玩家向下挖。
在空中飘浮2-3秒后，末影之眼就会掉落，变成可拾获的物品，或者在空中碎掉。有1/6的几率会碎掉。
末影之眼的飞行功能只在主世界工作，在下界、末路之地、超平坦模式的主世界以及没有末地传送门的世界中不会工作。
版本信息
末影之眼在 Java 版 1.0.0 快照Beta 1.9 pre-3版本和 携带版 1.0.0 预览版0.17.0.1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为381。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改纹理。
