简介
青蛙卵（Frogspawn）是一种可以孵化出蝌蚪的方块。

生成
给两只青蛙喂食黏液球使之交配后，其中一只青蛙会搜寻上方有空气的水（包括流动的），并在其上方产下青蛙卵。
获取
在生存模式中无法使用不作弊的方法获取青蛙卵，因为它被破坏后不会掉落任何物品，因此只能通过创造模式物品栏、鼠标中键选取和/give指令获取。
用途
青蛙卵用于孵化蝌蚪，孵化时间约为 3-10 分钟，每个青蛙卵会孵出 2-5 只蝌蚪。
青蛙卵可以放置在水或含水方块上，会被下落的方块破坏，活塞推动时也会破坏青蛙卵，无法被黏性活塞拉动。
版本信息
青蛙卵在 Java 版 1.19 快照22w11a 版本和基岩版 1.18.10 预览版1.18.10.24 版本中加入，在基岩版 1.18.30 预览版1.18.20.21 版本中，英文名由“Frog Egg”更改为“Frog Spawn”，注册名由minecraft:frog_egg更改为minecraft:frog_spawn，并修改了纹理。在基岩版 1.19.0 预览版1.19.0.24 版本中，英文名由“Frog Spawn”更改为“Frogspawn”，但并没有改变注册名。
