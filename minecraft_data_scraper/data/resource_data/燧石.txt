介绍
燧石是是合成箭与打火石的重要道具。
获得方式
挖掘沙砾有 10% 的概率会掉落 1 个燧石，使用时运附魔工具挖掘会提高掉落燧石的概率。

使用精准采集附魔工具挖掘沙砾只会掉落沙砾，而不会掉落燧石。
新手级别的制箭师村民有 2/3 ‌‌的概率以 1 个绿宝石和 10 个沙砾出售 10 个燧石。
用途
学徒级别的皮匠村民在 Java 版 中有 2/3 的概率以 1 个绿宝石的价格购买 26 个燧石，而在基岩版这项交易一定会出现。
老手级别的工具匠村民在 Java 版 中有 40% 的概率以 1 个绿宝石的价格购买 30 个燧石，而这项交易在基岩版中一定会出现。
老手级别的武器匠村民会以 1 个绿宝石的价格购买 24 个燧石。
版本信息
燧石在 Java 版 Indev 0.31 快照20100130 版本和 携带版 Alpha0.3.3 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为318。
在 Java 版 1.16 快照20w07a 版本中，燧石可以通过和猪灵以物易物获得，而在 Java 版 1.16 快照20w09a 版本中，燧石不再能通过猪灵以物易物获得。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改纹理。
