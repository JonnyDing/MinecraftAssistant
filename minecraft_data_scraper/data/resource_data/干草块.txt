介绍
干草块（Hay Bale）是一种可以储存小麦的方块。
生成
干草块可以在村庄中自然生成，也会充当掠夺者前哨站的标靶/稻草人生成。
干草块的硬度为0.5，能被任何道具或徒手采集，且速度均为0.75个/秒。
用途
干草块虽然无法被玩家食用，但比起其他食物来源可作为支持长期生活的高效压缩干粮。干草块也能喂给马、驴或骡以恢复20点（）生命值。在玩家想在短时间内治疗马匹时这会是十分有效的治疗方法。干草块也能用来为马驹成长加快 3 min的真实时间。
干草块可以像木头一般侧向放置。
干草块也可用于繁殖羊驼或合成标靶。
Java 版 1.9 快照 15w44b 版本后，生物掉落到干草块仅会受到一般情况下20%的掉落伤害。
在营火下方放置干草块可使其冒出25个方块的浓烟，而非10个。
版本信息
干草块在 Java 版 1.6.1 快照13w16a 版本和 携带版 0.8.0 预览版build 1 版本中加入，此时Java版的干草块没有名字，在更新日志中被称为Hay Bale。
在 Java 版 1.6.1 快照13w16b 版本中，干草块添加了标签，英文名为Hay Block，而在 Java 版 1.6.1 快照13w24a 版本中，英文名从Hay Block改为Hay Bale。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为170。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
