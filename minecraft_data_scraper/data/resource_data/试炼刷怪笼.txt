试炼刷怪笼（Trial Spawner）是一种只能在试炼密室中生成的刷怪笼变种，会根据周围玩家的数量调整刷怪总数。
试炼刷怪笼自然生成于试炼密室的柱廊和决斗室结构中。
当游戏难度不是和平，且游戏规则 doMobSpawning 为 true、处于冒险模式或生存模式的带有试炼之兆的玩家经过普通试炼刷怪笼时，试炼刷怪笼会变为不祥试炼刷怪笼。不祥试炼刷怪笼
激活的试炼刷怪笼每隔一段时间生成 1 个生物，48 格内同时存活的生物数量达到规定的上限后会停止生成，这些生物都消失后才会生成出下一波生物。一次试炼中生成的生物全部消失后，试炼刷怪笼结束运作，试炼完成。
不祥试炼刷怪笼生成的怪物一般装备着武器、盔甲等。
激活的不祥试炼刷怪笼每 160游戏刻（8 秒）在玩家或距离不祥试炼刷怪笼 14 格内的生物头顶生成不祥之物生成器。
试炼刷怪笼在 Java 版 1.20.3 快照 23w45a 和基岩版 1.20.60.20 版本加入，在 Java 版 1.20.5 pre1 和基岩版 1.21.0.22 版本中更改了纹理。
不祥试炼刷怪笼在 Java 版 1.20.5 快照 24w13a 和基岩版 1.21.0.20 版本中加入。
