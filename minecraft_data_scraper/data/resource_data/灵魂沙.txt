介绍
灵魂沙（Soul Sand）是自然生成在下界、能够减慢实体移动速度的方块。
生成
1.16前
灵魂沙会自然生成在下界Y＜65 的地方或者下界要塞的地狱疣房间里。
1.16后
灵魂沙会在灵魂沙峡谷作为地表层与灵魂土大量混合，在灵魂沙峡谷以外的下界生物群系，灵魂沙则以 1 - 12 个方块大小的团簇生成于Y ≤ 31 的地方，每区块尝试生成12次团簇。
灵魂沙也会在下界要塞的下界疣房、堡垒遗迹的藏宝室和疣猪兽棚、远古城市的中心和营房中生成。
用途
灵魂沙会降低所有站在其上方的实体（生物和玩家）的移动速度，穿上带有灵魂疾行附魔的靴子可避免此效果（并且还会让你加速）。并导致他们在灵魂沙上面行走时下沉 2 个像素。当灵魂沙被放置在水里或冰、浮冰或粘液块上时，这种效果会更加显著。它也可以导致尺寸小于一个台阶的生物窒息，也就是蠹虫、小海龟和末影螨。
灵魂沙可以用于种植下界疣以及制造凋灵。
也可用于合成灵魂火把和灵魂营火。
版本信息
灵魂沙在 Java 版 Alpha v1.2.0 快照preview 版本和 携带版 Alpha 0.12.1 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为88。
在 Java 版 1.13 快照18w07a 版本和 基岩版 1.5.0 预览版1.5.0.4 版本中，将灵魂沙放置在水中会产生一个上涌的气泡柱，它会快速让物品、玩家和生物升起，这个效果只发生在静止的水与灵魂沙直接接触才会发生。
在 Java 版1.16 版本中，气泡柱会正常影响弹射物的运动。
在 Java 版 1.16 快照20w07a 版本中，灵魂沙可通过与猪灵以物易物获得，在 Java 版 1.16 快照20w09a 版本中取消了这一设定，而在 Java 版 1.16 快照20w10a版本中重新加回了这一设定。
在 Java 版 1.14 快照18w43a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
