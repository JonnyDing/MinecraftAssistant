下界要塞是一种在下界生成的一种建筑，在1.0.0快照Beta 1.9-pre1中加入。
下界要塞的建筑群倾向于以“长条”的形式生成，在南北方向（Z轴）延伸。在X轴（东西方向）寻找最有可能找到下界要塞；若在南北方向寻找则可能要走数百格才能找到下界要塞。有时下界要塞会埋在地狱岩中，使玩家难以进入。在这种情况下，露天走道和桥梁会被覆盖，而室内不会堵塞。有时荧石会在室内生成，但这种情况非常少见。
下界要塞使用一个独有的生物生成列表，其中的生物生成规律与下界的其他地方有差异。下界要塞中岩浆怪的生成几率很高，并且还有两种特有生物：烈焰人和凋灵骷髅。生成骷髅时，也有20%几率生成一个普通骷髅而不是凋灵骷髅。恶魂不在下界要塞中生成。
下界要塞的楼梯角可能会生成地狱疣，并且至多有2个烈焰人刷怪箱。
下界要塞是唯一一个可以自然生成在所有下界生物群系的结构（堡垒遗迹不能生成在玄武岩三角洲）。

古迹废墟（Trail Ruins）是一个掩埋在地下的陶瓦建筑结构。
古迹废墟自然生成在丛林、针叶林、积雪针叶林、原始云杉针叶林、巨型针叶林和原始桦木森林中。古迹废墟是个掩埋在地下的结构，仅尖端露出地表。古迹废墟露出地表的样子
古迹废墟主要由泥砖、各色的陶瓦和带釉陶瓦以及其它部分方块构成，内部堆积大量泥土、砂土和沙砾，并混杂可疑的沙砾。在 Java 版 1.20 快照23w16a 版本和基岩版 1.20.0 预览版1.20.0.22 版本前还包含沙子和可疑的沙子。
古迹废墟包含数个子结构，这些子结构可以分为 4 大类：
塔楼：古迹废墟的入口，一般只有塔楼的顶端露出地表。
道路：由圆石和石砖构成的道路，两侧连接各种房屋和摆设。
房屋：大小各异的子房间。
摆设：小巧的装饰结构。
战利品
古迹废墟中没有箱子，其战利品完全存在于可疑的沙砾中。
探索指南
携带至少 2 把刷子，古迹废墟包含的可疑的沙砾数量相当多，如果没有刷子就可以直接放弃探索了。
携带一把或几把铲子，最好是效率较低的石锹，避免在挖掘时损坏可疑的沙砾不能使用挖掘机进行考古发掘。
清理多余方块时要从上往下发掘，避免可疑的沙砾坠落损坏。
探索过程中玩家会获得数量众多的物品，最好携带便携存储设备没有潜影盒就一趟一趟运东西吧。

含水层是大型洞穴中大小、高度不一的积水湖泊，在 Java 版 1.17 快照 21w06a 中加入。不同高度的含水层边缘相接处会被岩石围起。在 Java 版中，生成于含水层底部的熔岩会在世界生成期间转变成黑曜石或岩浆块，水下的岩浆块还会产生气泡柱。美西螈（1.18-exp1 后不再在这里生成）和发光鱿鱼能在此生成，鱼和溺尸也能生成在位于海洋或河流生物群系（及其变种）地下的含水层中。在 Y=0 以下的含水层中的水有几率被熔岩完全替代。
含水层的边缘有时候会露出缺口，让含水层的水能流到下方去。

团簇是一种地形特征，为自然生成于地下的一些矿石或其他方块。
玩家可以在洞穴或石头自然生成的任何地方碰到这些团簇。地底的泥土沉淀和沙砾沉淀是团簇生成的第一步，随后会生成更珍贵的矿石：煤、铁、铜、金、红石、钻石、绿宝石（仅在山地生物群系中出现）和青金石。它们只能在石头、安山岩、闪长岩和花岗岩中形成（也可以替换其磨制变种，当然其磨制变种通常是结构的一部分），并且不会相互替换或任何其他方块。两块或以上的团簇生成时可能会紧挨在一起，使其看起来像由多种矿石组成的一条矿脉。
团簇曾经被称为矿脉，但在大型矿脉加入后，最终换成了“团簇”。

地牢（Dungeon）是在主世界任何生物群系自然生成的一种小型地物。
在 Java 版 1.16.2 快照20w28a 版本之前，注册名为 minecraft:dungeon。
地牢自然生成在主世界的任何高度里。每一个区块会进行 8 次尝试以生成一个地牢。虽然几率很小，但仍有可能有多于一次的尝试成功，从而生成多个地牢，有时候甚至还会生成相连的地牢。
每次生成尝试都会选择一个位置和尺寸（5×5×4、5×7×4 或 7×7×4 的开放区域）。如果下列生成条件均达成，那么生成尝试会成功：
即将生成的地牢的地板区（包括墙下面的）必须完全是固体方块；
即将生成的地牢的天花板区（包括墙上面的）必须完全是固体方块；
即将生成的地牢的墙必须有 1-5 个 2 格高的缺口（所以通常地牢不会是完全封闭的，也不会是完全开放的）；缺口与地牢的地板不允许有落差（所以玩家可以走进去，怪物也可以走出来）。
如果位置测试通过，会生成空气和圆石，然后会为 2 个箱子进行3  次生成尝试。要生成一个箱子，被选中的位置必须是空气并且在它的四个侧面有且仅有一个固体方块。刷怪笼会被放置在地牢的中央，刷怪箱种类可能是僵尸、骷髅或蜘蛛。
地牢在“生成结构”的选项关闭时仍然会生成，这是因为它是地物而非结构。
地牢在 Java 版 Infdev20100625-2 版本和携带版 Alpha0.9.0 build 1 版本加入，此后其结构基本没有变化。
在 Java 版 1.18 快照21w40a 版本和基岩版 1.18.0 预览版1.18.0.21 版本后，提高了此地物的生成数量，尤其是 Y=0 以下的数量。
埋藏的宝藏是一种带有战利品箱子的结构，只会生成在沙滩，积雪沙滩或石岸生物群系，生成时通常上方会有2-4个方块（主要是沙子、沙砾、石头）。
可以通过藏宝图找到此结构，藏宝图上的红色“×”标记着宝藏所在位置。
只会生成在每个区块（9，？，9）【仅JAVA版】或（8，？，8）【仅基岩版】的位置，可以按F3查看。最理想的方法是先看藏宝图找大致位置，然后开F3精准定位。
箱子里有一个海洋之心和其他宝藏，包括钻石，绿宝石，铁锭，海晶砂粒等等。这是生存模式唯一一个正常获得海洋之心的途径。
埋藏的宝藏在 Java 版 1.13 18w10a 和基岩版 1.2.0 1.2.14.2 中加入。

堡垒遗迹是一种少见的方形城堡状结构，可生成于下界中除玄武岩三角洲外的所有生物群系。猪灵和疣猪兽会在此生成。
堡垒遗迹会生成无视玩家是否携带金制装备，对玩家进行无差别攻击且攻击力十分强大的猪灵蛮兵（手持金斧）。和远古守卫者一样，击败之后不会再次生成。基岩版中，它还会指挥附近的其他猪灵对玩家一起发动进攻。而JAVA版中，只要不攻击猪灵蛮兵等死啊？就不会吸引猪灵的仇恨。（由环境造成的伤害不会引起仇恨，因此可以使用打火石、熔岩等“间接”攻击）
堡垒遗迹会生成 4 种类型的结构：桥、疣猪兽棚、居住区和藏宝室，如果这些结构全部生成在同一个堡垒遗迹中，这个堡垒遗迹会十分庞大且壮观。但不是所有的结构类型都会在同一个堡垒遗迹中生成。猪灵和疣猪兽会在此间频繁生成。
堡垒遗迹的藏宝室内会生成由锁链吊着的岩浆怪刷怪笼。
堡垒遗迹中的錾制黑石砖块下更有可能找到金块。堡垒遗迹内的宝箱更有可能生成在镶金黑石上。也就是说用漏斗偷箱子的难度会增加
玩家在堡垒遗迹的战利品箱中可能会找到远古残骸、下界合金碎片、金苹果、破损的附魔（除胸甲）钻石装备。
这些建筑物是唯一能自然找到“pigstep”音乐唱片以及猪灵旗帜图案的地方。

峡谷是一种可以自然生成于主世界的巨大地面裂缝。
Java版中深度一般为27～62格，基岩版中有时可以延伸至基岩层。常常可以看到矿物暴露在峡谷壁上。峡谷是十分陡峭的，因此常常有水流下，再接触底下可能存在的岩浆形成黑曜石。
要塞、废弃矿井、废弃传送门等结构有几率生成在峡谷中，若是废弃矿井，穿插在峡谷中的部分的结构会被直接破坏。极其罕见的情况下，生成在峡谷中的废弃传送门会卡掉下方的基岩，形成一个通往虚空的入口。
1.13之后，下界也会生成简单的峡谷，同时峡谷在海底也会生成。珊瑚礁有概率在水下峡谷中生成。
巨型菌类是一类在下界自然生成的类似树的结构。
巨型菌类分为巨型绯红菌和巨型诡异菌，分别在绯红森林和诡异森林生成。同时也可以向相应菌岩上的相应小型菌种上撒骨粉。两者树冠中均有可能出现菌光体。绯红菌树冠下还可能生成垂泪藤沼泽表示很赞。
“树柄”宽度一般为一格，最多可能出现3格宽的，可以作为下界中很好的木材来源。除合成门、栅栏、栅栏门等物品外，合成中对应木板可以和其他木材混用等等……这不是菌类吗。

废弃传送门是自然生成的损坏的下界传送门，在下界和主世界都能生成。
废弃传送门可以生成在主世界与下界的任何生物群系，它们可以生成在地下、水底、或祼露在地面。废弃传送门附近的自然地形会生成下界岩。
废弃传送门会生成未完成的传送门框架，包含黑曜石和哭泣的黑曜石, 尽管后者并非用于建造下界传送门的材料（可能是为了给传送门一种古老的外观）。有些传送门框架生成时会平置在地面，就像是倒塌了一样。
废弃传送门有时会生成箱子，里面储存着各种金质物品，包括金苹果甚至附魔金苹果，以及其他的一些东西，如黑曜石。
废弃传送门在主世界生成时会被石头，石砖和铁栏杆组成的结构环绕，在下界生成时会被黑石变种和锁链围住。
如果废弃传送门在水下生成，其附近可能会有岩浆块，由此产生向下的气泡柱。
废弃传送门现有13个变种：10个一般大小的传送门和3个巨型传送门，其毁坏程度各不相同。
废弃传送门可由玩家使用结构方块加载ruined_portal/portal_<1-10>或ruined_portal/giant_portal_<1-3>来生成。
废弃传送门生成在寒冷生物群系时，结构中所有的熔岩会被下界岩所取代。
冷知识：在极其罕见的情况下，废弃传送门会生成在峡谷中，下方的基岩有可能因此被破坏。

废弃矿井是一种生成在地底中的遗迹。
废弃矿井在主世界自然生成，它们通常在地下被发现，但也会生成在海中。它们在矿石等区块地形生成前生成。就像其它结构，几个废弃矿井能生成在彼此附近并相交，从而形成更大更复杂的矿道。矿道在横穿空的空间时会生成木板作为桥梁。
仅Java版1.10后&PE版本：它们也会生成在恶地生物群系的地表。在这里的废弃矿井由深色橡木木板和栅栏组成，而不是橡木。在地上时（即天空亮度是8或以上），洞穴蜘蛛刷怪箱和一些木板不会生成，生成的铁轨也会比较不完整。
在废弃矿井中可能会生成运输矿车，在其中有一定几率可以获取到一些比较宝贵的物品（例如：附魔金苹果在1.9无法合成，可以通过此途径获取）。
废弃矿井如果横穿在峡谷中，在峡谷中的部分会直接被破坏掉。
在21w41a中，提高了恶地废弃矿井的生成高度。
可以用指令/locateMineshaft寻找（1.19后需使用/locate structure mineshaft）。
掠夺者前哨站（Pillager Outpost）是一种有掠夺者居住的结构。
掠夺者前哨站是一种较稀有的结构。它们比地牢略稀有一些。它们和林地府邸由同一组方块构成。
掠夺者前哨站可以在任何可以生成村庄的生物群系里面生成——平原、沙漠、热带草原、针叶林、积雪的冻原、积雪的针叶林（仅基岩版）和1.18加入的6种新山地群系。
在Java版中，如果一个前哨站在水中或者空气中生成（例如，靠近热带高原生物群系生成），则其底部会生成一个由草方块（或沙子）、泥土和石头构成的圆形平台。这同样适用于诸如帐篷、笼子等较小的建筑。
在基岩版中，掠夺者前哨站将会生成类似于林地府邸的地基：瞭望塔使用圆石、桦木板和深色橡木原木作为地基，帐篷和笼子则生成在深色橡木木板地基上； 原木堆使用水平放置的深色橡木原木作为地基，而标靶的地基则由圆石组成。
其他较小的建筑比如帐篷、笼子和稻草人标靶可能会生成在前哨站主要建筑的旁边，最多生成四个。笼子中会存在铁傀儡，但有时候它会消失。

月球基地（Lunar Base）是一个自然生成于月球的结构，全程为“ACME Self-Building Lunar Base”（ACME自建月球基地），是投掷器的名称。
投掷器上方有一个轻质测重压力板，踩下它时，方块展示实体和一些粒子会播放一段动画，铜块逐渐向四周和上方延伸，最终变为一个巨型树状结构，且树状的边缘都是由锈蚀的铜块构成的。
在主干的枝上有铜枝叶、末地烛、灵魂灯笼、锁链和磨制黑石按钮作为点缀。
在月球基地中会生成两种战利品箱：“Moon Misson Resupply Crate”（登月任务供给箱）的箱子和“Lunar Laboratory Equipment”（月球实验室配备物）的绿色潜影盒，前者出现概率更高，但战利品种类要比后者少。

末地城是在末地外部岛屿生成的一种建筑物，在Java版1.9快照15w31a中加入。
在末地城中会生成敌对生物潜影贝。
末地城可能会生成成一个单独的塔，或一个庞大而复杂的跨越连接的塔。在某些房间里，（通常在高耸结构的顶部）有箱子和一些末影箱；装饰性的紫珀楼梯从塔的两侧突出，塔的外沿会有一些旗帜；楼梯会彼此连接多个塔，它们由紫珀块、竖纹紫珀块、紫珀块台阶、紫珀块楼梯和末地石砖组成。

基本房间，有三层楼高，每一个房间都要比下面的一个更宽。
小房间，外面挂有旗帜，还有一些潜影贝吊在天花板上。
战利品房间，包括两个有 战利品的箱子。
稀有战利品房间，有一个末影箱以及正常的箱子。
没有任何东西的房间。

小塔是空心的，只有一些紫珀台阶当作楼梯。
大塔（是小塔的三倍直径），把紫珀台阶和末地烛当作楼梯。

末地折跃门是一个在击败末影龙后在岛屿的随机边缘上生成的一个结构，用于快速传送到外岛。
末地折跃门在Java版1.9快照15w31a中加入。
传送门被基岩包裹着所以需要使用末影珍珠或者穿戴鞘翅飞行进去。
当末影龙被打败之后末地折跃门会生成在末地主岛的边缘， 最多可为每个世界生成20个入口， 生成在末地主岛周围大约y=75的高度，大约与中心的末地传送门相距70-100个方块。已生成20个入口后，再击败末影龙将不会生成入口。
在主岛激活一个末地折跃门之后，将在一个主要的外岛上生成另一个末地折跃门，与主岛的门相连通.这些入口生成在y=66~69 之间。即使没有区域容纳末地折跃门，然后末地折跃门就会生成在高于平常水平的Y轴。末地折跃门的X坐标和Z坐标是随机的，但总是和末地中心(0,0)相距1000个方块。
当任意实体进入并被传送时，折跃门会发出一道类似紫色信标的光柱。
末地折跃门可以随机生成在外岛，它们可以将玩家传送至主岛黑曜石平台的位置，即使它们不会在主岛上生成/重新生成该平台。

末地船是一种生成在末地外部岛屿中的浮空建筑物，在这个遗迹中会生成潜影贝并且会生成一些比较宝贵的物品。
末地船在Java版1.9快照15w31a中加入。
在末地城外会有小几率生成末地船。这些浮空船的结构像传统海盗船，但用和末地城相同的材料。末地船通常在一个码头前生成，玩家可向末地船投掷末影珍珠以登上船。
末地船的内部很小。船内的第一个房间只包含一个酿造台，里面有2瓶瞬间治疗药水。一个楼梯下面会有一个宝藏室，这里有两个通常包含非常宝贵的战利品的箱子和在两个箱子中间的物品展示框里的鞘翅，这些都被一个潜影贝守卫着。末地船的地板上有两条黑曜石线。
在船头会有一个龙首，这是目前唯一能自然获得龙首的途径。
在Java版Beta 1.8pre-1中加入，在1.14快照18w48a到18w50a中修改了村庄外观。
村庄是一种自然生成在主世界中的建筑群落，其也是除了雪屋以外会生成村民的地方，但在BE版中，世界类型选择“超平坦世界”中则不会生成村庄。
村庄会自然生成在平原、热带草原、针叶林、沙漠、冰原、积雪的针叶林（仅基岩版）和草甸生物群系。村庄的种类取决于村庄水井西北角所在的生物群系（如果不是其他几种村庄的话，默认是平原类型）。
携带版和Java版1.10后，村庄有2%的几率生成为僵尸村庄。在僵尸村庄里，所有村民会变为僵尸村民（这些僵尸村民不会消失，但对阳光没有特别的免疫），没有火把和门，并会随机生成蜘蛛网，一些本来应该是圆石的方块也会被替换成苔石。
村庄如果生成在水上时道路会为对应生物群系的木板，若生成在陆地上时则道路会为沙砾（携带版和Java版1.10后被土径取代）。
基岩版和Java版1.14快照18w47a之后，击杀巡逻队队长后可获得不祥之兆buff，拥有此buff后接近村庄会触发‘袭击’，玩家在结束袭击后，将获得村庄英雄buff和进度。
1.14前，村庄的房屋大致分为铁匠铺、简陋小屋、小屋、住宅、肉店、图书馆、小农场、大农场、教堂。1.14后，村庄中的结构则多得多了，除各种住房以外还有职业房屋，并且每个群系的村庄都有自己的建筑风格。当然，所有村庄都会生成街灯这一照明结构。
尽管村民有丛林和沼泽变种，但村庄不存在丛林和沼泽生物群系变种，可能是因为藤蔓和海草会使村民的寻路机制出错，但未来仍有可能加入这两种类型的村庄。并且曾经提到关于过基岩版的沼泽村庄的特性，即道路是用圆石做的。
可以用门（1.14之后是有效的工作站点方块），床和村民来“制造”一个村庄，系统会自动认定它是一个村庄，这样的村庄可以触发袭击。
在1.14前，只有铁匠铺才会有箱子，而在1.14后，每个房屋都有概率会生成箱子。
杜鹃树高5-7格，由橡木、杜鹃树叶和盛开的杜鹃树叶组成。
对着杜鹃花丛使用骨粉可以让它生长成杜鹃树，但它的上方至少要存在5×5×5的空间。
杜鹃花丛也可以在水中种植及催生。
长成杜鹃树之后，它下方1格中的泥土、草方块、砂土、灰化土、耕地或苔藓块会变成缠根泥土。
林地府邸是一种生成在黑森林生物群系的一种大型建筑物，十分罕见。
林地府邸会非常罕见地在黑森林生物群系中生成，通常距离出生点有几万个方块远；玩家可以通过向制图师村民用绿宝石购买林地探险家地图寻找它们，或是使用/locateMansion 指令（1.19 后需使用 /locate structure mansion 指令）。
1.14 前林地府邸是卫道士、唤魔者以及恼鬼唯一自然生成的地方，并且他们像远古守卫者和猪灵蛮兵一样不会再次生成也就是说一旦你打下来，这房子就归你了。（1.14 后卫道士、唤魔者也会在袭击中出现）
同样的，唤魔者和卫道士也只会在特定房间内生成（当然他们也可以走动）。
由于林地府邸是随机生成的，所以它的结构会呈现不同。所有林地府邸都有下列共同之处：它们通常有 2-3 层，包含多种房间，且整个结构都生成于圆石地基上（与村庄房屋很相似）。顶层总是比下面的几层小一些。类似海底神殿，林地府邸的房间的长宽高是有一定比例的，也就是说，这些房间是由若干个“单元”组成的。
林地府邸有许多不同类型的房间，不是每个房间都有特定类型的入口，不是每个林地府邸都包含所有类型的房间。玩家可以 minecraft.jar 文件夹中的 structures 利用结构方块手动加载结构以参观林地府邸的个体结构。放置一个使用加载模式（Load Mode）的结构方块，在结构名称处填写 "Mansion/结构名称" 点击加载。
树木是一种由原木和树叶组成的地物，在我的世界原版中一共有9个种类，分别是橡树、云杉、桦树、丛林树、金合欢树、深色橡树、杜鹃树、红树和樱花树。
其中巨型云杉、巨型丛林树、巨型深色橡树的宽度是两格，高度可达30格。
基岩版或教育版中，可能会生成倒下的树木，包括一格高树桩和平躺的树干，有时上面还会生成蘑菇。
简介
一种会生成在樱花树林的植被地物，整体高度为 10-11 格。
每棵樱花树生成时有 5％ 的概率带有一个容纳 1-3 只蜜蜂的蜂巢。
一种常见的树木，包含橡木原木和橡树树叶，偶尔会附带一个蜂巢。
沉船主要生成在海洋（及其变种），有时也会在沙滩生成。其实，沉船可以生成在所有海洋群系的海底以及靠海的任何生物群系，包括山地的山顶哦上帝她怎么上去的。生成在海底的时候不会被方块掩埋，否则会，哪怕是一个被判定为河流并且足够深的湖。
沉船的形状像经典的帆船或海盗船，生成时有三种形式：直立、侧立或倒立，多数情况下，沉船会缺失船头、船尾、桅杆或者其他方块，显得好像损坏了，但是也有罕见的完好无损的沉船，下面会介绍。它们仅由木质材料组成，所有种类的木质材料（java版中金合欢材质除外）都可应用于沉船结构（当然除了来自下界的诡异木板和绯红木板）不过那个配色除了有点瘆人还是不错的，但正常情况下同一个沉船只使用两种木质材料。
沉船中有3个箱子。其中补给箱在船头甲板的下部，宝箱在船舱，地图箱在船尾甲板下部。如果一艘沉船的某个部分缺失了，那么处于该部分的宝藏箱也将缺失但为什么不能做成散落或掩埋在周围呢。其中地图箱一定会刷出藏宝图，宝箱有铁锭&粒，金锭&粒，绿宝石，青金石，钻石和附魔之瓶。补给箱会刷迷之炖菜，火药，TNT和竹子以及一些垃圾。
沉船一共有20个变种（算上组成材料基岩版是300种，java版是200种），朝向有正向的，侧翻的和倒翻的三种；在朝向的基础上，组成部分有完整的，船头和船尾三种，正向完整的沉船还有带桅变种。另外，每一个沉船都有几率成为侵蚀变种（在完好的基础上去掉几个特定位置非箱子的方块）。
已知bug：当一个沉船生成时跨越2个区块，那么这艘船会被分成两半，构成的材料也会不同，甚至会发生上下错位的现象。这种情况一般不会发生在海底的沉船，而且由于战利品箱位置是固定的，所以不会破坏战利品箱，也不会出现多个战利品箱。

沙漠水井是一种生成在沙漠生物群系中的一种建筑。
沙漠水井由砂岩、砂岩台阶与中心的水构成。
沙漠水井在任一个沙漠区块中都有0.1%的几率生成沙漠水井，因此沙漠水井很罕见。
创建地图时即使取消“生成建筑”沙漠水井依旧会产生，因为沙漠水井属于小型结构。沙漠水井生成时可以套住仙人掌，其中的砂岩可以生长甘蔗，讽刺的是，它经常在水源旁生成——但在13w17a（1.6.1的快照）后沙漠中不会再有湖泊，这使得沙漠水井（以及沙漠村庄中的农田和水井）成为沙漠地形中仅有的“天然”水源。
沙漠神殿是一种生成在沙漠生物群系中的类似金字塔的遗迹。
在沙漠神殿中央有着一片由蓝色陶瓦和橙色陶瓦组成的图案，图案正中心9格下方则是宝藏室，里面有4个箱子，1个石质压力板，压力板下的砂岩下面有9个TNT。蓝色陶瓦在石质压力板的正上方，如果直接跳下去就有可能激活压力板从而激活下方的TNT，然后发生爆炸，箱子会被破坏，玩家也很容易死亡。因此建议破坏蓝色陶瓦旁边的地面并从此时出现的洞中进入宝藏室并在进入宝藏室后立刻破坏压力板。
但因宝藏室内没有照明，所以有时候也会刷出怪物踩到压力板，等到玩家过去的时候，宝藏可能已经被炸毁了。为避免这种情况，玩家可以在发现沙漠神殿后尽快前往探索，这样也许可以在刷新出怪物前进入宝藏室。
沙漠神殿生成时有时会穿插在峡谷中，此时藏宝室会悬空在峡谷里。

洞穴是在地下生成的一系列空气方块区域。1.17之后，主世界洞穴的种类得到了大规模扩充，而下界只会生成简单的小型洞穴。可以生成在Y=256以下的区域中，包括水下。部分洞穴可以延伸至基岩层。
洞穴中经常是十分昏暗的，因此怪物和蝙蝠会在这里生成。但可能存在的岩浆为洞穴提供了一些亮度。地牢和废弃矿井等可能会与洞穴相连。
1.18快照exp1版本以后，洞穴有两种类型：雕刻器洞穴和噪声洞穴。
雕刻器洞穴：
小型洞穴：呈细长的条形，一般出现在接近海平面的高度。
中型洞穴：空间比小型洞穴更加宽大，拥有丰富的基础矿石和少量较为稀有的矿石。这里也会可能出现小型涌泉。
大型洞穴：可能会裸露于地表，也可能接近基岩层。体积更大，如果在基岩层附近几乎所有主世界的矿石会在这里生成。
水下洞穴：有可能出现海草，黑曜石和岩浆块也会生成，通常完全封闭。
除此以外还包括连环洞穴、深坑等。
噪声洞穴：
分为芝士洞穴、意面洞穴和面条洞穴还挺形象的。
芝士洞穴通常呈气泡状。意面洞穴是长而蜿蜒的洞穴；面条洞穴是意面洞穴更狭窄、弯曲而密闭的变种。
下界洞穴：
下界一般只有小型的洞穴，有时也可以抵达基岩层，但由于下界地下的熔岩分布没有一定的规律，此时基岩可能裸露在外。下界洞穴也会刷怪。

浮岛是一种悬浮在空中的结构，在山地或悬崖附近生成的概率会高一些。浮岛上可能会生成诸如树木之类的结构，极其罕见的情况下，村庄的部分也会在此地生成（见封面）。
一般情况下，浮岛可能仅仅是悬崖边上的单个方块。很少有大型的浮岛。
生成
寒带海底废墟生成于海洋、冷水海洋与冻洋生物群系，且包含了它们的深海变种，由石质材料构成。
热带海底废墟则生成于暖水海洋、温水海洋以及温水深海生物群系，由石质和砂岩材料构成。
虽然大部分海底废墟生成在水中，但它们有时也会生成在陆地上。
大部分结构都会藏着内含战利品的箱子。根据海底废墟结构的大小和类型，箱子会是海底废墟小箱子或海底废墟大箱子。
结构
寒带海底废墟主要由石砖和沙砾构成，同时它们也会包含海晶石、沙子、岩浆块、海晶灯、磨制花岗岩、带釉陶瓦、砖块、木板、圆石、苔石、石砖楼梯、黑曜石和/或一个内含战利品的箱子。
热带海底废墟主要由沙子和砂岩构成，同时它们也会包含磨制花岗岩、磨制闪长岩、海晶灯、岩浆块、沙砾、陶瓦、砂岩楼梯和/或一个内含战利品的箱子。
海底废墟的形状和大小变化很大，而它们通常会成群地生成来组成遗迹村庄。
自Java版1.20快照23w12a开始（以下特性在基岩版1.19.80.21作为实验性玩法加入，1.20.0.21后不再属于实验性玩法），可疑的沙砾及可疑的沙子会在海底废墟中生成，使用刷子持续扫刷可以获得金粒、陶片、绿宝石等奖励，在你获得至少两只可以进行繁殖的成年嗅探兽前，扫刷热带海底废墟中生成的可疑的沙子是获取嗅探兽蛋的唯一途径（概率约为6.67%，扫刷沙漠神殿及沙漠水井中生成的可疑的沙子不会出现嗅探兽蛋）

海底遗迹是在深海（Deep Ocean）中自然生成的遗迹，其类似沙漠神殿和丛林神庙。
在海底遗迹周围包括里面均生成守卫者，生成区域以遗迹为中心的58x58x23的长方体。
海底遗迹的核心区域中拥有被暗海晶石包裹的8个金块。
海底遗迹中有少量的概率会生成一个布满海绵房间。
每个海底遗迹中都会生成3只远古守卫者，分别在两个侧翼和顶层房间（这是除了海绵房间外的唯一获取海绵的来源），且不会再次生成。
海底神殿内部的布局分为一个核心区域，两个相连的、围绕神殿一周的侧翼，和一个顶层房间。
核心区域是由46个8*4*8的基本单元组成的，分为3层，第一、二层各有20个单元（5*4，从正面看是横着的），第3层对着第二层的最前面的3*2的空间，也是横着。神殿前面那个带花纹的墙不算在这个空间内。
核心区域的房间一共有11种，1*1*1的有5种，分别是（都是官方名称）入口房间，十字形房间，暗海晶石小房间，细支柱房间和海绵房。1*2*1，1*2*2，1*1*2，2*2*1，2*1*1的各有一种，分别是DY房间，DYZ房间，DZ房间，DXY房间和DX房间。懵就对了，wiki上就这么写的还有一个是核心藏宝室，中间有8个金块。顺带提一句，整个海底神殿构成的方块都是完整方块，因为1.8版没有含水机制，所以箱子或楼梯这种看起来就比较违和，1.13之后官方也没有加入战利品箱的意思。但要是给我8个钻石块我肯定愿意征服整个海洋
入口房间只有一个，只能在第一层的正前（in the front of）并直接占用一个单元。核心区域后部有4个位置可能生成藏宝室（但最终只有一个），然后占用2*2*2的空间。所有相邻的单元的出口一开始都会打开（连接顶层房间的特定入口和入口房间的特定入口也会打开），然后每个单元执行5次以下这个操作，直到2次成功或满5次。
1，从6个方位中任意选择一个。
2，如果这个方向上的通道已经关闭了，这次尝试失败。
3，将这个方向上的通道标记为“关闭”。
4，对方才关闭的通道连通的两个单元进行检查，确保从入口房间到这两个单元均存在至少一条通路。如果此条件不满足，尝试失败，通道重新打开。
最后，随机不断检查所有未占用的单元，如果若干个相连的单元能够形成上面5个多单元房间，则生成可生成的房间，并占用这些单元。如果只有下方一条通路，那么生成一个海绵房。因此海绵房稀有的原因就在这里了，因为自身单元需要关闭2个通道，剩余3个方向的房间也要关闭对应通道，还必须保证下面那条路能通到入口房间。剩余的未占用的都是1*1*1的单元，会生成剩余3种小房间（各1/3的几率）。也正是因为这个机制，才导致神殿里的路线跟迷宫一样。
再然后，神殿生成完毕。关于此生成机制更完整的描述以及整体结构剖面图还请看此wiki。
另外，虽然当时版本只有1.8，但是海底神殿的内部构造生成机制直到现在仍然是所有建筑里最复杂的。

涌泉是一种用于随机生成单个液体方块的地物，会在所有主世界与下界的生物群系中生成。
在主世界中，涌泉会以水源或熔岩源的形式生成于石头侧面，在崖壁上或洞穴中尤为常见。若熔岩涌泉生成在森林等树木较多的群系中则有可能导致森林大火。涌泉不会在y轴192以上生成。
在下界中，涌泉会以熔岩源的形式在下界岩的侧面以及天花板上生成。另外熔岩涌泉也会生成在方块的内部形成隐藏的熔岩，因为这种涌泉不像其他类型的涌泉一样暴露在外，因此在下界中挖掘时要尤其小心。
一种在黑森林生成的树木，包含深色橡木原木和深色橡树树叶。
玄武岩柱是一种生成于下界灵魂沙峡谷的、由玄武岩组成的石柱结构，在Java版1.16快照20w06a中加入。其柱体从地面一直延连到天花板上，水平方向上的宽度在1到3格之间不等；基部则相对平整，由一片3×3左右的玄武岩堆成。
注：玄武岩三角洲生成的结构与玄武岩柱有一定区别，称为玄武岩石柱林，高度比玄武岩柱低，并且通常是数根柱的聚合体。

珊瑚礁是生成在暖水海洋中的地物[仅Java版]或结构[仅基岩版]。它由许多成簇的珊瑚块、珊瑚和珊瑚扇组成。这些小簇的形状、大小和颜色各异，包括从只有几个方块的脑珊瑚到巨大的树状的火珊瑚（共11个变种）。

苍白橡树（Pale Oak Tree）是一种自然生成于苍白之园生物群系中的地物。
苍白橡树分为两种：
普通苍白橡树，注册名为 minecraft:pale_oak；
嘎枝苍白橡树，注册名为 minecraft:pale_oak_creaking，该类型的树会附带一个嘎枝之心。
苍白橡树与深色橡树相似，树叶、树干和根部还分别有 15%、40% 和 80% 的概率附有苍白苔藓。
嘎枝苍白橡树的树干中会尝试生成一个嘎枝之心，生成位置的 6 个方向需被标签为 #logs 的原木包围，因此嘎枝之心的生成概率较低。
苍白橡树有 20% 的概率附带一个嘎枝之心。
苍白橡树生成时会将下方 1 格的方块转变为泥土。
树苗长成的苍白橡树永远不会是嘎枝苍白橡树。
苍白橡树在 Java 版 1.21.2 快照24w40a 版本和基岩版1.21.50.20 版本中加入。

要塞是一种会生成在主世界任何生物群系地底下的一种遗迹，需要用末影之眼定位，直到末影之眼向下飞行。
要塞中蕴含了许多房间有些房间有各自特别的东西（如：图书馆、牢房与储藏室等），其中最主要的就是传送门房间，每个要塞仅会存在一间，里面有激活末地入口的末地传送门。
在Java版1.9以前，每个世界会生成3座要塞，1.9以后，会生成128个要塞。在基岩版中，要塞会无限生成。在主机版中，只有一座。
要塞主要生成在地下，并且倾向于生成在海平面以上的生物群系，但如果有必要，它们依然会生成在水下。在基岩版中，要塞可能生成在村庄的水井下。
在java版中，要塞生成方式为圆环区域生成，要塞从中间的第一个环开始生成，然后在第二个环生成，然后在第三个环生成，以此类推，直到总数达到128。
要塞可能会被穿过的废弃矿井，峡谷或其他天然结构割断，而如果要塞在水下，断面会使整个房间被灌满水。要塞的某些房间可能会浮现在海平面，但这种可能性极小。有末地传送门的房间（包括要塞的其他部分）可能被峡谷、洞穴、废弃矿井、地牢或者要塞本身的其他部分覆盖，这会迫使玩家去寻找另一个要塞去前往末路之地。（内容摘自中文Wiki）

试炼密室（Trial Chambers）是一个包含试炼刷怪笼和宝库的地下结构。
试炼密室自然生成于主世界的深板岩层，其起始结构为生成旋风人的决斗室，在 y=-40 到 y=-20 的高度生成。
Java 版中，每 34×34 个区块的区域中总会有一个试炼密室生成。
基岩版中，生成概率相较于 Java 版更低。
结构
主要分为柱廊、楼道、交叉口和决斗室。
结构文件夹在 client.jar/data/minecraft/structures/trial_chambers（Java 版）或 data/behavior_packs/vanilla/structures/trial_chambers（基岩版）中。
版本信息

返回传送门始终在X=0, Z=0, Y=(地形表面)生成，在没有杀死末影龙之前不会被激活。
它的外观与末地传送门类似，但细节上不同：其边框由直径为7格的碗状基岩组成，而不是末地传送门框架，中间的石柱被火把环绕，最上方摆着一颗龙蛋[1]。16个末地石会在每个基岩边框下面各生成一个。当末影龙重生并被杀死时，被破坏的终末之池会重新激活。当复活末影龙仪式开始或终止时都会重新生成返回传送门。如果复活末影龙所用的水晶有一个被推动过（1.18+），那么复活若失败（水晶爆炸）所重置的传送门是不完整的的，只有两格传送门方块。
当玩家从末地进入终末之池时，会出现终末之诗（在此期间玩家会被隐藏在终末之池的石柱上）之后玩家会被传送到主世界的出生点。若玩家在一存档内已经用终末之池传送到主世界了，则下一次进入终末之池时不会出现终末之诗。如果是非玩家实体，会被传送到世界出生点。
玩家使用返回传送门回到主世界后会被清除状态效果。
返回传送门中的传送方块和末地传送门的传送方块是同一种方块。所以在末地创建一个完整的末地传送门框架并用末影之眼激活也可以返回主世界。
在1.9快照15w31a之前，并不会生成未开启的终末之池，而是会在玩家击败末影龙后在末影龙下方生成一个开启的终末之池。若玩家在主世界击败末影龙，则在主世界生成。由于传送门方块是同一种，因此和末地传送门的效果一样，将玩家传送至末地。

远古城市生成在深暗之域生物群系，固定生成Y轴为-52。主体由深板岩等构成，结构被城墙包围。远古城市中心是一座大型建筑物，外观为监守者的雕像，其中隐藏的房间里有红石电路。还包含一些其他建筑物，几乎所有建筑物没有屋顶，外观上显现出带有一定程度的破损。建筑物中会生成战利品箱。

雪屋是一种在冰原或者冷针叶林生成的一种建筑。
雪屋中由一张床、一个熔炉、一个工作台、一根红石火把及一些地毯所构成。
在50%的雪屋里，地毯下能找到一个活板门，里面有一个很长的梯子通向地下室。通道和地下室是用石砖及其变种做成的，其中有一些是怪物蛋。地下室里有一个放有喷溅型虚弱药水的酿造台，一个2/3满的炼药锅和箱子，箱子内必有一个金苹果。最里面有一个铁栏杆做的笼子，左边有一个牧师村民，右边有一个僵尸牧师村民，不会出现其他职业。雪屋里的金苹果和喷溅型虚弱药水能用于治疗僵尸村民。
有地下室的雪屋会被认定为一个“村庄”，因为其包含被村民绑定了的工作站点（酿造台）。因此，携带不祥之兆状态效果的玩家靠近会触发袭击可我就一个光杆司令这打啥啊。注：雪屋上方的床没有被村民绑定的情况下不会触发袭击。

黑曜石柱在Java版1.0.0快照Beta 1.9-pre4中加入。
黑曜石柱是末路之地中的一种高大圆柱形结构。黑曜石柱是从上到下贯穿整个末地的，即浮空在虚空之上，周围有末地石；在不挖掘的情况下，站在塔顶刚好能看到黑曜石柱底部黑曜石。黑曜石柱上有一块末影水晶治疗末影龙。
1.9快照15w31a版本以后其中两个较低的黑曜石柱顶端会出现铁栏杆。（搬运自中文Wiki）
