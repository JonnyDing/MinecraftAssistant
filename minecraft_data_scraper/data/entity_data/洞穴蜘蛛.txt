简介
洞穴蜘蛛是一种类似于蜘蛛但能够造成玩家中毒的中立型生物。
洞穴蜘蛛的碰撞体积比普通蜘蛛小，可通过 0.5 格高，1 格宽的通道。其生命值为 12，外表呈深蓝色，但它却比普通蜘蛛更危险。它们只能通过废弃矿井里的刷怪笼生成，但如果通过编辑手段允许洞穴蜘蛛自然生成，那么它们只能生成在亮度等级小于等于 7 的地方。（像其他大多数怪物一样）
并且在基岩版中，还会生成洞穴蜘蛛骑士。若洞穴蜘蛛在寒冷生物群系生成且判定成功生成洞穴蜘蛛骑士，上方的骑士有80%的概率成为流浪者；若用刷怪蛋等方式使其在下界生成，上方骑士有 80% 的概率成为凋灵骷髅。
Java 版 1.20.5 快照 23w51a 和基岩版 1.20.60.23 后，洞穴蜘蛛或蜘蛛会主动远离半径 7 格（Java 版）和 7*7*7格（基岩版）的未蜷缩犰狳。
与普通蜘蛛的不同之处：
当亮度等级小于等于 7 时（普通蜘蛛为11）为攻击型，亮度等级大于等于 8 时（普通蜘蛛为 12）为中立型。
可穿过 1 格宽 0.5 格高的空间。能够穿过下界砖栅栏和普通栅栏之间的空隙。
生成时不会自带状态效果。
洞穴蜘蛛的碰撞箱体积只有普通蜘蛛碰撞箱体积的 13.9%，但它的模型体积却占普通蜘蛛的 34.3%。洞穴蜘蛛的实体阴影更大，覆盖了 49% 的区域。
版本信息
洞穴蜘蛛在 Java 版 Beta 1.8pre1 中加入，在 1.14 快照 18w43a 中修改纹理。
