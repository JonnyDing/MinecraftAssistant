蝙蝠（Bat）是一种会飞的友好生物，生成于地下洞穴中，渲染洞穴中气氛。
蝙蝠拥有 6 点（）生命值，死亡时不会掉落任何物品和经验。
蝙蝠可以生成在除蘑菇岛、河流、海洋、深暗之域、雪原外的任何生物群系。
在 Java 版中，蝙蝠生成在主世界亮度≤3 的地方，但在每年的 10 月 20 日至 11 月 3 日期间，只需要亮度≤6 即可生成。在基岩版中的亮度需≤4 才会生成蝙蝠。
蝙蝠空闲时会倒挂在方块下面，当玩家接近（即使拥有隐身状态效果）或破坏它倒挂的方块时会起飞并远离玩家。
蝙蝠在 Java 版 1.4.2 快照12w38a 版本和携带版0.11.0 build 1 版本中加入，在 Java 版 1.14 快照18w43a、1.20.3 快照23w43a和23w44a 版本以及基岩版 1.10.0 预览版1.10.0.3、1.20.50 预览版1.20.50.22 版本中修改了纹理。
