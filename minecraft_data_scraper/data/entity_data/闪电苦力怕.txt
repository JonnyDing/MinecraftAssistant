Java 版召唤命令：
基岩版召唤指令：
简介
闪电苦力怕在 Java 版 Beta1.5 中加入。
1.15 前称为高压爬行者。
当闪电劈中普通苦力怕附近 3~4 个方块时就生成了闪电苦力怕。使用带有引雷附魔的三叉戟或使用避雷针引雷也可以生成闪电苦力怕。
它的爆炸强度比一般的苦力怕要强得多（闪电苦力怕的爆炸威力是 6，是普通爬行者的 2 倍），甚至强过TNT。
最高伤害
简单：49点（× 24.5）；
普通：97点（× 48.5）；
困难：145点（× 72.5）。
行为
闪电苦力怕与普通苦力怕的不同之处是高压爬行者有着明显的蓝色电弧围绕，即使隐身也会显示蓝色电弧。
闪电苦力怕的倒计时器与普通苦力怕一样，包括范围和时间。
由闪电苦力怕杀死的苦力怕、骷髅、凋灵骷髅、僵尸和猪灵会掉落一个他们的头颅，被骷髅和流浪者射杀后会掉落唱片。
在基岩版 1.17.0 之前，如果流浪者被闪电苦力怕炸死，则会掉落凋灵骷髅头颅，但是在基岩版 1.17.0 之后失效，流浪者不再会掉落头颅。
在 Java 版中，即使炸死了许多能掉落头颅的生物，一只闪电苦力怕也只会造成一个头颅掉落。
