简介
蜘蛛骑士（Spider Jockey）是一种由骷髅、凋灵骷髅或流浪者骑在蜘蛛或身上形成的生物组合，在基岩版中，骷髅还可以骑在洞穴蜘蛛上。
生成
每只蜘蛛或洞穴蜘蛛（基岩版）在生成时都有 1% 的概率再生成一个骷髅骑乘在自身上，在 Java 版 1.4.2 快照12w34a 版本及以后，使用蜘蛛刷怪蛋可以生成蜘蛛骑士。
凋灵骷髅或流浪者形成的蜘蛛骑士无法自然生成，但可以通过指令生成，在 Java 版 1.11 快照16w32a 版本和基岩版 1.16.20 预览版1.16.20.50 版本以前，或在下界要塞或寒冷的生物群系中使用蜘蛛刷怪蛋有概率生成，但在此后，生成蜘蛛时无法再生成带有凋灵骷髅或流浪者的蜘蛛骑士。

行为
蜘蛛骑士的蜘蛛控制着移动，但行为都是独立的，若骷髅射中了敌对生物，则敌对生物会攻击骷髅而不是蜘蛛。
当蜘蛛骑士的骷髅处于战斗状态时，蜘蛛骑士的移动路径取决于骷髅的移动路径（但蜘蛛依旧会进行突袭），在闲置状态下，蜘蛛骑士的移动路径则取决于蜘蛛。
在进入水中时，蜘蛛骑士会立刻分离。
当蜘蛛骑士在矿车里时，矿车会自动加速，蜘蛛的腿也会像在走路那样移动。
蜘蛛可以爬墙，蜘蛛骑士也可以爬墙，但骷髅可能会因为爬墙遇到障碍物窒息而亡。
版本信息
骷髅蜘蛛骑士在 Java 版 Alphav1.0.17 版本和携带版 Alpha 0.11.0build 1 版本中加入，在 Java 版 Alphav1.2.0、Beta1.8、正式版1.0.0、1.9 快照15w31a、1.14 快照18w43a版本，以及携带版 Alpha0.15.0 build 1、基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
凋灵骷髅蜘蛛骑士在 Java 版 1.4.2 快照12w36a 版本和携带版 Alpha0.12.1 build 1 版本中加入，在 Java 版 1.9 快照15w31a 版本和携带版 Alpha0.15.0 build 1 版本中加入了闲置时的纹理。
流浪者蜘蛛骑士在 Java 版 1.10 快照16w20a 版本和携带版 Alpha0.15.0 build 1 版本中加入。
