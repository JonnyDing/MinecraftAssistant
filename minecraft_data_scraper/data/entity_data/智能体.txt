注：生成智能体只能通过 WebSocket 端执行 /agent create 或使用刷怪蛋等方法生成。
智能体（Agent）可以让玩家进行编程来控制，来实现帮助玩家学习编程的目的。为基岩版和教育版特有。
智能体拥有 20 点（）生命值。
智能体不能被任何影响移动的效果移动，不会下落，不会被水流移动，不会被爆炸推动，且免疫任何伤害。
在没有接收到来自 WebSocket 端的命令时，脚下没有方块会上下浮动，有方块则站在方块上。
智能体的命名规则为：主人名称.Agent，并随机赋予一种颜色。
使用 /agent tp 指令可以将智能体传送到玩家身边。
控制智能体只能通过 /agent 指令控制。
智能体在携带版 Alpha 0.16.0 build 1 和教育版 1.0.1 版本加入，在携带版 Alpha 0.16.0 build 2 中移除，但又在携带版 0.17.0.1 版本中重新加入。在教育版中，智能体最初的名字为代码学习吉祥物（Learn to code mascot），在 1.0.2 版本中重命名。
