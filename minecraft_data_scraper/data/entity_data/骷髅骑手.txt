简介
骷髅骑手（Skeleton Horseman）是一种由骷髅陷阱马生成的亡灵生物。

除了指令生成骷髅陷阱马，在雷暴天气时，闪电劈下会有概率生成骷髅陷阱马，但人工产生的闪电无法生成骷髅陷阱马，生成概率取决于游戏难度，简单为 0.75–1.5%，普通为 1.5–4%，困难为 2.25–6.75%。
当玩家与骷髅陷阱马的距离小于等于 10 格时，会生成一道没有伤害且不会产生火的闪电击中骷髅陷阱马，被劈中的骷髅陷阱马会变为骷髅骑手，并额外生成 3 只骷髅骑手。若骷髅陷阱马在 15 分钟（18000gt）内没有变为骷髅骑手，则会消失。
行为
骷髅骑手与骷髅一样，或左右移动进行射箭，玩家靠近时会后退。
骷髅骑手的装备是附魔的，其中弓的附魔为力量I 或耐久I，铁头盔的附魔为耐久I 或 I 级的任意保护附魔，如果骷髅骑手生成时自带了其他帽子，则不会再穿戴铁头盔。
骷髅骑手的骷髅马是驯服的。
在生成时，有 3 秒的受击后伤害免疫效果。
版本信息
骷髅骑手在 Java 版 1.9 快照15w38a 版本和携带版 Alpha0.15.0 build 1 版本中加入，在 Java 版 1.14 快照19w08a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
