简介
猪在 Java 版 classic 0.24_SURVIVAL_TEST 中加入，在 1.14 快照 18w43a 中修改纹理。
猪是一种生成于主世界草方块上的被动生物，生命值 10 点（）被杀死后掉落（1-3）生猪排（烧死掉落熟猪排）与 1-3 点经验。
行为
跟随手持甜菜根、胡萝卜、马铃薯的玩家，可喂食甜菜根、胡萝卜、马铃薯以繁殖后代或加速小猪生长一次喂食可以加快 10%（2 分钟）。
猪给它上鞍后可骑乘，与马不同，猪不需要被驯服，但不能装备马铠，也不能跳跃。
骑乘时需要使用胡萝卜钓竿控制方向，否则猪只会带着你四处游荡。
当猪被闪电劈中时，猪会变为僵尸猪人或僵尸猪灵。不同于哞菇和苦力怕的是，此时生成的僵尸猪灵血量为满。
在 1.16 快照 20w06a 后，幼年猪受到雷击后会转化为幼年僵尸猪灵。
杀死幼年的猪不会有任何掉落物，包括经验值。
Java 版中，/give指令获得的刷怪笼默认生成猪。
