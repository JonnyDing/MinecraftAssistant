区域效果云（Area Effect Cloud），又被称为药水云，是由滞留药水产生的实体。也可由拥有状态效果的苦力怕爆炸后产生。
滞留药水撞击后爆裂，会创造出一片药水云。药水云由对应效果的药水颗粒组成。
药水云初始的覆盖半径为3方格，并在30秒内渐渐缩小到0。在云没有消失之前，任何进入药水云的玩家或生物在一秒后会受到相应的状态效果；同时药水云的半径会立即缩小1⁄2方格，使药水云的消失变快。
对于存在持续时间的药水效果, 药水云赋予效果的持续时间是对应药水时长的1⁄4。对于不存在持续时间的药水效果，例如瞬间治疗与瞬间伤害，药水的效力会减至对应药水的1⁄2。
当生物一直站在药水云中时，药水云会每一秒给予一次效果（即默认两次生效的间隔冷却时间为20游戏刻）
通过/summon指令和NBT标签可以自定义区域效果云的各项属性，包括药水效果、影响半径、持续时间、颗粒颜色、两次施加效果的冷却时间、施加效果后的半径改变量、每刻的半径改变量、组成区域效果的颗粒类型等。
概述
卫道士是一种敌对生物，属于灾厄村民，生成于林地府邸与袭击事件中，生命值为24点（× 12）。
生成
在困难模式下，灾厄巡逻队中的每个掠夺者有 20% 的几率被替换为卫道士（仅基岩版），会倾向于生成于较后的波次以及困难难度中。
由于在 1.14.4 以前没有袭击，所以卫道士只会出现在林地府邸中，他们不会重生也不会被被刷新系统刷没。基岩版中将难度设定为和平后不会消失，但此时他们不会对玩家造成攻击伤害。
攻击
卫道士外形类似村民，他会主动攻击玩家、村民、铁傀儡，在他发现攻击目标时，会掏出一把铁斧，铁斧附魔的概率以及攻击伤害会随着难度的增高而变大（攻击伤害简单为7点（），普通为13点（），困难为19点（））。
基岩版中，卫道士追逐目标时的速度与玩家疾跑的速度相当。
行为
袭击事件期间，卫道士和村民一样可以把门打开。在普通和困难难度中，卫道士可以破坏木门。免疫唤魔者的尖牙和劫掠兽的咆哮攻击。
基岩版中，可以用发射器给卫道士穿戴上护甲，但他们的外观不会发生变化，因为他们的贴图比护甲大，死后盔甲不会掉落。被掠夺者误伤也会发起反击。
JAVA 版中，敲钟会对与其距离在 48 格内的卫道士造成 3 秒的发光效果。如果一个卫道士收到攻击，周围的卫道士会对发起攻击者产生敌意。
当他们在袭击中将村民全部杀死后，会举起并摇动双手以示欢呼。
掉落物
卫道士被杀死后掉落 0～1绿宝石，8.5% 的概率掉落铁斧，铁斧有概率带有各种附魔。
基岩版中，生成于袭击中的卫道士有概率额外掉落 0～5 绿宝石、铁护甲、铁剑、铁锹、各种附魔书。其中铁制装备通常带有一定程度的损坏。
彩蛋
如果一个卫道士被命名为“Johnny”（使用命名牌、已被命名的刷怪蛋等，或者卫道士的 Johnny 标签被设为 1b），它会攻击除唤魔者、恶魂、幻术师、掠夺者、劫掠兽[仅 Java 版]和其他卫道士以外的所有生物，包括恼鬼。除末影螨‌‌[仅基岩版]、幻翼、守卫者、远古守卫者、史莱姆、岩浆怪、恼鬼、女巫和劫掠兽之外的敌对或中立生物，在受到卫道士攻击后均会尝试反击卫道士。这个彩蛋来自于恐怖电影《闪灵》。

发光鱿鱼是一种有着荧光纹理的被动型生物，生命值为 10 点（）。其生成的位置必须完全黑暗，且下方5格距离内必须有类石头方块。在 2020 年举办的 Minecon Live 中被玩家抽出，现在已经在 1.17 版本里加入。
发光鱿鱼的荧光纯粹体现在纹理上，并不像光源那样对周边环境的亮度造成实际影响。
发光鱿鱼的行为与普通鱿鱼相似，它们都会在水中漫无目的地四处游动，但发光鱿鱼还会发出一种荧光的星形粒子。暴露在空气中 300游戏刻（15 秒）后，它们会开始受到窒息伤害。
基岩版中存在幼年个体，但实际无法进行繁殖。
守卫者、远古守卫者和美西螈会主动攻击发光鱿鱼。带有穿刺魔咒的三叉戟会对发光鱿鱼造成额外伤害。发光鱿鱼死亡后，会掉落荧光墨囊。
发光鱿鱼会在被攻击后的 100 游戏刻（5秒）内停止发光，并且喷射出一种荧光墨汁粒子。在水中的发光鱿鱼会以较快的速度朝受到伤害的反方向逃跑，其速度明显快于普通鱿鱼。
在 21w40a 后，仅生成于 Y=30 高度以下的水中。
在官方的荧光鱿鱼宣传片中，盯着荧光鱿鱼可能还会造成类似“迷惑”的效果。但目前仍没有加入到游戏中，可能那只是一种夸张的表现手法。
在 2021 年的愚人节中，官方发布了“Minecraft Plus!”屏保，图标就是发光鱿鱼。
简介
史莱姆（Slime）是一种生成在沼泽生物群系的攻击型生物，偶尔也会在地下生成。
史莱姆在 Java 版 Alpha v1.0.11 中加入，在 Alpha v1.0.13 和 1.14 快照 18w43a 中修改了纹理。
生成
史莱姆生成在沼泽和红树林沼泽生物群系 50～70 层亮度等级小于等于 7 的地方，在沼泽生物群系且当天为满月时，它们的生成几率最大，而在新月时从不生成。它们只在普通沼泽和红树林沼泽中生成，而不在它的变种沼泽山丘生成。
史莱姆也可以生成在 Y 坐标 40 以下的特定的“史莱姆区块”里（不限亮度）。因此，超平坦世界中，史莱姆会大量生成，除非你在自定义中把方块层度设置为 40 格及以上。
若生物拥有渗浆，则死亡时生成 2 只中等体型的史莱姆，包括玩家。
史莱姆不会在距离以玩家为中心的半径 24 格（球形）内生成。
注：不同大小的史莱姆生成的概率取决于副区域难度，生成大、中、小型史莱姆的概率分别为 [33×(1+CRD/2)]%、33%、[33×(1-CRD/2)]%（其中CRD为副区域难度的简称）。
行为
史莱姆的尺寸由nbt 标签控制，只有 1、2 和 4 尺寸的史莱姆会自然生成，通过/summon指令可以生成 1～256（1.17 以前）或 1～128（1.17 及以后）尺寸的史莱姆。
史莱姆的各种属性都和尺寸成正比，包括移动速度，具体计算公式为 (尺寸+1)×0.43 格/秒。所以，玩家在不喝药水或调整自身属性的情况下跑不过一个尺寸大于 8 的史莱姆。
攻击
史莱姆的攻击方式为近战。

命令方块矿车（Minecart with Command Block）是一种装着脉冲型命令方块[仅Java版]或循环型命令方块的矿车[仅基岩和教育版]。
命令方块矿车在Java版1.7.2快照13w39a中加入。
命令方块矿车只能够通过指令获得或召唤出来。它不能通过矿车和命令方块合成，破坏后也只会掉落矿车。
命令方块矿车结合了矿车和命令方块的功能。运行在激活铁轨上时，它会反复运行命令（每4刻）。当它经过被红石比较器连接的探测铁轨时会输出命令方块的结果。
矿车是一种载具实体，在Java版Infdev 20100618中加入。
矿车虽然不是生物，但是如果它受到超过6点（）伤害就会被破坏，被破坏后会掉落自身。
对着矿车按下鼠标右键使用可以坐上矿车，部分生物在经过矿车附近时会进入矿车且不会主动下车。
矿车需要铁轨才能正常运行，但它不会主动移动，需要玩家在矿车上按前进键或者经过充能铁轨被给予速度或者被其它实体例如动力矿车推动才能移动。如果矿车脱离了铁轨，那么它将很难移动。
在乘坐矿车时，如果矿车从高处落到铁轨上，乘坐它的生物不会受到坠落伤害。
运输矿车、漏斗矿车、动力矿车、TNT矿车在被摧毁时除了掉落矿车外，还会分别掉落1个箱子、漏斗、熔炉或TNT。
命令方块矿车和刷怪笼矿车只能够通过/give和/summon命令获得，被摧毁后只能掉落1个矿车。
曾经mojang放出过一张有关发射器矿车的图像，但其可能只是显示在下一个快照13w02a中加入的矿车自定义显示。
[你知道吗] 矿车的英语 Minecart 和游戏名 Minecraft 非常相似，容易混淆。曾经基岩版中，命令方块矿车就被翻译为“带有命令方块的《我的世界》” ，十分搞笑，令人忍俊不禁。之后虽然被修正，但是玩家社区中有时提及此事，暗含了玩家对基岩版不认真翻译的不满。
哞菇是一种在蘑菇岛生物群系中生成的独特的牛，在Java版Beta 1.9-pre1中加入。
生命值为10点（）。
如果一对哞菇被喂养小麦，它们会繁殖。哞菇不能与牛繁殖，包括曾是哞菇的牛。
小哞菇的背上不会有蘑菇，但是长成之后就会有了。
哞菇掉落0-2皮革、1-3生牛肉（如果被火烧死则为牛排）。用剪刀为哞菇除菇会掉落5个红色蘑菇，哞菇会发出颗粒效果并变回牛，且无法再变回哞菇。哞菇被玩家或驯服的狼杀死时掉落1-3经验。
除了可以对哞菇使用桶来挤奶，也可以对哞菇使用碗来获得蘑菇煲。
哞菇会跟随手持小麦的玩家。
它们躲避悬崖、火焰、熔岩，但不会试图远离水。
在1.14后，哞菇有两个变种：棕色哞菇和红色哞菇。自然生成的都是红色哞菇。红色哞菇在被闪电击中后会生成棕色哞菇，相反棕色哞菇在被闪电击中后会生成红色哞菇。在繁殖红色哞菇时，会有极小的概率生出一只小棕色哞菇。相反在繁殖棕色哞菇时，也会有极小的概率生出一只小红色哞菇。
手拿碗或桶对着棕色哞菇右键也可以获得蘑菇煲和牛奶桶，但在喂食不同的花之后用碗右键可以获得与花对应的不同种类的迷之炖菜。在此之后，如果不再被喂食花，棕色哞菇会正常产生蘑菇煲。
概述
唤魔者是一种攻击型生物，其生命值为 24 点（× 12），会主动攻击玩家、村民、铁傀儡，并且属于灾厄村民，因此不会受到其他唤魔者的尖牙或者劫掠兽的咆哮攻击等。
唤魔者在死亡后掉落不死图腾*1（不受“抢夺”附魔影响）和绿宝石*0-1。
生成
可生成于林地府邸，这种情况下被杀死后不会再次自然生成，Java 版中就算不使用命名牌对其命名也不会被系统刷新而移除。
唤魔者也会在袭击中生成，从第五波开始，最多生成 9 个唤魔者，其中袭击第七波和第八波各有 1 个唤魔者骑着劫掠兽。
生成的唤魔者有很小概率成为袭击队长。
攻击
他有两种攻击方式（每次攻击前都会高挥双臂，冒出不同颜色的粒子）：
尖牙攻击：冒出淡紫色的粒子，与攻击对象之间的线上冒出 16 个尖牙，最远可延伸至 30 个，尖牙会对所有非灾厄村民生物造成伤害，造成 6 点（）魔法伤害。如果玩家与他距离在 3 格之内，他会在自身周围召唤两圈尖牙，外圈有 8 个尖牙，内圈有 5 个尖牙。施法冷却时间为 5 秒。
召唤恼鬼：冒出白色的粒子，唤魔者会无规律召唤 1-3 个恼鬼帮助他对抗敌人。在 Java 版中，召唤出的恼鬼在一段时间后会死亡。施法冷却时间为 17 秒
基岩版中唤魔者每次攻击 75% 的概率产生尖牙，25% 的概率召唤恼鬼。Java 版中首次攻击一定会召唤恼鬼。和平难度中唤魔者只会产生尖牙，不召唤恼鬼。
你知道吗
唤魔者会把附近蓝色的羊变成红色，并冒出橙色的粒子（在 Java 版中要求/gamerule mobGriefing 为 true）。
基岩版的唤魔者和卫道士有装备栏，可用发射器给其穿上，但只有卫道士会显示，唤魔者不会（但有效果）。
唤魔者的攻击音效来自游戏《帝国时代》中的牧师。

尖牙是由唤魔者召唤的实体。
在发动尖牙攻击前，唤魔者会冒出淡紫色的颗粒，并发出喇叭一样的声音作为警告。一些尖牙从地上冒出，然后突然合上并消失。玩家会受到6点（）魔法伤害（与难度无关）。由于其伤害属于魔法伤害，所以尖牙无视盔甲，但保护附魔和抗性提升效果可以吸收部分伤害。
尖牙会生成在不低于攻击者的脚，不高于攻击者的脚上方一格的位置。尖牙会尝试在这两者之间的不透明方块上生成，但当这个方块被固体方块挡住时会生成失败。实际上，这意味着它无法生成在深坑里或高墙顶。但是，当唤魔者在底下而目标在上面时，它会走上楼梯，反之亦然。
唤魔者通常会在与玩家所在的直线上召唤尖牙，但如果目标与唤魔者距离很近，唤魔者会以自身为中心召唤两圈尖牙。
尖牙生成会把唤魔者的施展咒语冷却时间重置为5秒，且尖牙生成冷却时间会重置为17秒。
唤魔者召唤的尖牙间有一定延迟。延迟结束前，玩家无法看见尖牙，但它不像invisible标签为1的实体，此时它仍然拥有碰撞箱。
唤魔者召唤的尖牙不会对其它灾厄村民造成伤害，但由玩家生成的尖牙会对唤魔者造成伤害。
尖牙不会被和平模式所影响。
1.13版本以上，尖牙行动方式按照一排出现，每次最多15~16个。

嘎枝是一种被玩家看见时不能移动的敌对生物。
嘎枝有两种：
注册名为 minecraft:creaking，可以使用指令 /summon minecraft:creaking ~ ~1 ~ 生成，拥有 1 点（）生命值，在白天不会消失，称持久化嘎枝；
注册名为 minecraft:creaking_transient，不可以使用指令生成，只能通过嘎枝之心生成，称临时嘎枝。
嘎枝的攻击伤害如下：
简单难度：2 点（）；
普通难度：2 点（）；
困难难度：3 点（）。
生成
持久化嘎枝可以通过指令和刷怪蛋生成，而临时嘎枝只能通过嘎枝之心生成。
掉落物
嘎枝死后不会掉落任何物品和经验。
行为
嘎枝会主动追击非创造模式和旁观模式的玩家，当被玩家看到时不会移动，此时的嘎枝不会进行攻击、移动、转身、无法被实体推动，但钓鱼竿、水等仍可以移动嘎枝，铁傀儡、疣猪兽等也可以对其击退，会受到爆炸击退。“被玩家看到”的定义为视场角为中的玩家屏幕中有嘎枝，且没有佩戴雕刻过的南瓜，若玩家用更大的视场角或在第三人称视角里可见，嘎枝仍会移动。
当嘎枝追击玩家时，移动速度与玩家疾跑速度相当，当嘎枝没有追击目标时，会以 1/22 格每秒的速度缓慢移动。
嘎枝免疫熔岩伤害，不会着火。
嘎枝无法上船，当被玩家看到时不会上矿车。
灾厄村民会主动远离嘎枝。
临时嘎枝
临时嘎枝免疫除虚空和/kill外的伤害，在任何进行试图对其造成伤害的行为时，临时嘎枝会产生灰色的粒子效果，这些粒子效果最终飘向其绑定的嘎枝之心，当临时嘎枝受到来自玩家的攻击时会轻微抖动身体。
当临时嘎枝离开对应的嘎枝之心的曼哈顿距离超过 34 格时消失，嘎枝之心也会再次生成一个临时嘎枝。
在白天且非雷雨时，临时嘎枝会消失。
临时嘎枝不会被保存和加载。
临时嘎枝不能穿过下界传送门和末地传送门。
版本信息
嘎枝在 Java 版 1.21.2 快照24w40a 版本和基岩版1.21.50.20 版本中加入。
概述
女巫是一种生成在主世界当中的敌对性生物。雷电劈在村民（包括幼年村民）身边也会使村民变成女巫，在这样情况下生成的女巫不会被系统移除，并且血量为满。
1.4.2 快照 12w40a 的版本中，女巫只会生成在沼泽小屋。1.7.2 快照 13w42a 及以后，女巫会生成在任何亮度小于 7 的地方。女巫共有26点（× 13）生命值。
攻击
女巫会向攻击对象扔出药水（中毒、虚弱、缓慢或者瞬间伤害药水）。若攻击对象与自身距离超过 8 格且不带有缓慢效果，会投掷缓慢药水；若攻击对象位于 3 格之内，且没有虚弱效果，则有 25% 的概率投掷虚弱药水；若攻击目标生命值超过8点（），则会投掷剧毒药水；如果上述条件均不满足，会投掷瞬间伤害药水。
非玩家生物中，女巫是唯一一个会投掷药水的，女巫和流浪商人是唯二会饮用药水的。
女巫会自己喝治疗药水、抗火药水、迅捷药水、水肺药水。
女巫在发现玩家后会先喝下药水，再向玩家投掷药水。
行为
女巫对魔法伤害有 85% 的抗性，包括中毒、瞬间伤害、唤魔者的尖牙和荆棘，但是凋零仍然正常造成伤害。
女巫也会在袭击事件产生，但是并不会主动攻击村民，并且会向灾厄村民和劫掠兽投掷喷溅型生命恢复药水，若灾厄村民和劫掠兽的生命值小于等于4点（），女巫还会投掷瞬间治疗药水。如果在袭击中所有村民被杀死或所有的床被破坏，那么女巫会跳跃并大笑以示胜利。
基岩版中，女巫免疫中毒效果，并且像蜘蛛那样，视线可以透过方块。仅 JAVA 版中，敲钟会使距离 48 格内的女巫带上 3 秒的发光效果，使其更容易被发现。
掉落物
女巫会随机掉落：糖，玻璃瓶，红石，火药，蜘蛛眼，荧石粉，木棍，木棍掉落概率为 25%，其他物品掉落概率为 12.5%。如果在女巫喝药水的时候杀死女巫，就会有 8.5% 概率作为装备掉落其正在喝的药水，抢夺每级增加 1% 的概率。
女巫每秒都有 1.5% 的概率在头顶生成紫色粒子。
版本信息
女巫在 1.4.1 快照 12w38a 中加入，在 1.4.1 快照 12w38b 和 1.14 快照 18w43a 中修改纹理，在 1.19 快照 22w17a 中修改了模型，现在可以渲染出袍子纹理的最下方两行像素。
相关进度
对女巫使用喷溅型剧毒药水，会获得进度“自食其果”。
概述
守卫者（Guardian）是一种生成于海底神殿的敌对生物，在 1.8 快照 14w25a 中加入，在 1.14 快照 18w43a 中修改了纹理。
守卫者只有一只眼睛，只生成于海底神殿及附近，身体呈和海晶石砖一样的深绿色。
守卫者的生命值为30点（× 15）。
攻击
守卫者有两种攻击方式：
激光——守卫者会停下来紧盯生物开始充能，且其与目标之间有一道激光相连，一小段时间后，当射线深紫色渐变为亮黄色时造成伤害。充能时如果目标退出到射程之外或激光被方块阻挡，则会停止充能，也不造成伤害。
简单难度4点（）。其中有1点（）魔法伤害，其余3点（）直接伤害。
普通难度6点（）。其中有1点（）魔法伤害，其余5点（）直接伤害。
困难难度9点（）。其中有4.5点（）魔法伤害，其余4.5点（）直接伤害。
尖刺攻击——守卫者尖刺伸出时，会对攻击者造成伤害，伤害类型是荆棘。
其伤害值为2点（），困难模式下是3点（）。
行为
守卫者会在水中游动，攻击任何进入视线的玩家、鱿鱼（以及在 1.17 加入的发光鱿鱼和美西螈）。它们的游泳方式是快速甩动尾巴进行突然的冲刺。游泳时它们的尖刺收回。守卫者在玩家靠近或受到攻击时逃跑。不游泳时它们会缓慢下沉，并伸出它们颤抖的尖刺。守卫者的眼睛会看向附近的任何玩家，并且始终直视它们正在攻击的目标。在 Java 版中，它们的眼睛不会跟随完全没有装备盔甲的隐身玩家，也不会攻击他们。在隐身状态下，穿着一件盔甲的玩家也不会被攻击。
守卫者离开水时会到处扑腾，并愤怒地向玩家发出吱吱声。它们在空气中不会窒息，在陆地上可以一直存活下去。在陆地上它们的尖刺总是伸出的。
和其他水生生物一样，守卫者也会受穿刺附魔影响。
掉落物
守卫者死亡时会固定掉落 0-2 个海晶碎片，抢夺附魔每等级增加 1 掉落数量，最多为 5。
另外有 40% 几率掉落 1 个生鳕鱼（如果被烧死会掉落 1 个熟鳕鱼），每一级抢夺会增加 0-1 掉落数量，在带有抢夺III时最多掉落4条鱼。
有 40% 的几率掉落海晶砂粒，每一级抢夺会增加 0-1 掉落数量，在带有抢夺III时最多掉落 4 个。
有 20% 的几率两者都不掉落。

小火球（Small Fireball）是烈焰人发射的火球实体。
烈焰人发出的火球会直线飞行，不受重力影响。火球会在击中的地面生成火。火球如果击中了对火不免疫或没有防火效果的实体，在所有难度中均会造成5点（）的弹射物伤害，实体会着火5秒，通常受到额外的4点（）火焰伤害。与恶魂火球不同，此火球的弹道不能被近战或箭击偏转，但是，它能被盾牌抵挡，这可以防止玩家受到火球的伤害。
通过/summon指令和NBT标签，可以自定义生成的烈焰人火球的前进方向和加速度。
简介
尸壳在露天的沙漠生物群系生成，是僵尸在沙漠群系的变种，生命值为20点（），自带护甲值为2点（）。同时也有小尸壳，像小僵尸一样拥有较快的移动速度，并且永远不会长大。

行为
尸壳的视野有 32 格，不同于一般怪物的 16 格。
尸壳属性与普通僵尸相似，但有以下不同：
不会在阳光下燃烧；
攻击到的生物会获得饥饿减益，此效果的持续时间会根据游戏难度而不同（7×难度指数/秒）。Java 版中手持物品的尸壳攻击失去了饥饿效果，然而基岩版中尸壳的攻击永远会附带饥饿效果；
它们没有对应的村民变种；
它们外观上比普通僵尸高 2 像素，但具有相同的碰撞箱；
它们的声音比普通僵尸更刺耳；
它们会尝试避开水，但是尸壳强行待在水里超过 30 秒后尸壳会转化为僵尸；
试图使用发射器给予它们装备是不可行的；
在创造模式物品栏中有尸壳的刷怪蛋，这种刷怪蛋只会生成尸壳；
不会在僵尸围城事件中出现。

山羊生成于积雪的山坡、尖峭山峰和冰封山峰生物群系，生命值为10点（）。
在Java版1.17快照21w13a版本加入。
山羊具有较强的跳跃能力，每隔30-60秒会高跳一次，最多能跳10格高，并且山羊受到的摔落伤害相较正常水平少10点。与其他生物相同，山羊无法在蜂蜜块上跳跃。山羊会避免进入细雪。
山羊不会反击伤害它们的生物，其受到伤害时会四处逃窜。
每隔30-300秒，山羊会随机选择半径16格内的生物（其他山羊和恶魂除外，但也有可能会误伤）作为冲撞目标并开始蓄力。尖叫山羊会更加频繁地发起冲撞，每隔5-15秒就会试图冲撞一次。山羊发起冲刺需要满足以下条件：
山羊与某一可冲撞目标之间存在至少4格空间，且不被非空气方块阻挡；
冲撞目标在山羊蓄力期间不移动。
山羊快速冲向目标后会对其造成较强的击退。成年山羊的击退距离可达9格，幼年山羊则为4.5格。被山羊冲撞的生物不会因此反击。
若山羊在蓄力和冲刺过程中受到攻击、撞到选定目标以外的生物或是发现手持小麦的玩家，它就会瞬间停止冲刺。
在Java版中，山羊冲撞潜影贝时，脑袋总会在潜影贝的外壳卡住一段时间，且无法对其造成伤害和击退。
山羊有2%的概率生成为尖叫山羊，其声音与一般的山羊不同，且相比普通山羊有更高的几率发起冲撞攻击。繁殖一对普通山羊也有2%的几率产下尖叫山羊。
对山羊使用桶可以挤奶，并获得奶桶。
攻击伤害在简单模式下为3点（），在普通模式下为5点（），在困难模式下为7点（）。
在1.18-exp1前，山羊会在风袭丘陵/山地生物群系及其变种中生成。在1.18-exp2~21w39a中，山羊还会生成于裸岩山峰生物群系。但在21w40a后，山羊不再生成于裸岩山峰生物群系。

简介
一种生成于下界的史莱姆，免疫火焰伤害。当尺寸大于 1 的岩浆怪被杀死时，它会分裂成 2～4 个较小的岩浆怪，尺寸减半并向下取整，并且如果最小尺寸的岩浆怪被杀死，它们会像其它生物那样被杀死并掉落战利品。
岩浆怪在 Java 版 Beta1.9-pre1 中加入，在 1.14 快照 18w43a 中修改了纹理。
行为
岩浆怪的生成不受亮度等级限制，可以生成在下界的某些地点，体型受区域难度影响。
大型岩浆怪需要 3×2.5×3 的空间。
中型岩浆怪需要 3×2×3 的空间。
小型岩浆怪需要 2×1×2 的空间（上部为透明方块仅需 1×1×1）
岩浆怪没有寻路能力，只是向着目标前进，故可能掉下悬崖或无法绕过墙壁。
巨人（Giant）是不能被自然生成的、与僵尸相似的巨型生物，在Java版Indev 0.31 20100205中加入，在1.14快照18w43a中修改了纹理。
巨人在死亡时掉落5经验。
没有生物群系允许生成巨人，因此它们不会自然生成。
巨人的生成需要亮度等级在8以下且在11以上。这使得它们不能自然生成或由刷怪箱生成。
可以使用/summon giant这样的命令来召唤巨人。‌‌[仅Java版]

目前巨人没有AI（在1.8版本被移除AI），因此他们在生成时总会面朝南，永远不会移动、转头、追逐玩家。即使玩家攻击它，它也只会像其他生物一样被击退，不会有任何其他反应。它们的视角方向只能通过命令修改。
巨人在Xbox 360版和Xbox One版中真实存在，但是它们的刷怪蛋需要修改才能放到玩家的物品栏里。

幻术师（Illusioner）是一种尚未使用的敌对生物，也是一种灾厄村民。它可以隐身并召唤无敌的分身。
幻术师在Java版1.12快照17w16a中加入。
幻术师目前只能通过/summon illusioner命令生成。
被玩家杀死时，幻术师会有8.5%的几率掉落它们的弓。如果使用带有抢夺附魔的武器杀死幻术师，弓的掉落几率为抢夺I的9.5%、抢夺II的10.5%以及抢夺III的11.5%。它们掉落的弓通常有一定程度的破损。
如果幻术师是一个袭击队长，那么被杀死后掉落一个灾厄旗帜。
幻术师会攻击12个方块内的玩家、村民、流浪商人和铁傀儡。它可以使用咒语和弓来攻击。
幻术师是唯一一种能够透过墙壁发现目标的灾厄村民。
幻术师十分敏捷，总是试图让自己与玩家之间保持一定的距离；若与玩家离得太近，它们便会后退。
幻术师有两种咒语：一种能使攻击对象失明，另一种能召唤分身并使本体隐形。
尽管幻术师无法自然生成，但袭击一旦发生，由命令生成的幻术师就会参与其中。这也意味着幻术师将会有机率在袭击中成为队长，携带灾厄旗帜。
只有在区域难度大于2时，幻术师才会使用失明咒语。
每当幻术师遇到一个新的攻击对象，它会举起手臂、发出声音并生成黑色烟雾施法，使其受到持续20秒的失明效果。幻术师不会在同一个攻击对象身上多次施放这个咒语，除非它先将注意力转移到其他攻击对象上，再转移回原来的攻击对象。
这个咒语会重置幻术师的咒语冷却时间到1秒，以及重置失明咒语的冷却时间到9秒。
在幻术师参与到战斗之中时，它会先举起手臂、发出声音并产生蓝色烟雾施法，使自己得到持续60秒的隐身效果，并且每当隐身效果快结束时会刷新效果。
当幻术师隐身时——不论是通过任何方式——它都会召唤4个假的分身。
这些来自于隐身本体的分身会轻微摇晃，只有幻术师受到攻击它们才会消失。这些分身面对的方向与本体相同，而且有时候只要本体稍微移动了一步，分身就像是穿过了墙壁、天花板和楼梯。每当本体使用弓攻击时，它们会使用射击的动画，虽然只有本体才能射击和造成伤害。当幻术师的分身生成时，其四个分身有时并不会离开本体，而是四个分身会重叠和本体一起，由于分身无敌，此时状态的幻术师可以弹开箭矢。
当幻术师本体受到伤害，其幻影会迅速返回本体位置，又迅速到达新的位置，发出攻击信号。当幻术师并没有受到攻击时其分身有时也会自行瞬移到新的位置。
如果一个隐身的幻术师得到发光效果，所有的分身都会发光，而本体仍然处于隐身状态。
一旦幻术师的隐身效果结束后，分身会消失。
这个咒语会重置幻术师的咒语冷却时间到1秒，以及重置隐身咒语的冷却时间到17秒。

在 2017 的 Minecon Earth 中被玩家抽出，现已在正式版 1.13快照18w07a 加入。添加的新型亡灵类敌对生物（最初在快照中加入的时候不是亡灵生物，是的，太阳高度角多大都不会自燃，也就是说如果晚上不睡觉，就算捱到白天，玩家也照样会受到它的攻击）。
幻翼会在玩家至少 3 天没有睡觉且暴露在月光下时在主世界和末地生成（1.13快照18w09a 以前）；自 1.13快照18w09a 版本起，它只能在主世界生成，不会在末地生成，且一次最多生成 6 只。
幻翼在刚加入时除经验球外没有任何掉落物，但后来加入了可以用于修复鞘翅的幻翼膜。

概述
恶魂（Ghast）是下界的一种巨大的、浮空的、会向玩家发射火球的生物。恶魂是游戏中自然生成的第二大生物，仅次于末影龙。
可以生成于下界荒地、灵魂沙峡谷和玄武岩三角洲。生成条件为下方为固体方块，且5×4×5的范围内是空气时才会生成。
生命值为10点（）
攻击
攻击力：
火球碰撞伤害：6点（）（无视难度）
简单难度火球爆炸伤害：9点（）
普通难度火球爆炸伤害：17点（）
困难难度火球爆炸伤害：25点（× 12.5）
恶魂会向玩家发射恶魂火球，玩家可以通过击中火球（近战与远程皆可）来将其反弹回去。
恶魂火球击中方块或玩家会爆炸，并会使周围方块起火，其的威力为1，承受该威力的最低保证抗性为4.034，因此恶魂无法摧毁爆炸抗性高于此值的方块，例如圆石，下界砖块，玄武岩等。
行为
恶魂在一般情况下会漫无目的地在下界游荡，并且发出一个高亢的类似于猫叫的声音声音来自c418家的猫，并且闭上双眼和嘴巴。当它发现玩家并尝试攻击时，会将血红的眼睛和嘴巴张开。
恶魂拥有100格的搜索范围、爆炸的攻击力以及可观的移动能力。不像其他敌对的生物，恶魂不会主动接近玩家（也不会试着远离）。
恶魂只有看见玩家时才会发射火球。任何固体方块（包括透明方块）都可以阻挡恶魂视线。
在Java版中，只会与玩家敌对，任何生物皆不会尝试反击恶魂；在基岩版中，只会与攻击它的生物和玩家敌对，任何生物（除女巫）皆不会尝试反击恶魂。
掉落物
死亡后掉落恶魂之泪和火药。有抢夺加成，前提是需要使用带抢夺附魔的武器击杀恶魂或打中恶魂的火球，并炸死恶魂。当恶魂被玩家反弹回来的火球炸到时会受到1000点（× 500）的伤害。如果被其他恶魂发射的火球（非反弹）不会有这个效果。如果玩家将火球弹回去并杀死一个恶魂会完成一个挑战：见鬼去吧。
将恶魂带到主世界并击杀是会取得挑战进度：脆弱的同盟。
在1.17版本之后，用望远镜观察恶魂可以获得进度“那是气球吗？”。
加入版本&历史
恶魂在java版Alpha v1.2.0中加入。
在Java版1.0.0快照Beta 1.9-pre1前，恶魂的火球的纹理是雪球，并且没有掉落物。而在这个快照，恶魂的火球变成了独立图标（类似于烈焰人的火球纹理），并会掉落恶魂之泪。
在1.14快照18w43a中，恶魂和恶魂的火球的纹理被修改。
小特性
船可以在恶魂上面行驶，但是玩家却不能恶魂上面行走，被点燃的TNT也无法在恶魂上面停留。

简介
恼鬼（Vex），曾被基岩版（微软的外包）译为威克斯。
恼鬼在 Java 版 1.11 快照 16w39a 加入。
虽然有恼鬼刷怪蛋，但不会在世界中自然生成，而是由唤魔者召唤而出。
恼鬼手持铁剑，身体发光，会在空中漫无目的地飞行，可以穿过所有方块，并且不会受到火焰与熔岩的伤害。恼鬼会主动攻击玩家和由唤魔者指定要攻击的生物，若受到其他生物的攻击也会进行反击（基岩版中，恼鬼也会主动攻击村民，铁傀儡和流浪商人），发动攻击时，其身体上会出现红色的裂纹状的纹理，并快速飞向目标。恼鬼在冲向目标的过程中不会拐弯，所以向侧向闪避就很容易躲开攻击。
恼鬼的血量为14点（），死后只会掉落 5点经验值（但如果有武器有抢夺附魔，会有很低的概率掉落铁剑，因为铁剑掉落的概率原本是 0%，但每级抢夺附魔会增加 1% 掉落概率），由唤魔者召唤的恼鬼会在一定时间后开始受伤并最终死亡。

悦灵生命值为20点（），自然以1-3个为一组生成于掠夺者前哨站的深色橡木笼子或林地府邸的监狱房中。在2021年举办的Minecon Live中被玩家抽出，现在已经在1.19快照22w13a里加入。
悦灵会发光，但并不会改变周围方块的实际亮度。
可以将物品给予悦灵，它会搜寻32格（Java版）/64格（基岩版）范围内的同种掉落物并将其给予玩家，冷却3秒后会再次进行搜寻。空手右击悦灵可以收回它所持有的物品。不死图腾在悦灵手上能正常生效（当它的生命值归零时手持不死图腾会正常复活，同时图腾会被消耗）。
悦灵死亡时只会掉落它手上的物品。
对悦灵附近的音符盒右击会给悦灵传达一个信号，可以让悦灵标记为最喜欢的音符盒，此时它会把物品丢在这个音符盒上。在音符盒上放置羊毛可以阻隔信号。
悦灵在播放中的唱片机附近会跳舞，此时如果它获得紫水晶碎片时，会生成另一只悦灵，然后被复制的悦灵和新生成的悦灵进入5分钟冷却时间，此时它们不会通过此方式生成新的悦灵。
悦灵无法潜水，但是会尝试捡起水中的物品。
悦灵会每秒回复2点生命值（）。

拴绳结（Leash Knot）是玩家使用拴有生物的拴绳拴对着栅栏右击后生成的实体。
当动物被拴绳拴住时，在栅栏右击即可将玩家手持的所有拴绳拴在该栅栏上同时会显示一个可见的结，在一个栅栏上可以有多个结。被拴在栅栏上的生物会试图留在栅栏附近 5 格之内。
当拴绳结受到一些伤害时（例如爆炸伤害、弹射物伤害等），会被破坏掉，并在被拴住的生物处掉落拴绳。
拴绳的最大有效长度为 10 米，超过拴绳就会断裂，拴绳结会掉落为物品。
当栅栏被破坏时，拴绳结会存在一段时间后消失。
使用/summon指令生成的没有拴在栅栏上的拴绳会在一段时间后消失，且不会返还拴绳。
拴绳结在 Java 版 1.6.1 快照13w16a 版本和携带版 Alpha0.15.0 build 1 版本中加入。

掠夺者是以弩（有概率附魔，取决于难度）为武器的敌对生物，灾厄村民的一种，血量为24点（× 12），与骷髅不同，掠夺者不会起内讧（但是劫掠者误伤卫道士会起内讧)。主要生成于掠夺者前哨站。
掠夺者也是灾厄巡逻队中的主要成员，五个游戏日过后灾厄巡逻队不论昼夜均可生成，成员包含卫道士。
击杀掠夺者有概率掉落弩，击杀首领会获得灾厄旗帜和不祥之兆效果。仅基岩版中击杀生成于袭击中的掠夺者会掉落0～1个绿宝石。
袭击是灾厄村民进入村庄攻击村民的事件，掠夺者也是参与者之一，当他们将村民全部杀死后，会举起并摇动双手以示欢呼。
当掠夺者的弩耐久值消耗完后，掠夺者的双手会什么也不拿，且不会攻击村民。但村民仍然会躲避掠夺者。基岩版中掠夺者弩的耐久度不会消耗。
仅基岩版中，掠夺者沉入水中时会启用近战攻击。除此以外还可以用发射器使掠夺者穿戴上装备，他们的外观不会因此发生改变，但护甲的防御效果和附魔可以作用于他们。杀死后不会掉落相应装备。
掠夺者在Java版1.14快照18w43a中加入，在1.14.4-pre4中修改了纹理。

投掷末影珍珠时生成的实体，内部空间ID为ender_pearl。
当末影珍珠被垂直往上扔，它能穿越约30格。当末影珍珠被以40°角扔出，它能向前穿越约50格。
末影珍珠会与 蜘蛛网、线、 藤蔓、草丛、枯萎的灌木、蕨、花、所有种类的树苗、所有种类的蘑菇、所有种类的矿车、船和下界传送门碰撞。
由玩家投掷的末影珍珠会在NBT标签Owner中记录该玩家的UUID。末影珍珠在碰撞到方块或实体后即会将UUID相同的玩家传送至碰撞位置。
掉进虚空里的末影珍珠并不会传送玩家（但掉入水或岩浆的末影珍珠会将玩家传送至水或岩浆底部）。
当末影珍珠把玩家传送到1.5格以下的空隙里，玩家会进入防止窒息状态（俗称爬行状态）。
在1.16.2 Pre Release 1及其之后的版本中，被掷出的末影珍珠会受到气泡柱的影响。
