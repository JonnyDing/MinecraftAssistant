鲑鱼生成于冷水海洋和冻洋生物群系及它们的深海变种中，但不同于其他三种鱼，鲑鱼还能生成于河流与冻河中。
鲑鱼被杀死后掉落生鲑鱼和骨粉（5%的概率），被烧死后掉落熟鲑鱼。
所有四种鱼离开水后都无法存活。如果离开了水，它们会像守卫者一样扑腾一段时间，最后像鱿鱼一样窒息而死。
所有四种鱼都可以被装进水桶，变成对应鱼的鱼桶。
鲑鱼在Java版1.13快照18w08b中加入。
21w40a后，仅生成于Y=50至64高度之间的水中。

鸡是一种自然生成于主世界草方块上的友好生物，生命值为4点（）。在自然生成时有5%的概率替换成幼年鸡。
鸡在Java版Alpha v1.0.14中加入，在1.14快照18w43a中修改了纹理。
通过右击砸鸡蛋的方式，有1/8会砸出1只幼年鸡，而在砸出幼年鸡的基础上，又有1/32的几率砸出4只幼年鸡四黄蛋。
掉落物：
鸡被杀死后掉落0-2个羽毛和1个生鸡肉，烧死掉落1个熟鸡肉。被玩家或驯服的狼杀死时会掉落1-3。
与其他幼年动物相同，杀死幼年鸡不会掉落任何物品，包括经验。
可喂食小麦种子、甜菜种子、西瓜种子或南瓜种子以繁殖后代（12w36a加入）。幼年鸡需要20分钟长大，给幼年鸡喂食前文所提到的种子可以减少10%的生长时间。由于两种特别生成鸡的方式，在MC中很难判断鸡是卵生还是胎生动物
鸡会每隔5-10分钟生出一个鸡蛋，除非它是组成鸡骑士的一部分。鸡被杀死时，尸体没消失的那一瞬间，它仍然可以下蛋。
从高处下来时，鸡会扇动翅膀来消除掉落伤害（Alpha v1.0.17加入）。
鸡会被野生的豹猫和狐狸攻击。
鸡骑士是由一只小僵尸（及其变种，包括僵尸猪灵）与一只鸡的罕见组合，在 Java 版 1.7.4 快照 13w49a 中加入。
在 JAVA 版中，也可以通过指令召唤：
鸡骑士具有小僵尸的杀伤力和速度以及鸡的无摔落伤害的双重优点，这使得鸡骑士比普通的小僵尸更难对付。
鸡骑士中的鸡会以小僵尸的速度移动，它不会下蛋。
鸡骑士采用小僵尸而不是鸡的 AI，因此行为和小僵尸类似。
当鸡骑士被命名为 Dinnerbone 时，小僵尸会倒立，但鸡却不会。
当鸡骑士进入水中时，小僵尸会和鸡分离。
MC 中概率最低的事件就是遇到穿着附魔钻石套，拿着附魔铁剑的村民鸡骑士。概率为 1.33*10∧-24，基岩版中，骑着棕色皮毛熊猫的穿附魔钻石套，拿铁剑的村民鸡骑士出现概率更小。
基岩版中自然的状况下（非指令），除了鸡外，小僵尸还可以骑在猫、狼、熊猫、马等甚至僵尸及其变种身上离 大 谱。
生成于丛林的友好生物，生命值为6点（）。
在Java版1.12快照17w13a中加入。
Minecraft 1.12 pre3版本后，玩家可通过给鹦鹉吃小麦种子、甜菜种子、南瓜种子、西瓜种子驯服，
而给鹦鹉喂食曲奇会使鹦鹉获得45秒的中毒减益并造成3.4×10^38点伤害，这会立即杀死鹦鹉（Java版）/会获得1000秒中毒（致命）减益。显示消息为 鹦鹉 被魔法杀死了。
（因为在1.12的较早快照中，给鹦鹉喂食曲奇可以驯化鹦鹉，但现实中鹦鹉并不能食用曲奇（更准确来讲是曲奇上的巧克力）和鳄梨，所以就有了这个彩蛋。切勿随意投食）
在1.21的24w12a快照中，因为官方的一个疏忽导致喂食曲奇会导致鹦鹉死亡被改为了喂食胡萝卜。
掉落物：
1-2个羽毛。
1-3（被玩家或驯服的狼杀死）。
驯服后可以对它按右键使它坐下，坐下后不会跟随玩家移动。
驯服的鹦鹉有可能飞到主人肩膀上，但玩家跳跃，受到攻击，睡觉和进入飞行状态都会使鹦鹉掉下来，目前鹦鹉无法繁殖，也不存在相应幼年个体。
如果在鹦鹉周围将音乐唱片放入唱片机的话，它们就会跳起舞来，但坐在玩家肩膀上的鹦鹉不会跳舞（Java版）。
鹦鹉一共有5种颜色：红色、绿色、蓝色、青色、灰色。
实用价值：
鹦鹉会模仿玩家附近的敌对生物的游荡声音，对一些无声游荡的生物也会有声音（例如苦力怕，鹦鹉会模仿即将爆炸的嘶嘶声），在发出声音时会盯着敌对生物的方向，利用这点可以更方便地在地下寻找到地牢。
在1.17版本之后，用望远镜观察鹦鹉可以获得进度“那是鸟吗？”。
