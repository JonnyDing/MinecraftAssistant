概述
唤魔者是一种攻击型生物，其生命值为 24 点（× 12），会主动攻击玩家、村民、铁傀儡，并且属于灾厄村民，因此不会受到其他唤魔者的尖牙或者劫掠兽的咆哮攻击等。
唤魔者在死亡后掉落不死图腾*1（不受“抢夺”附魔影响）和绿宝石*0-1。
生成
可生成于林地府邸，这种情况下被杀死后不会再次自然生成，Java 版中就算不使用命名牌对其命名也不会被系统刷新而移除。
唤魔者也会在袭击中生成，从第五波开始，最多生成 9 个唤魔者，其中袭击第七波和第八波各有 1 个唤魔者骑着劫掠兽。
生成的唤魔者有很小概率成为袭击队长。
攻击
他有两种攻击方式（每次攻击前都会高挥双臂，冒出不同颜色的粒子）：
尖牙攻击：冒出淡紫色的粒子，与攻击对象之间的线上冒出 16 个尖牙，最远可延伸至 30 个，尖牙会对所有非灾厄村民生物造成伤害，造成 6 点（）魔法伤害。如果玩家与他距离在 3 格之内，他会在自身周围召唤两圈尖牙，外圈有 8 个尖牙，内圈有 5 个尖牙。施法冷却时间为 5 秒。
召唤恼鬼：冒出白色的粒子，唤魔者会无规律召唤 1-3 个恼鬼帮助他对抗敌人。在 Java 版中，召唤出的恼鬼在一段时间后会死亡。施法冷却时间为 17 秒
基岩版中唤魔者每次攻击 75% 的概率产生尖牙，25% 的概率召唤恼鬼。Java 版中首次攻击一定会召唤恼鬼。和平难度中唤魔者只会产生尖牙，不召唤恼鬼。
你知道吗
唤魔者会把附近蓝色的羊变成红色，并冒出橙色的粒子（在 Java 版中要求/gamerule mobGriefing 为 true）。
基岩版的唤魔者和卫道士有装备栏，可用发射器给其穿上，但只有卫道士会显示，唤魔者不会（但有效果）。
唤魔者的攻击音效来自游戏《帝国时代》中的牧师。
版本信息
唤魔者在 Java 版 1.11 快照16w39a 版本和携带版 1.1.0 快照1.1.0.0 版本中加入。
唤魔者在 Java 版 1.14 快照18w43a 版本、1.19 快照22w17a 版本、基岩版 1.10.0 快照1.10.0.3 版本中修改了纹理。
