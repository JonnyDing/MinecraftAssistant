洞穴藤蔓在Java版1.17快照21w05a中加入。
手持发光浆果对准完整方块的底面右击产生洞穴藤蔓植株。在洞穴藤蔓下方为空气方块，且尚未达到它的最大长度（介于2-26之间）时，它会向下生长，新生长出的每格洞穴藤蔓有1⁄9的概率附带发光浆果。
长有发光浆果的洞穴藤蔓亮度为14，对着带有发光浆果的洞穴藤蔓按下使用键或者将它破坏可以获得1个发光浆果。
站在洞穴藤蔓的地方并按住跳跃键就可以爬上它。如果洞穴藤蔓的后面有固体方块，按下移动键也可以攀爬。在洞穴藤蔓上潜行可以让玩家悬浮在所在格上。
破坏未结籽的洞穴藤蔓没有任何掉落物，不过在基岩版中，使用附有精准采集的工具破坏它可以获得一个发光浆果，且破坏结籽的洞穴藤蔓则可获得两个。
在java版中向尚未达到最大长度的洞穴藤蔓的尖端使用剪刀可以使其停止生长。
骨粉和沾有花粉的蜜蜂可以使洞穴藤蔓结籽，但未结籽的洞穴藤蔓不会自然长出果实。
结籽的洞穴藤蔓能允许蜜蜂进行采蜜。
简介
活塞可以徒手破坏，但用镐破坏它们速度更快。镐也是破坏活塞臂的合适工具，尽管它几乎完全由木头制成。
特性
活塞在被放置时永远朝向玩家。当其被充能时，活塞的木质表面 （活塞头）会立刻开始伸出，且最多推动 12 个方块，伸出会持续 2 个游戏刻或 1 个红石刻（0.1 秒，不计延迟）。另外，这个过程会发出以活塞为中心 31×31×31 的区域内都能听到的声音。在伸出的活塞头方向上的任何实体将会随着方块被一起推动，如果实体没有地方可以去，方块会被推进生物里，如果该方块是非透明方块会造成该生物窒息。
当活塞解除充能，它会简单地通过拉回活塞头来缩回，缩回是立刻发生的，但缩回过程持续 2 个游戏刻或 1 个红石刻（0.1 秒，不计延迟）。
活塞推动一个粘液块会向活塞朝向方向弹飞它移动中碰到的任何实体。
在基岩版中，附着在墙上的方块（比如拉杆）可以放在活塞或粘性活塞上。‌‌
局限性

部分方块不能被活塞推出，或者无法在推出后保持完好。

铁轨是一个特殊情况。只要它们的新位置是在一个固体方块的顶端，并且此固体方块不被同时移动时，就可以被推动。铁轨被推动后，会像被手工放置一样重新调整指向。
此情况的一个例外是，当铁轨和支撑它的方块顶着两个平行的活塞，然后活塞同时伸出时，铁轨会保持附着。无法使用一个活塞同时移动粘液块和其上的铁轨，甚至使用两个同步活塞推动也不行。
充能
活塞可以使用多种方法充能：
如果红石线是直线形指向活塞，而且线不自动弯向活塞。‌‌（仅 Java 版）
活塞能被直接相邻的已充能方块充能，无论是强充能还是弱充能。
活塞能被直接相邻的红石火把充能。
活塞能被它上方一格任何已充能方块充能，包括它上方的“激活空间”（如果普通活塞或粘性活塞朝向上，且活塞头上有一个红石块，当充能时它会伸出, 但是当能量来自旁边或背后时它不会缩回）。然而在活塞接收到方块更新前不会伸出或缩回，这个属性被称做半连接性，可以用来制做一个 BUD 开关。
因为活塞是一个透明的方块，中继器不能将能量传送过活塞。
倒置的活塞不能被它上方已充能方块充能，除非它已经伸出。
无头活塞（Java 版）
当一个活塞（粘性活塞）充能时，一个爆炸物刚好爆炸，且破坏范围只破坏到活塞头，就得到了一个无头活塞。
需要红石信号进行充能，若断开，则会将前面的方块替换为活塞头，然后收回可以破基岩。
在 1.13 以上的版本中，可以用调试棒来实现。
活塞与粘液块
当活塞推一个粘液块的时候，相邻的方块也会和粘液块一起移动，除非一个非活塞可移动的方块阻止了被粘液块“带走”的方块。这些方块可能会依次推动其他方块，而不单是在活塞前面的方块。举例来说，一个在地上的粘液块会试图移动它自身下方的地面方块，从而在移动的方向上依次推动附加的地面方块，就像它被活塞直接推动一样。
带釉陶瓦是一个例外，在毗邻的粘液块被移动时，带釉陶瓦不会移动。
当粘液块被相邻的粘液块移动时会发生同样的情况。举例来说，当一个单独的活塞作用于一个 2×2×2 正方体的粘液块里面的任何一个方块时，它能做为一个单位被整体推动或拉动。
当粘液块和活塞不能移动的方块相邻时，它会忽略不动的方块。但是如果相邻的方块可被移动但被不动的方块挡住了粘液块将不会移动。
活塞可移动的最大 12 个方块依然适用。举例来说，一个 2×2×3 的粘液块可以被一个粘性活塞推动，而再相邻其他可移动方块就不行了。
活塞不能通过粘液块的“挂钩”构造来移动自身，但是可以用多个活塞建造自动行走装置。

浮冰（Packed Ice）是一种较为罕见的、不可再生的（1.13后可再生）不透明方块，于1.7.2快照13w36a正式被加入。它可在罕见的冰刺平原生物群系中找到。它是一种固体方块，因此可以在浮冰的上面放置任何物品。
浮冰的硬度为0.5，爆炸抗性为2.5。任何带有精准采集附魔的工具开采浮冰都可以获得它本身。如果不是使用带有精准采集附魔的工具开采浮冰的，则不会掉落任何东西。使用钻石镐和金镐是最快的，为0.1秒/个。
浮冰有些滑，因此会让实体和物品也会滑动。在水流下面放浮冰来增加速度也是允许的。当一个非完整方块放在浮冰上面时，方块上面会像覆盖了雪一般的滑。然而，如果在浮冰上面放置灵魂沙，行动的速度只会变得更慢，而不是变得很快。
不像普通的冰，生物可以生成在浮冰上。这使得基于被推动的怪物可能滑出冰块这一点设计的紧凑型怪物农场得以实现。并且，浮冰永远不会融化。
在浮冰下方放置灵魂沙时，在浮冰上行走会比在灵魂沙上行走更慢。
注意：由于某些BUG，在1.7.2至1.8之间的版本，使用效率魔咒的工具采集不会加速浮冰的采集。该BUG在1.8版本被修复。

海带是一种生成于海洋生物群系的非固体植物方块。其物品形式可以被烧炼为干海带，烧炼后方可食用。
海带在Java版1.13快照18w07a中加入。
海带会自然生成于除暖水海洋外的所有海洋生物群系，常和海草一起生成。
海带可以用任何工具或徒手采集，破坏海带茎秆的一部分会摧毁植株其上所有部分，每一格茎秆均有一定几率掉落海带。
海带可在多种方块上种植，但必须在水下，其上方一格为水源或流动的水时才会生长。当其上方一格为流动的水时，海带会在生长至这一格时将其转化为水源。玩家或发射器皆不能移除生长有海带的水方块，除非先破坏其内的海带。
海带的生长没有光照等级的要求，它们可以在完全黑暗的环境中生长。它们也不需要开放的空间。

海晶灯是一种生成在海底神殿和海底废墟中的照明方块，照明等级为 15，为游戏中照明等级最高的方块之一。海晶灯只可以用精准采集附魔的任意工具采集，否则仅掉落 2-3 个海晶砂粒。原版材质下表面贴图会有波纹式的动画效果，所以是我的世界中最受欢迎的装饰方块之一。海晶灯也可以用来参与建造潮涌核心周围的结构。
海晶灯在 Java 版 1.8 快照 14w25a 中加入。
海晶石是自然生成于海底遗迹的石质方块，颜色会缓慢变换。可以用 4 个海晶碎片有序合成。
海晶石的一种用途是作为一个有一定爆炸抗性的建筑或装饰性方块。普通海晶石有动态材质，可以作为独特的装饰。普通海晶石的裂纹的颜色在水色、绿、紫、及靛蓝之间缓慢地循环变化。普通海晶石有 22 次不规则的材质变化，每一次变化都需要消耗 15 秒（300 游戏刻），22 次变化一共需要消耗 5 分 30 秒（330 秒，6600 游戏刻）。
海晶石建成的结构也可以激活潮涌核心，效果取决于结构的大小。
海晶石可以用来合成海晶石台阶等建筑方块。
海晶石在 Java 版 1.8 快照 14w25a 中加入。
海晶石的变种，带有锯齿状的花纹，同样自然生成在海底遗迹。海晶石砖形成遗迹下方的 23 根柱子，并整个结构中随处可见。
海晶石砖可以通过 9 个海晶碎片合成。
海晶石砖可以用来激活潮涌核心，也可以用来合成各种建筑方块。
海晶石砖在 Java 版 1.8 快照 14w25a 中加入。

海泡菜是以小群落生成的，可以在水下发现的静止的仿生物方块。
自然生成于海泡菜群落在温暖的海洋的底部，也可以在珊瑚块的顶部找到它。
海泡菜可以放在大多数固体方块的顶部，每个方块最多可以放4个，类似于海龟蛋。它们在水下时会产生光亮，第一个会产生6光照等级，之后每增加一个会增加3光照等级。
骨粉可以在海泡菜上使用，使它们在水下和珊瑚块顶部蔓延，这使得其可再生。 如果骨粉使用在珊瑚块以外的任何区域或者暴露在空气中，那么海泡菜就不会蔓延，但这仍会消耗骨粉。
海泡菜在Java版1.13快照18w14b中加入。

海龟壳（Turtle Shell）是一种酿造原材料，用于酿造神龟药水，也可作为头盔使用，戴上它的玩家会获得 10 秒的水下呼吸状态效果，由鳞甲合成。
穿戴海龟壳后获得的水下呼吸状态效果仅当玩家处于水下时才会倒计时，使得玩家可以在水下总共停留 25 秒而不受窒息伤害。配合水下呼吸魔咒，玩家在水下可呼吸的时长最大能延长至 70 秒。
装备海龟壳会增加玩家 2 点（）盔甲值，与金质、锁链和铁质头盔所给予的盔甲值相同。
海龟壳的耐久仅次于钻石头盔和下界合金头盔。

海龟蛋是一种能够孵化出幼年海龟的蛋，一个方块中最多可放置4只海龟蛋。
当喂食海草给两只海龟时，其中一只海龟会在沙子中挖洞，并在几秒钟后产下含有1-4个蛋的海龟蛋方块。海龟蛋需要放置在沙子或红沙上才能进行孵化。孵化成功之前，海龟蛋会经历3个孵化阶段，每经历一个阶段，海龟蛋上都会多出一些裂纹。孵化的时间各不相同（通常需要几个晚上），即使是同一天在同一个海滩上的海龟蛋。海龟蛋在白天和黑夜都可以孵化，但是在夜间孵化更快。
使用附有精准采集魔咒的工具可以获得海龟蛋。
海龟蛋在上方有实体站立时也会破碎，它们将逐个破碎，直到一个方块中的所有海龟蛋都被破碎。如果一个玩家或者足够大的生物、盔甲架落在或者停留在一个蛋上一段时间，蛋会破裂。掉落物也可以。大部分亡灵生物会主动寻找并踩踏海龟蛋，直到海龟蛋全部被打碎为止，除非把mobGriefing 设置为false。

介绍
深板岩（Deepslate）是一种在主世界深处代替石头生成的方块。
获取
深板岩不可再生。
自然生成
世界生成
在主世界中，石头在 y=1~7 高度处逐渐向深板岩过度；
在 y=-64~0，深板岩作为世界生成的基石完全代替了石头；
深板岩在 y=-64~-60 逐渐被基岩替换。
深板岩在 y=-59 丰度最高，约为 90.26%。
深板岩和石头过渡的景观
结构
在远古城市中心的一种地下室房间中会生成一个熔炉，其输出槽会生成 24 个深板岩。
合成
将深板岩圆石在熔炉中熔炼 10 秒可以获得 1 块深板岩和 0.1 点经验（x 0.1）。
创造模式物品栏
深板岩在建筑方块[1.19.2及以前]或自然方块[1.19.3及以后]标签页中。
用途
在工作台中合成各种深板岩材质的装饰性方块（如磨制深板岩、深板岩砖、深板岩瓦墙等）；
切石机有相同作用，且更省材料；
深板岩本身就是一种美观的建筑材料。
特性
基本特性
破坏
深板岩的硬度是 3，为石头（1.5）的两倍，需要使用木镐或更高等级的镐挖掘才能掉落。
若使用带有精准采集附魔的镐挖掘深板岩，将掉落深板岩本身，否则将掉落深板岩圆石。
深板岩的爆炸抗性为 6，同大多数石头相同。
蠹虫钻入
蠹虫会钻入周围的深板岩，并将其转化为被虫蚀的深板岩。
方块标签和额外特性
具有#minecraft:mineable/pickaxe标签，使用镐类工具挖掘深板岩更快；
具有#minecraft:azalea_root_replaceable和#minecraft:moss_replaceable标签，杜鹃树生长时会将其下方的深板岩替换为缠根泥土；
具有#minecraft:stone_ore_replaceables、#minectaft:deepslate_ore_replaceables和#minecraft:dripstone_replaceable_blocks标签，深板岩在世界生成时可以被各种深层矿石、泥土等方块替换；
具有#minecraft:base_stone_overworld标签，标记了深板岩是主世界的基石之一；
具有#minecraft:lush_ground_replaceable标签。
方块状态
和原木类似，深板岩方块具有一定朝向。
在世界中生成的深板岩默认为上下走向。
玩家放置的深板岩的朝向由玩家的朝向决定。
一般性质
深板岩是不透明方块，可以作为红石导体，可以窒息生物。
版本信息

在 Java 版 1.18 快照21w44a 版本中，深板岩和石头的过渡从 y=-7~-1 改为 y=1~7。

介绍
深板岩圆石（Cobbled Deepslate）是一种用途类似于圆石或黑石的石头变种。
获得方式
可通过用任意镐破坏深板岩后掉落。
用途
可代替圆石参加合成，也可以用于在铁砧中修补石制工具。
在Java版中，石质工具合成配方中的类圆石方块（即圆石、黑石、深板岩圆石）可以任意混合；但在基岩版中，石质工具只能使用类圆石方块当中的一种合成，不可混合。对于木质工具两种版本均无此限制。
版本信息
深板岩圆石在 Java 版 1.17 快照21w08a 版本和 基岩版 1.17.0 预览版1.16.230.52 版本中加入，此时在基岩版中属于实验性玩法。
在 1.17.0 预览版1.17.0.52 版本中，深板岩不再属于实验性玩法。
在 Java 版 1.17 快照21w10a 版本中修改了纹理。

深色橡树树苗（Dark Oak Sapling）是一个会长成树的物品。它可用任何工具破坏，并且总是会掉落它自身。
深色橡树树苗在Java版1.7.2快照13w43a中加入。
它也能通过破坏深色橡树树叶获得。当树叶被除了剪刀以外的工具破坏时，它们有5%的几率掉落1个树苗。时运附魔等级I会把树苗的掉落几率增加到6.25%，等级II为8.33%，等级III为10%。
种植它时必须将4棵深色橡树树苗以2x2的形式种在一起，且上方有至少7格空间时，才能长成一棵深色橡树，种植单个深色橡树树苗是不能长出树木的。
深色橡树树苗也可以被放在花盆里做装饰用。也可以用作熔炉的燃料，每个树苗可烧炼0.5个物品。
Minecraft 1.7.10版本有“可以利用深色橡树树苗突破基岩”的特性，在Minecraft 1.8及以后版本已经修复。
