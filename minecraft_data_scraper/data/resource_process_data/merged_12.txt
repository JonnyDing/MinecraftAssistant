简介
毒马铃薯植物（Poisonous Potato Plant），简称 P.P.P.（PoisonousPotatoPlant），是一个引导玩家游玩 24w14potato 的物品，直到击败超级马铃薯怪并获得马铃耶什，农民的传说之杖。
在物品栏中显示的是毒马铃薯的纹理，当玩家手持或鼠标指针移动到毒马铃薯植物上时，才会显示出佩戴格鲁乔眼镜的马铃薯纹理。
Poisonous Potato Plant 不仅指植物，在俚语中，Plant 也表示密探，也解释了一些关于毒马铃薯植物的对话和文本。
拿起放下其 4 次后，在提示框中会出现一条紫色斜体的信息：“On closer inspection this 毒马铃薯 is a plant”（仔细端详，毒马铃薯是一名植“务”），信息中的“毒马铃薯”的语言取决于游戏语言。
将多个毒马铃薯植物放置在箱子中，每次打开箱子时会随机产生被重命名的纸，内容如下：
Are you a plant?（你是植“务”吗？）；
Do you thinktheplayerknows?（你觉得那个玩家知道吗？），其中 the player 会被替换为曾经佩戴过它的玩家名称，若没有则依旧是 the player；
I think they know（我觉得他们知道）；
Let's stay quiet（嘘——）；
Destroy this message after reading（阅后即焚）。
这些纸在是一样的毒马铃薯植物产生的情况下才可以叠加，例如 A 和 B 产生了一张纸，B 和 C 产生了内容一样的纸，但这两张纸并不能堆叠。
获取
毒马铃薯植物可以从奖励箱中获取。
当蝙薯和毒马铃薯僵尸被玩家或驯服的狼杀死时，有 33.33% 的概率掉落。
挖掘马铃薯植株时有 2% 的概率掉落此物品，不受时运魔咒影响。
用途
毒马铃薯植物是一个引导物品，可以将其佩戴到头盔槽中（若此前没有佩戴过将要佩戴的毒马铃薯植物，则右击无法快捷的佩戴到头盔槽中），佩戴在头上时毒马铃薯植物会在屏幕左上角摇晃并弹出对话，每弹出一个字符都会播放一次音色为 flute 的音符盒的音效，事件进度储存在玩家数据 PotatoQuest 中。
玩家可以通过跳跃或完成特定的任务来引出下一段对话，部分事件完成时会给予玩家一些奖励。
摘下再戴上会重复上一句话。
无论马铃薯事件是否完成，玩家重生或在马铃薯维度掉入虚空都将重新开启事件。
在一定时间内若玩家没有再触发下一段对话，则对话会暂时消失，直到玩家触发下一段对话。

水在Minecraft里是一种最常见的流体。它分布在河流、湖泊、海洋、洞穴里，也可以通过打碎冰获得（前提是冰下面 1 格有方块），是有透明度的流体。
水在Java版Classic 0.0.12a中加入，在Classic0.0.19a和1.13快照18w15a中修改纹理。
水方块不能以物品的形式获得，但可以用桶装。右击装入，再次右击倒出。
玩家在生存模式和冒险模式下，如果无水下呼吸效果在水下过一段时间会淹死。水下呼吸效果可让你在水中看得清楚一点，夜视效果可以让你看得更明亮些。
在水里移动比在地上走慢，如果靴子有深海探索者附魔可以加快移动速度。
当水方块周围有足够的空间时，它会向周围流动。在平地上一格水在每一边上最多可以延伸 7 格，比普通方块稍矮，它会优先流向附近的比此方块 Y 坐标小 2 及更小的方块，而不向周围扩散。
当水方块（包括流动的水）周围以它为中心 9×9 的空间内出现耕地方块，它就会使耕地方块变成含水的耕地。

水桶（Water Bucket）是一个充满着水的桶。
水桶在Java版Infdev 20100615中加入。
获得方式
手持桶对水点按鼠标右键来得到水桶，水随之被消耗。流动的水无法填充桶。
在熔炉中将空桶置于燃料槽来烘烤湿海绵后也会得到水桶，湿海绵也将还原为海绵以待再次利用。
水桶有24.2%几率会在村庄渔夫的箱子里以1–3个为一组的形式出现。‌‌（仅限Java版）
使用
手持水桶对固体方块点右键，水将被置于其上，同时水桶还原为空桶。然而在下界中，被放置的水会立即蒸发。
发射器可被用来装载水桶。当其被激活时，水会在其正面被放置，同时其内的水桶被清空；此时再次激活，所放置的水将被回收进发射器内的空桶。
可挖一个2*2的方格，再沿着对角线放置两桶水可得到无限水。
炼药锅可被一个水桶填满，装满水的炼药锅也可以装满一个水桶。
对鱼使用水桶可以得到鱼桶，获得鱼桶的一瞬间会获得“战术性钓鱼”的进度。
注意：用水桶将水放置在末地传送门方块所对应的方块一侧，末地传送门方块会被破坏（该特性已在1.15被修复）。

沙子（Sand）是一种在海滩、沙漠和河流等自然生成的方块。
生成
沙子会在主世界的许多生物群系里生成，尤其是沙漠、海滩和河流，生成在被石头和沙石支持着的4格厚的层里。
沙子也会生成在部分埋藏的宝藏的箱子上，将沙子掩埋。
获得方式
沙子有59.0%的概率生成于沙漠神殿的箱子里，以1-8个为一组的形式出现。
流浪商人有概率以 1 个绿宝石的价格出售 8 个沙子。
挖掘时长
它的爆炸抗性非常低，仅为2.5。它是合成TNT和沙石、制作玻璃的重要方块。
它可使用任何工具摧毁，但用锹是较快的选择。其中，钻石锹和金锹是最快的，均为0.1秒/个。
用途
沙子在下方没有方块时会掉落，直到下方有方块为止。当沙子被重力效果影响掉落时，会显示平滑的掉落动画。如果沙子在掉落的过程中砸到生物或玩家，那么就会对生物或玩家造成窒息伤害，除非沙子被摧毁、走出方块或者死亡。如果掉落的沙子占据了非固体方块的空间，（例如火把、台阶、铁轨和红石），沙子就会掉落成为一个物品。沙子掉落到蜘蛛网的时候速度会变得很缓慢，直到沙子掉出蜘蛛网或已经着陆（变成一个资源方块）。沙子可以在不掉落的情况下放在非固体方块上。
仙人掌和甘蔗可以在沙子上面种植。
有时沙子会在生成后浮空。
在 Java 版 1.18-exp2 版本中，浮空的可能性被降低了，且浮空的沙子部分会被砂岩取代。
备注
在旧版Minecraft中，沙子与沙砾的行为相似。

沙砾（Gravel）是一种生成在主世界和下界的多种地形中、可以提供燧石的方块。
生成
主世界
沙砾会以团簇的形式生成于Y ≤ 320的地方，单个团簇包含了1～33块沙砾，每区块会进行14次生成尝试。
会生成于风袭沙砾丘陵/沙砾山地、石岸、河流或湖泊岸边的岸边或水底、除暖水和温水变种外的海洋。
沙砾还会作为古迹废墟、寒带海底废墟的一部分生成。
下界
沙砾会以团簇的形式生成于5-41层除玄武岩三角洲以外的生物群系中，单个团簇会包含1-160块沙砾，每区块会进行2次生成尝试。
沙砾还会以大片沙砾层的形式生成在熔岩海岸。
获得
沙砾还可以和猪灵以物易物获得。
掉落
掉到地上后会变成方块，如果碰到透明方块会掉落自身。
挖掘后概率掉落自身，较小概率掉落燧石。
沙砾生成后可能会浮空。
在 Java 版 1.18 快照exp2 版本中，浮空的可能性降低了，且部分浮空的沙砾会被石头取代。
用途
可用于在其上方种植竹子，也可用于合成混凝土粉末和砂土。
制箭师村民会以 10 块沙砾和 1 个绿宝石的价格出售6- 10 个燧石。
版本信息
沙砾在 Java 版 Classic0.0.14a 版本和 携带版 Alpha0.1.0 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为13。

河豚(Pufferfish)是一种可通过钓鱼获取的可食用药水材料。
获取
河豚可通过钓鱼获得，每次钓鱼有 11.5% 概率获得河豚。或直接通过击杀河豚掉落。
用途
食用
食用河豚后会恢复：1点（）饥饿值及0.2点（）饱和度，同时给予玩家 15s饥饿III、15s反胃II（在1.16.2中改为了反胃I）与 60s中毒IV 的效果。这会使玩家的生命值强制维持在非满血状态长达48秒（和平模式下为30秒）。这使得食用河豚变得十分地危险而且没有任何益处。
虽然河豚会让玩家中毒，但是使用河豚驯服豹猫并不会让豹猫中毒。
酿造
河豚可以与粗制的药水一起酿造出水肺药水，能使玩家在水下不减少氧气值。一个河豚足够制作出让玩家在水下逗留最多24分钟的药水数量。河豚是唯一能够制作水肺药水的酿造材料。
河豚桶在 Java 版 1.13 快照 18w08b 中加入。
河豚桶可通过对河豚使用水桶或空的桶‌‌[仅基岩版]获得，当使用空桶时，河豚必须在水源方块中。
流浪商人有 1/6 的概率以 5 个绿宝石的价格出售 1 个河豚桶。
手持河豚桶点击使用键，会放置一个水源方块，并令河豚重新生成于其中，玩家物品栏中的河豚桶会变成空桶。在下界使用河豚桶时放置的水源方块会立刻消失，只生成一条河豚。
由河豚桶放出的河豚不会自然消失，这一点和自然生成的河豚不同。
若河豚桶在发射器中，激活发射器会将一格水源方块和一条河豚生成在它所面向的位置。

挖掘用时
可以用任何工具开采，但用锹最快。
当使用没有精准采集附魔的工具开采草方块、灰化土以及菌丝时，它们会掉落泥土而不是它们自身。
用途
泥土可以用锄右击来变成耕地，但如果生物在上面践踏或者周围4格内没有水时，耕地也会变回泥土。
把树苗、甘蔗、蘑菇和竹子直接放在泥土或砂土上，在合适的条件下就会生长。在泥土使用锄就能把它变为耕地，使小麦种子、南瓜种子、西瓜种子、马铃薯、胡萝卜和甜菜种子能在上面生长。
当泥土方块与草方块相邻并暴露在亮度4或以上的环境时，它会在随机的时段后变成草方块。菌丝的传播方式也相似，但需要亮度9或以上的环境。
对泥土使用水瓶会将其变为泥巴。

介绍
泥坯（Packed Mud）是一种通过合成获得的方块。
获得方式
由泥巴和小麦合成得到，也可自然生成在古迹废墟中。
用途
可用于合成泥砖，也可作为装饰。
版本信息
泥坯在 Java 版 1.19 快照22w11a 版本和基岩版 1.19.0 预览版1.19.0.20 版本中加入。
在 Java 版 1.20 快照23w16a 版本和基岩版 1.20.10 预览版1.20.10.20 版本后，泥坯可以自然生成在古迹废墟中。
对泥土、砂土、缠根泥土右击使用水瓶转换得到。
会在红树林沼泽中自然生成。
方块高度与灵魂沙一致，为0.875格。
将泥巴放置于下方附着滴水石锥的滴水石上，泥巴会缓慢地变为黏土块。
版本信息：
泥巴在1.19快照22w11a中加入。
