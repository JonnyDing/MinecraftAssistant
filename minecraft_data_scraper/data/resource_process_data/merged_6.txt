地基方块（Foundation Block）是一种用于测距的方块。
当放置两个地基方块时，地基方块会测出与另一个地基方块之间的距离。
地基方块在 MinecraftEdu0.975 版本中加入。
地毯（Carpet）为用于装饰的方块，在Java版1.6.1快照13w16a中加入。它们的大小只有最薄的雪的一半。它们可以用于制造出地板为羊毛制作的景象。它们没有碰撞箱，这代表走过它们并不会提升16分之一方块的高度，这项特性和雪和睡莲相似，然而，地毯和其他固体方块一样能被活塞推动。

实体不可以穿过地毯。
将羊毛放在粘液块上可以弹得更高。

简介
基岩（Bedrock）是一种在生存模式中无法破坏的方块（但某些模组的道具可以破坏它）。
1.14快照18w50a前，基岩使用的是高对比度的石头的纹理，是 Minecraft 中硬度最大（在代码中它的硬度是 -1，在 2022 年的愚人节版本中硬度是 600）、爆炸抗性第 2 大的方块（爆炸抗性是 3600000，爆炸抗性最大的方块是屏障，爆炸抗性是 3600000.8）。它的主要作用是防止玩家或生物掉入虚空或进入下界世界的上方，也防止生物生成。它也是唯一信标和末地折跃门的光束能穿透的固体方块，这使得信标在下界可以使用。
它有可能生成在地图最底部的五层，但一般在最底部的第四层就全部由没有任何间隙存在的基岩组成，这使得最底部的第五层无法到达。下界中底部和顶部四层都存在基岩。在下界的顶部，基岩防止玩家到达高度 127 以上，它也能在主机版下界的边界找到。无论使用了什么种子，在主世界和下界的基岩总以相同模式生成。（仅 Java 版）（Java 版 1.18快照21w40a 及以后，主世界的基岩分布和种子有关，但是在下界其分布仍然是固定的。）
它不能被原版任何工具破坏，也不能被活塞推动或被正常游戏的爆炸破坏。原版中基岩只能通过创造模式物品栏或使用命令获得。
在末路之地的基岩上点燃火（包括床和重生锚的爆炸），火会像在地狱岩上面一样不会熄灭。
有趣的是，虽然在原版生存模式中没有正常的破坏方法，但是可以利用某些游戏中的 bug 来破坏它，以达到一些特殊的目的。尽管绝大多数的这些手段都只能破坏、而不能获得它。
可以利用末影珍珠和梯子穿过下界的顶层基岩。
尽管如此，该物品仍然可以在原版生存模式下用某些特殊手段获得，原理是利用粘性活塞的推动判定（该方法适用于 Java 版的 1.8 版本）。
在极其罕见的情况下，可能会发现包裹在基岩中的矿石。
基岩可以被自然生成的要塞和废弃传送门破坏。
恼鬼和末影龙可以穿过基岩。
指令可以破基岩，但是在基岩版中才会掉落。
在 1.13 扁平化之前，数字 ID 为 7。
冷知识
在 Java 版 1.14 纹理更新之前，基岩和石头的纹理是一样的，不过对比度更高，所以颜色区别更大；
原版游戏中只有 5 种方块存在于三个维度之中，而基岩是其中一种（其余为黑曜石，铁栏杆，箱子和空气）；
基岩版中有极小概率碰到废弃传送门生成在峡谷中，而恰好峡谷底部接近基岩层，这时下方基岩可能会被下界岩取代。

安山岩（Andesite）是一种岩石方块。
生成
安山岩会以团簇的形式生成在主世界Y＝0～60或Y＝64～128的范围，前者每区块生成2次，后者每区块尝试生成6次且有5/6的概率失败，团簇大小为0～64个方块。
获得方式
可使用圆石和闪长岩合成获得。
用途
可用于装饰，也可用于合成磨制安山岩。
版本信息
安山岩在 Java 版 1.8 快照14w02a 版本和 携带版 Alpha 0.9.0 预览版build 1 版本中加入。
在 Java 版 1.13 快照17w47a 版本扁平化前，数字 ID 为1。
在 Java 版 1.14 快照18w43a、18w44a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版 1.18 快照exp4 版本中，石岸地表有时候会生成条带状的安山岩地层，而在 Java 版 1.18 快照21w37a 版本中回退了这一改动。

解锁与激活宝库
宝库是一种自然生成于试炼密室的入口和决斗室结构中的装置。在宝库被激活时可以用试炼钥匙右击将其解锁并消耗一个试炼钥匙，接着宝库张开的嘴会合上，顶部的闸门开启并喷出战利品，投掷完毕后闸门会关闭。当玩家解锁宝库后，宝库会在方块实体数据中记录此玩家，只有未被该宝库记录的玩家才能解锁此宝库；但是，宝库最多能储存 128 名玩家的记录，当记录数大于 128 时，宝库会在添加新记录时删除最早的记录，使被删除了记录的玩家可以再次解锁这个宝库。
玩家只有在 4 格以内才会激活宝库；若宝库被激活后玩家退出到 4.5 格以外，宝库将熄灭。
当宝库进入激活状态时其纹理会变化，内部会出现火焰粒子和展示其战利品表的旋转掉落物，每秒随机更换一次物品。
战利品表
每次解锁时，宝库都会从战利品表中随机抽取物品 2-5 次（先在普通战利品表抽取 1 次，然后 80% 的概率从稀有战利品表抽取 1 次，20% 的概率从普通战利品表抽取 1 次，另有 25% 的概率从独有战利品表抽取 1 次）。
关于宝库的战利品表，详情请见Minecraft中文Wiki。
