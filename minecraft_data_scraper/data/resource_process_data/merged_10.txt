简介
活板门是一种透明的固体方块，可以被用作能够打开的栅栏。使用不同的木板可以获得对应种类的活板门。
木质活板门可用任何工具拆除，但斧是效率最高的。
木质活板门可能在雪屋生成，作为其地下室的入口。木质活板门也可能在沉船中上生成，作为船舱的入口。
使用
玩家可以手动开关木质活板门。
要放置一个活板门，对选定的位置使用一个活板门物品。一旦活板门安置好了，它附着的方块可以被移除。活板门被放置后会占用一个方块的顶部或底部，具体情况取决于玩家放置活板门的方式。若活板门被放置在方块顶部，它会向下开启。若被放置在方块底部，它会向上开启。活板门的“合叶”会连接在它附着的方块上。
活板门可以被活塞移动。水和熔岩流动时会绕过活板门。熔岩能在木质活板门旁的空气方块燃起火焰，使木质活板门看起来像是在燃烧，然而实际上活板门不会烧毁（也不能以任何其他形式烧毁）。
若一个开启的活板门在梯子上方且与梯子附着在同一方向，这个活板门可以像梯子一样被攀爬。
与大多数生物的声音类似，开启和关闭活板门的声音最远可以在16格外听到。
活板门能够在玩家或生物在活板门里面的情况下被开启或关闭。
屏障与陷阱
活板门是一个可开关的屏障，能限制实体运动。虽然其主要用于限制玩家和生物移动，活板门也可以用于控制船（例如，在一个两格宽的水渠中放置一个占据方块顶部的活板门，这个活板门关闭时会阻挡船的运动，开启时则会使船通过）、物品和矿车（关闭的活板门能阻止物品或矿车掉落，开启之后则允许它们掉落）等。活板门的厚度为 0.1875 (3/16) 米。在其余的空间，活板门不会阻挡实体运动，但不能在水下为玩家和生物提供空气。要开启或关闭木质活板门，对活板门点击右键。活板门开启或关闭时会立即改变朝向，不影响它“穿过”的空间中的实体。移动的活板门也不会像活塞那样推动实体。
生物的寻路系统会将活板门视为关闭的，不管它是否真的关闭。这经常会导致生物从开启的活板门井盖掉下去。
村民会将活板门视为可穿过的方块，但它们并不能打开活板门；同时又和其他生物一样，会将活板门视为关闭的，不管它是不是真的关闭了。
红石机械
木质活板门能被红石信号控制。活板门是一个红石机械，它能被下列红石元件激活：
一个相邻的电源元件，包括上面和下面的，例如：一个红石火把，一个红石块，一个阳光传感器，等等。
一个相邻的充能的方块，包括上面和下面的。
一个指向活板门的充能的红石比较器或红石中继器。
指向活板门的充能的红石粉，或活板门旁充能的无指向的红石“点”；活板门不受指向其他方向的红石粉控制。
活板门被激活时，它会自动绕着它的“合叶”“旋转”到其开启状态。当活板门没有被激活时，活板门会立即回到其关闭状态。每次活板门状态的更改需要1游戏刻的时间。
被红石信号激活的木质活板门仍然能被玩家手动关闭，并且在红石信号更新前不会再被激活（也就是说需要将红石电路关闭再开启才能重新控制活板门）。
燃料
木质活板门可以被用作熔炉的燃料，每个活板门烧炼1.5个物品。

末地传送门是一种用于将实体传送至末地或从末地传送至主世界的方块。它是激活的末地传送门结构和激活的返回传送门的一部分。
在Java版中，末地传送门没有物品形式；在基岩版中虽然有物品形式，但是只能通过编辑物品栏来获得。末地传送门无法在生存模式中被破坏，也不会被TNT等爆炸物炸毁。

末地烛可以通过合成获取。

末地石（End Stone）是一种构成末地维度的方块。
挖掘用时
末地石的硬度为3，爆炸抗性为45。必须使用镐来开采，否则不会掉落任何东西。其中金镐效率最高，为0.4秒/个。它既不能被末影人捡起来，也不能被末影龙摧毁。
Java 版1.9 版本后，末路之地里会生成附加岛屿，可以制造出接近无限（但不可再生）数量的末地石。
在 Java 版中，末影龙重新生成时，会替换在返回传送门的角落下面的16个末地石，末影龙死亡后会再次变回。由于末影龙可再生，因此末地石也变成了可再生资源。
用途
可用于合成末地石装饰方块，也可用于复制盔甲纹饰锻造模板。
版本信息

介绍
末影之眼（Eye of Ender/Ender Eye）是一种用于定位并激活要塞里面的末地传送门的可再生道具。它可通过合成获得。
Java 版 1.3.1 快照12w22a 版本至 1.9 快照15w41a 版本中，可从牧师村民那里购买末影之眼。
用途
末影之眼可以用于定位并激活末地传送门。
12个末影之眼可以激活末地传送门，每个末地传送门框架在生成时有10%的概率会被放置末影之眼。对着末地传送门框架按下使用键（默认为鼠标右键）就可以放置末影之眼，一旦12个全部放满后，末地传送门就会激活。
合成末影箱和（1.9及之后）末影水晶。
定位要塞的过程（以及传送门所在的房间）
手持末影之眼并按下使用键（默认为鼠标右键），末影之眼会飞向最近的要塞，穿过必要的方块，并在其经过的地方留下一条带有紫色颗粒的痕迹。
（仅限PC版）末影之眼会飞向世界里生成的第一个要塞。如果退出并重新加载，那么它就会飞向生成的第一个螺旋楼梯。
（仅限主机版）末影之眼只会飞向要塞，而不是在要塞里面的末地传送门。
如果距离末地传送门很远，末影之眼就会飞得更高，以便在垂直方向上更明确指示玩家必须通过的路径。
当玩家接近末地传送门所在的纵向坐标轴时，末影之眼会更精准的飞行来与定位这条 Y轴。这也就意味着末影之眼将会向下飞行，指示出玩家在末地传送门的上方，让玩家向下挖。
在空中飘浮2-3秒后，末影之眼就会掉落，变成可拾获的物品，或者在空中碎掉。有1/6的几率会碎掉。
末影之眼的飞行功能只在主世界工作，在下界、末路之地、超平坦模式的主世界以及没有末地传送门的世界中不会工作。

末地水晶（End Crystal）是可以在末路之地中找到的一个实体。它的ID为end_crystal。它可以在末路之地黑曜石柱的最顶端找到，在1.9及之后也可以通过使用玻璃、恶魂之泪和末影之眼合成得到。
当它自然生成于末地黑曜石柱基岩上方时，末地水晶会附带一个基岩底座，可以通过命令修改其数据来使手动放置的水晶显示底座。
它的主要用途是恢复末影龙的生命值，末地水晶会治疗所有在32方块距离之内的末影龙并且显示一道指向末影龙的光束，末影龙会在每0.5秒恢复1点生命。一个末地水晶可以同时恢复多条末影龙的生命值。末地水晶的治愈光束不会被任何实体或方块所阻挡或减弱。
当它受到攻击时，甚至这次攻击只造成0伤害值时，末地水晶都会爆炸。末地水晶的爆炸与高压苦力怕的威力相同，比TNT的威力还要大50%。任何正在接受该末地水晶治疗的末影龙都会受到10点（）伤害。
末地水晶的基座有着基岩的纹理，上方有一颗悬浮的水晶。在末地中，基座嵌于一块基岩的底部，而水晶在基座的之上一个方块距离处悬浮，且在该位置持续产生火焰；水晶占据位置的任何其它方块都都会被移除。末地水晶不是固体，可以自由穿过。末影龙亦会毫发无损地穿过水晶。末地水晶可以被活塞推动，但如果被推动的末地水晶带有火焰，则会被点着并爆炸。
物品形式的末地水晶只能在黑曜石和基岩上放置，而且没有底座。这可以用于在末地传送门处重生末影龙，也可以用于在主世界实施炸矿（成本高昂，但效率很高）。
重生末影龙时也会重生在黑曜石柱上的末地水晶。但复活过程中只要末影龙还未生成，其它的水晶将无法被破坏。（除了自己放在末地池上的四个水晶，但如果破坏了，复活会中止）

备注：在1.9快照15w44b及之后的版本，末地水晶被添加了合成表。而在此之前，只能通过代码才能获得末地水晶。

末影珍珠（Ender Pearl）是一种可以用来投掷以传送使用者的物品。
当末影珍珠被使用时，会有5%的概率生成末影螨。
获得方式
末影人被杀死时会掉落0-1颗末影珍珠，每1级抢夺魔咒都会增加1个最大掉落量，最高为抢夺III的0-4个；
末影珍珠有23.05%（Java版）或22.39%（基岩版）的几率生成于要塞走廊的箱子里，以1个为一组的形式出现，有19.61%的概率生成在试炼密室柱廊的箱子里，以1～2个为一组的形式出现。林地府邸的伪末地传送门房间内总是会生成1个陷阱箱，当中总有2个末影珍珠。
专家级别的牧师村民有2/3的概率（Java版）或总是（基岩版）以5个绿宝石的价格出售一颗末影珍珠。
猪灵获得金锭后会有10/459的概率给予玩家2–4个末影珍珠。
行为
使用末影珍珠可以将其投掷出去，生成弹射物，并引发20游戏刻（1秒）的末影珍珠物品冷却时间，期间受影响玩家不可使用末影珍珠。
掷出的末影珍珠击中物体并破裂后，会将投掷者传送到其破裂位置并在这一位置生成许多聚拢的紫色粒子，同时对被传送的玩家造成5点（）摔落伤害，穿戴具有保护或摔落保护魔咒的盔甲能减少掷出的末影珍珠造成的伤害。
当末影珍珠被垂直往上扔，能竖直向上穿越约30格。当末影珍珠被以40°角扔出，能水平向前穿越约59格。
以足够的保护和末影珍珠，玩家能快速地穿越很远的距离。末影珍珠也能在创造模式被扔出。
如果掷出的末影珍珠被丢入末地传送门或末地折跃门，那么这些方块传送实体的功能会直接对投掷者激活，并移除掷出的末影珍珠，而不会产生摔落伤害或其他效果。
末影珍珠会与藤蔓、草丛、枯死的灌木、蕨、花、所有种类的树苗、所有种类的蘑菇、所有种类的矿车、船和下界传送门碰撞。
Java 版 1.9 快照15w34c 版本及之后，末影珍珠在再次使用之前需要等待一秒的冷却时间。冷却时间可以在物品栏里的末影珍珠一层白色的覆盖物那里看见。
1.13版本及之后，末影珍珠可以在投掷后被TNT爆炸推动。
注意
掉进虚空里的末影珍珠并不会传送玩家。使用末影珍珠瞬移可能会被服务器以“作弊”为由踢出。这是由于触发了防止修改客户端而使得移动过快的检查机制。

介绍
末影箱不仅可以像普通箱子一样存储物品，而且与其他的末影箱共用一个储存栏，它还可以跨维度、无视距离地存储物品。1.13版本之后，末影箱表现出含水的能力，即把末影箱置于水中，水会与末影箱处在同一方格中。
末影箱在被破坏之后并不会掉落内部存储的物品。哪怕所有的末影箱都被破坏，只要重新放置一个，就可以恢复功能与内部存储物品。
实际上，末影箱不是容器，而是一个访问接口，用于访问玩家的属于末影箱的额外物品栏，即数据储存于玩家，因此有以上特性。
末影箱的爆炸抗性是黑曜石的一半（600），但是仍然能够抵挡绝大部分的爆炸伤害。它可以发出7的亮光，同时周围会有类似末影人的紫色粒子。你知道吗：如果末影箱被炸毁（使用一般生存无法获得的高威力爆炸）且有掉落物，那么会掉落一个黑曜石。
注意
末影箱如果不使用带有精准采集附魔的镐来采集，将会掉落8块黑曜石，而不是它本身，相当于损失一个末影之眼。
末影箱可以并排摆放，但不像箱子，末影箱并不会进行合并。
末影箱当中的物品不能使用漏斗进行放入或取出。
末影箱的物品栏对每位玩家都是单独存在的，玩家之间不能进行共享。

材料分解器能将Minecraft中的方块分解成构成它们的元素。
材料分解器中有10个物品栏槽位。顶部的中心槽是输入槽，放入方块后，在下方的9个输出槽就会出现构成该方块的元素方块。
某些方块，如灵魂沙和下界岩，并不存在于现实世界，因此这些方块的分解结果会包含一个带有问号的未知元素。
一旦有1个元素被从输出槽中取出，输入槽就会清空。
如果输出槽有其他元素，则输入槽会变成红色且不接受任何方块的输入，直到输出槽内被清空为止。
