简介
下界疣（Nether Wart）是一种生长在下界的植物，是酿药必须品。
在 1.16 版本中，地狱疣译名改为下界疣。
获取
下界疣可以在下界要塞中的种植房间或部分堡垒遗迹居住区中生成的灵魂沙上找到，此外，在下界要塞的战利品箱中还有 19.04% 的概率获得 3-7 个下界疣。
除此之外，还可以通过交易获得下界疣，大师级的牧师村民用 1 个绿宝石购买 22 个下界疣。
用途
下界疣所酿造的粗制的药水是除虚弱药水外所有药水酿造的基础，下界疣还可以合成下界疣块和红色下界砖块。
放进堆肥桶时 65% 的概率使堆肥层数加 1。
在 Java 版 1.4.2 快照12w36a 版本到 1.8 预发布版1.8-pre1 版本期间，还可以用来繁殖和喂养鸡。
种植
下界疣只能种在灵魂沙上，不需要水，不需要用锄耕地，也不会被踩坏，不能用骨粉催化，不受光照因素的影响。
下界疣有三个生长阶段，且生长速度很快，平均约每 40960gt（四舍五入为 34 分钟）就可以完全长成。
版本信息
下界疣在 Java 版 1.0.0 预发布版Beta 1.9-pre1 版本和携带版 Alpha0.12.1 build 1 版本中加入。
介绍
下界疣块（Nether Wart Block）是一种由下界疣合成的装饰性方块。
获得方式
由9个下界疣合成。
下界疣块也会在绯红森林中作为巨型菌类的菌盖出现。
它可以被徒手挖掘，且使用任何装备、道具挖掘的速度都是一样的，均为1.5秒/个。
用途
可用于装饰，也可用于堆肥。
注意
下界疣合成为下界疣块的操作是不可逆的，即下界疣块不能合成为 9 个下界疣。所以说，它不能储存下界疣方块。
版本信息

在 Java 版 1.13 版本17w47a 版本扁平化前，数字 ID 为214。
在 Java 版 1.14 快照18w43a、19w08a 版本和 基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理。
在 Java 版1.16 版本中，译名由地狱疣块改为下界疣块。
在 Java 版 1.16 快照20w09a 版本和 基岩版 1.16.0 预览版1.16.0.57 版本中，使用锄类工具可以快速采集下界疣块，采集效率受效率附魔的影响。

下界砖块（Nether Bricks）是一种构成下界要塞的方块。
获得方式
下界要塞由下界砖块组成。
也可由下界砖合成。
用途
下界砖块是不可燃尽方块，且对恶魂火球免疫，这使得它成为合适的下界庇护所材料。当走在地狱砖块上面时有特殊的脚步声。
如果一个下界砖块是在一个下界要塞中的一个完全封闭的空间里（不仅仅在单独的部分里），会在下界要塞生成的生物（比如烈焰人和凋灵骷髅）可以在其他生成条件都符合时生成在上面。

一本书和一块插了羽毛的墨囊的结合，放在手上右键可以在上边写字（打出的字会自动保存）。通过这个界面，玩家最多可以创作50页的内容，每页最多256个字符，每次换行记2个字符。没有任何一页长于14行，每行平均可输入19个字符，具体由所使用字符的宽度决定。玩家能够将内容复制与粘贴到书中。然而，单次粘贴的内容超过一页所能容纳的字数的话，粘贴会失败。目前大部分Unicode字符都能被成书所支持，并且在粘贴时不会出太大问题。
在单人游戏中，编辑时游戏会暂停内置服务端，即如同按下ESC唤出菜单后的暂停世界，如果已经向局域网开放，那么则不会暂停内置服务端。
玩家在书与笔内写进内容后保存的动作，即是玩家编辑书与笔NBT的一项操作，而在早期被称为“神书”的游戏特性则正是利用了这一项内容/特性，使得玩家可以在生存模式下创建一个拥有恐怖附魔与属性的杀戮武器，在没有附魔限制的早期版本，玩家甚至可以借助部分破解过后的NBT编辑器使得一本书与笔同时拥有几十上百个“锋利32767”附魔和同样多的属性修饰符。
在早期版本没有加入移动光标功能，也不能选择文本。
签名后会变成一本成书，书与笔的NBT将会被保留，例如属性修饰符或者附魔等。成书的名字可以在签名时自定义。成书内会显示你在书与笔中写的东西。书与笔一旦被署名，书会带上写作此书的玩家名以及在署名时所输入的书名。在书与笔被署名后，它就不能继续被编辑。
书的标题最长可达16字符，不支持复制粘贴。如果键盘支持的话，可以在标题中添加特殊字符。Alt代码无法输入，而且即使你的键盘和输入法有特殊设定可以输入§字符也不能输入到标题上，这意味着你不能通过正常方式让书的标题设定样式。
小知识：
用 Alt + 小键盘 167 可输入“§”符号，样式代码前缀，在书与笔中输入此符号后输入特定的字母即可改变字体颜色和样式，但是此方法对部分设备无效，可以尝试直接复制百科中的小节符号，即“§”。
虽然默认书名最多只能到16个字符，但你可以通过铁砧给它起个更长的名字。

传送方块（Teleport Block）是一个可以传送玩家的方块。
教师可以设置传送位置，当玩家站在传送方块上时，会被传送至指定地点。
传送方块在 MinecraftEdu0.98 版本中加入。
传送门底座（Pedestal）是一个生成马铃薯传送门的方块。
传送门底座会自然生成在自然生成于平原村庄、要塞的传送门房间中心、废弃马铃薯传送门和角斗场。
传送门底座的硬度为 55，且没有工具可以加速挖掘，挖掘不掉落任何物品。
侦测器能侦测目标方块ID或数据值的变化，还有方块的放置和破坏。侦测器能探测到基础方块状态的更改，但不能探测到延伸/实际方块状态的更改。这意味着像农作物年龄的更改会被探测到，因为它们是基础方块状态的一部分，它在世界被保存时会转换为元数据。然而，栅栏形状之类的更改不会被探测到，因为它们是延伸方法状态，不会和世界一起保存。

信息方块（Information block）是一种可由学生和教师玩家写大量文本的方块。
信息方块有特有的 GUI，类似于书与笔，可以从任何一行开始输入，每行仅限输入 35 个字符，写满后，当前单词被移到下一行。
信息方块的 GUI所有的玩家都可以查看信息方块中的内容，但想要修改其中的内容，需要满足以下任意一个条件：
玩家是一名教师；
尝试编辑其的玩家是放置这个方块的玩家；
Everyone can edit 为 true。
信息方块还可以传送玩家至指定位置，并显示信息方块中的文本。
对于教师，信息方块还有一个编辑功能，可以编辑信息方块的一些功能。编辑 GUI
信息方块在 MinecraftEdu0.9652 版本中加入，编辑功能在1.5.1 Build 10 版本中加入。
信息标志（Information Sign）是一种装饰方块，无特殊用途。
信息标志在 MinecraftEdu0.9652 版本加入。
信标是一种功能性方块。自身会为周围提供相当于荧石的亮度。
方块状态下的信标其自身会携有{Primary:-1,Secondary:-1,Levels:-1,Lock:""}的NBT，其中Primary与Secondary分别代表该信标当前的主要和次要效果，-1代表未选中任何效果；而Levels代表当前层数， -1为无层数。Lock:""则为容器的通用NBT，括号内的一切字符为容器的密码，在没有手持对应名称的物品时无法打开容器。
信标在Java版1.4.2快照12w32a中加入。
将它放在由铁块、金块、钻石块、绿宝石块或下界合金块等（有的模组方块也能使用）构成的类似金字塔结构的基座上，可以发出一束明亮的光束；若在信标上方有除基岩外的不透明方块，信标将不会发射光束。组成基座的方块可混用，基座也可混用（即相邻信标只需多圈一层）。光束默认为白色，穿过染色玻璃（板）后，颜色会发生改变；叠加染色玻璃（板）亦可叠加颜色：
最高级金字塔基座与玻璃（板）与相邻信标
右击信标可以打开GUI，放入一个矿物（铁锭、金锭、钻石、绿宝石、下界合金锭）可以设定信标效果。设定后信标会为周围的大片区域的玩家提供增益效果，从左到右从上到下依次为：速度、急迫、抗性提升、跳跃提升、力量、生命恢复。
信标基座的等级会限制可选择的效果类型，详见信标金字塔。

僵尸的头是生物头颅中的一种可再生物品。它是一种可穿戴物品，也是一种固体透明方块。它也可以用来合成对应的烟火之星。
僵尸的头在 1.4.2 快照 12w36a 中加入；1.8快照14w30a中修改纹理。
僵尸的头的爆炸抗性为 5，硬度为 1，可用任何工具或徒手花费 1.5 秒的时间开采它。
1.9快照15w32a后玩家戴上该头颅将会减小默认僵尸发现你的范围的 50%。这类似于潜行时或拥有隐身效果时会减少检测范围的特性。
当一只闪电苦力怕（被闪电击中的苦力怕）炸死一只僵尸，此物品就会掉落。
从1.19.3快照22w46a（内置数据包）开始，若音符盒顶面放置了僵尸的头，当音符盒被玩家敲击或被红石信号激活时，会播放僵尸的叫声。
