Etho 台阶（Etho Slab）是使用TNT合成的 TNT 台阶，ID 为 160。
拥有与其他台阶一样的特性，并且可以被点燃，爆炸威力与 TNT 相同。
当玩家位于爆炸范围内时，会在玩家头顶生成一个铁砧。
若处于附近爆炸的爆炸半径内，右击使用打火石，接触火，火焰弹，岩浆（被岩浆蔓延的火点燃），火矢弓(附魔)或者红石信号，TNT就会消失，而在方块的中心（+0.5 +0.5 +0.5）会生成一个实体TNT，将遵循物理定律（如重力），开始冒烟，闪烁白光，并在4秒（80游戏刻）或者8次闪烁后爆炸。几乎所有在爆炸半径内的方块（见下方列表），生物，和玩家都会受到伤害或被完全摧毁/杀死。但是，爆炸对方块的破坏（对玩家和生物则不是如此）可通过使TNT在液体中爆炸来避免。在爆炸半径内的掉落物品会被毁坏，但是如果这个物品和爆炸中心之间有方块阻挡（无论什么材料的方块），就可以保护物品免于被毁坏。
TNT的爆炸会根据公式f=e-r来传播, f是爆炸穿过某方块后的能量, e是爆炸穿过该方块前的能量, r 是该方块的爆炸抗性。因此，TNT的爆炸在爆炸抗性为0的空气中传播最远（7个方块）。 但是，被密封在石头中的TNT会破坏掉以爆炸中心为中心的3x3x3的方块。 如果你靠得太近，TNT爆炸就会杀死你。
一些方块和实体可以在TNT爆炸中幸存，甚至可以保护玩家。 就方块来说，下界合金块、黑曜石、基岩和水对TNT爆炸免疫, 它们分别能用来做掩体，密封世界和TNT大炮。另外, 一个坐在矿车中的玩家或生物在TNT爆炸时只会受到很小的伤害。这个效果可以把它们弹出很远的距离。
很不幸地，使用TNT时要注意很多问题。 TNT着火以后不会烧掉，而是会被激活(但TNT依然处于燃烧状态，并会将火传给其它方块比如另一个TNT), 因此，在TNT周围布好防火措施是十分必要的。 如果在下界或很深的地下使用TNT, 岩浆会引燃TNT并激活它。 再者, 由于TNT爆炸需要许多物理计算，规模较大的爆炸会在单人模式下引起卡机，在多人模式下引起较大的延迟。一个TNT爆炸会引燃爆炸范围内的其他TNT并四处飞溅，飞溅范围可能很大，因此要做好充分防护。
另外, TNT还有许多有趣的特性。 理论上说, 无限数目的激活的TNT可以全部塞进一个方块中。沙子和沙砾（实体）会因为爆炸而被炸出原来的位置。 如果沙子或沙砾掉进了一个在水下的，已激活的TNT中，这个TNT将会正常破坏周围的方块。
而且，在点燃TNT的时候要注意，TNT被点燃后会有轻微的移动，导致会掉落到其他方块上。
在教育版中，TNT可以和钠合成水下TNT。
实体TNT可以被附有击退的剑打飞，但这个特性在1.17-pre2被移除。

TNT矿车是一种装载着TNT的矿车，由1个TNT和1个矿车合成。
在静止时被摧毁会掉落1个矿车和1个TNT。
爆炸条件
经过已充能的激活铁轨（像TNT方块那样延迟4秒引爆）。
掉落或脱轨运行超过3个方块。
在运行时被摧毁。
被火、熔岩或爆炸摧毁。
在一个方块里放置6或以上个tnt矿车，被其他实体推动，tnt矿车就会合体并诡异地往一个方向缓慢地平移。撞到方块会产生剧烈的爆炸，摧毁大量方块，并对实体造成巨大的伤害和击退。
以过快的速度过弯、撞上轨道旁边的固体方块或实体。（在之前的移动方向上）
被火矢箭射中。
如果在一个转圈的轨道上，并且放置了过多TNT矿车，将会在过度拥挤的情况下爆炸，一般在这种情况下爆炸的TNT矿车会将玩家推出很远。
爆炸效果
TNT矿车会像普通TNT那样摧毁附近的方块，对周围的实体以及玩家造成伤害，而且爆炸威力与速度成正比。
TNT矿车的爆炸效果有多种情况：当TNT矿车被已充能的激活铁轨激活时，它会延迟4秒爆炸，这种情况下TNT矿车的爆炸会忽略任意铁轨及其支撑方块，但是任意铁轨及其支撑方块并不会阻挡并吸收爆炸。
当TNT矿车被其它形式引爆时，它会正常爆炸，不会忽略任意铁轨及其支撑方块。
若TNT矿车爆炸时其下方有任意铁轨，那么爆炸只会对生物造成伤害，没有冲击效果。（1.20.4已失效）
JAVA版中两个TNT矿车相撞会产生剧烈爆炸，而在基岩版中则只会弹开。

一叉戟（Dent）是一种用于合成三叉戟的材料。
在瘟疫鲸台阶死亡时会掉落 0-1 个一叉戟。
丁香在Java版1.7.2快照13w36a中加入。
发现于森林和繁花森林群系，用于合成品红色染料。
概述
三叉戟是一种用于近战或远程攻击的投掷武器。
三叉戟的近程攻击伤害为9点（），远程攻击伤害为8点（）。在激流冲刺的过程中若撞击到生物则使用主手武器进行一次近战攻击，其伤害由主手持有的武器决定。
三叉戟有 1.1 的攻击速度，这会有约 0.91 秒的冷却时间（仅 Java 版）。
获取方式
自然生成的持有三叉戟的溺尸（Java 版持有概率为 6.25%，基岩版为 15%）有概率（Java 版为 8.5%，基岩版为 25%）掉落随机耐久度的三叉戟。每一级抢夺附魔可以增加 1% 的概率（即抢夺III为 11.5%）这是三叉戟唯一的可再生来源。
三叉戟有 8.33% 的概率在试炼密室的宝库或奖品箱 - 独有中开出。
在 Java 版中，溺尸持有的三叉戟可能是附魔的，但溺尸投掷的三叉戟是没有魔咒的，且不可被回收进物品栏。
三叉戟在击中方块后会插在方块上，摇晃 7 游戏刻（0.35s）后插入方块。三叉戟会在插入方块后 60s 后消失（仅 Java 版）。三叉戟只有在击中方块且停止摇晃后才可以被玩家回收，飞行中的三叉戟接触实体后会命中或被弹开，而不能被回收。玩家投掷出的三叉戟正常情况下只能被投掷的玩家回收。
使用方式
手持三叉戟时对生物或玩家按攻击键可近战攻击对方，击中后，三叉戟的耐久度减 1。类似剑，玩家在创造模式中持有三叉戟时无法破坏方块。
按住使用键可以为三叉戟蓄力。若三叉戟具有激流魔咒，则触发激流魔咒的效果（冲刺），否则会将三叉戟掷出。待三叉戟掷出后，若击中实体，则对其造成伤害。其弹道轨迹类似于箭，但强度只有 80%。三叉戟不受水的阻力影响，所以是优秀的水下远程武器。
若三叉戟命中方块，它会粘滞在方块上，若命中实体，则会弹开到附近的地面。它可以被盾牌防御。掷出的三叉戟可以像箭那样触发木质按钮。甚至可以把三叉戟投掷到紫颂花和滴水石锥上来破坏它们，使它们以物品形式掉落。三叉戟对实体造成弹射物伤害，可以被弹射物保护魔咒防护。在基岩版中，可以用发射器来掷出三叉戟。
每造成一次近战伤害或每投掷一次都会减少 1 点耐久度。只有 1 点耐久度的三叉戟无法被投掷。用三叉戟每次成功挖掘任何硬度不为 0 的方块会消耗 2 点耐久度。
持有三叉戟的溺尸会无限发射三叉戟，僵尸转化而来的不会。被溺尸的三叉戟击败的村民也有概率转变成僵尸村民（简单难度下为 0%，普通难度下为 50%，困难难度下为 100%）。像骷髅和流浪者的箭一样，溺尸掷出的三叉戟无法被捡起。在基岩版中，溺尸可以使用三叉戟近战攻击距离其 3 格内的目标，每次对目标造成11点（）伤害。
附魔
附有忠诚魔咒的三叉戟会很快自己回到投掷者手中，速度取决于忠诚附魔的等级。三叉戟被投出并正在返回投掷者身边时，三叉戟会被投掷该三叉戟的生物吸引，且这个过程中三叉戟会穿透方块，投掷者接触到该三叉戟时即可将其回收。如果投掷者失踪，那么三叉戟会以物品形式掉落在地上。在 Java 版中，被掷入虚空的三叉戟最终会抵达临界高度并被摧毁，即便有忠诚魔咒也无法返回（Combat Test 中忠诚附魔可以避免这一点）。在基岩版中，具有忠诚魔咒的三叉戟会在虚空中移动约 15 格后返回。
具有激流魔咒的三叉戟可以用于推进鞘翅，但只能是玩家在水中或者在雨天中。但是实际上寒冷和积雪类生物群系下雪时它也可以推进鞘翅。在 Java 版中，每次投掷造成8点（）伤害，且主手物品中只有与战利品表相关的魔咒会生效，且不消耗主手物品耐久。在基岩版中，每次投掷与玩家主动进行的近战攻击无异，主手物品决定伤害，会消耗耐久度，且魔咒会生效。玩家有机会通过该自动的近战攻击造成暴击。三叉戟投出后还会对玩家旁边的生物造成“溅射伤害”。
具有穿刺魔咒的三叉戟会对水生生物（仅 Java 版）或水中与雨中的生物（仅基岩版）造成更多的近战及投掷伤害。
具有引雷魔咒的三叉戟被玩家投掷出去在雷暴天气下击中实体（仅 Java 版）或生物（仅基岩版）后，会在被击中的目标所在处召唤一道闪电。
忠诚和引雷分别与激流互斥，但忠诚和引雷不互斥。

简介
下界传送门（Nether Portal）是构成下界传送门的半透明方块，可以让玩家在下界和主世界维度之间传送。
生成
下界传送门（方块）会作为下界传送门（多方块结构）的一部分生成。
获取
在 Java 版 1.8 快照14w25a 版本前，下界传送门可以通过指令获取，此后的下界传送门没有对应的物品形式。
在基岩版中，仅能通过编辑物品栏获得。
用途
下界传送门用于进出下界和主世界。
当玩家进入下界传送门时，屏幕会出现类似反胃Debuff 的屏幕扭曲，并出现下界传送门的纹理，持续处在其中 4 秒钟后，玩家就会被传送到另一个维度，在传送途中，玩家可以离开传送门以取消传送，在创造模式中，玩家在接触下界传送门方块时会被瞬间传送到其他维度。在进入维度中时，玩家会默认生成在下界传送门中，此时玩家并不会被传送到另一个维度，而是持续播放传送动画，直到玩家离开传送门。在传送门中玩家无法进行打开物品栏、打开游戏菜单、与其他方块互动等。
下界传送门会发出呼啸的音效（block.portal.ambient），并产出紫色的粒子，这个粒子同样会出现在末影人、末影螨、末影珍珠和末影箱中。
下界传送门的爆炸抗性只有 1，因此在受到爆炸的时候会直接破坏掉下界传送门且不掉落任何物品，此时在另一个维度中与被破坏的下界传送门相通的下界传送门方块也会被破坏。
在下界传送门（多方块结构）的黑曜石上放置任何液体都会破坏下界传送门，挖掘黑曜石也会破坏下界传送门。
下界传送门无法被活塞推动或拉动。
下界传送门也可以传送其他部分实体，并且是瞬间传送。
手动放置的下界传送门（方块）在接触其他方块时会消失。
下界传送门会发出亮度等级为 11 的光。
版本信息

下界反应核是组成下界反应器结构的主要方块，用于生成一个小型下界。
挖掘之后会获得6个铁锭和3颗钻石（0.12之后）或一个下界反应核（0.12之前），除使用NBT编辑器或在原先就存在下界反应核的存档使用带有精准采集的镐挖掘，否则无法获得此方块。
在0.12之前，挖掘使用过的下界反应核获得全新的下界反应核，可以对一个下界反应核进行反复利用。
此方块在基岩版1.0时被移除。

由于下界合金套还提供了12点盔甲韧性，所以在面对高伤害时下界合金套会比钻石套有着更出色的防御效果。

简介
下界岩（Netherrack）是一种在下界处处可见的方块。
在 Java 版 1.10 快照16w20a 版本到 1.16 快照20w07a 版本、携带版 Alpha0.15.0 build 1 版本到基岩版 1.16.0 预览版1.16.0.51 版本期间，末影人可以搬起下界岩。
生成
下界岩会自然生成在下界中的各个生物群系中。
在 Java 版 1.16 快照20w16a 版本和基岩版 1.16.0 预览版1.16.0.57 版本及以后，它也会出现在废弃传送门和堡垒遗迹中。
就像草方块和泥土，当绯红菌岩或诡异菌岩上方放有不透明的固体方块时，就会退化成下界岩，但菌岩不会蔓延。
在基岩版中，生成下界传送门时，如果下面的方块是空气，就会将空气替换成下界岩生成。
在携带版 Alpha0.6.0 版本到0.12.1 build 1 版本期间，下界岩也会作为下界反应器的一部分生成。
获取
当使用没有附有精准采集魔咒的镐挖掘下界岩或菌岩时，会掉落下界岩。
作用
如果在下界岩上方以任意方式生成火，火将不会熄灭，可以作为很好的火源或装饰。
下界岩可用于烤制成下界砖以合成下界砖块。
在 Java 版 1.19.4 快照23w04a 版本和基岩版 1.19.80 预览版1.19.80.21 版本后，可用于合成锻造模板和肋骨盔甲纹饰锻造模板。
版本信息
下界岩在 Java 版 Alphav1.2.0 preview 版本和携带版 Alpha0.6.0 版本中加入。
在 Java 版 1.0.0 预发布版Beta 1.9-pre5、1.14 快照18w43a、18w47a 版本和基岩版 1.10.0 预览版1.10.0.3 版本中修改了纹理，相比之前它变得更加地柔和了。
在 Java 版初次加入时，被叫做血石（Bloodstone），也被称作下界石头（Netherstone）。在 Java 版 1.16 中，译名由地狱岩更改为了下界岩。
